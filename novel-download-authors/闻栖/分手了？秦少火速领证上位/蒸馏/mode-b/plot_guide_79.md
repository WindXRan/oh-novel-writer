# 情节指南：第79章

> 来源：分手了？秦少火速领证上位 第79章

## 写章目标
**前任清算 + 感情确认**。盛惜与姜霖的最终摊牌，盛惜明确说"我不喜欢你，不爱你了"，姜霖彻底出局。同时推进盛惜-秦钰的感情线（秦钰要盛惜唱歌哄他）。

## 写章节奏骨架
- 起始情绪：平静（盛惜守夜）
- 情绪强度变化：4→5→6→9→7
- 峰值位置：盛惜说"到此为止吧"
- 结束情绪：决绝、释然（盛惜离开）

- 段落节奏：前半段慢（医院守夜），后半段快（别墅摊牌）
- 场景切换次数：3次（病房→别墅门口→别墅内）
- 场景切换节奏：慢→快→中

## 写章钩子布局
- 章首钩子：秦钰要盛惜唱歌哄他睡觉，读者期待互动
- 章中钩子：姜霖在别墅门口等盛惜——前任摊牌
- 章末钩子：姜霖心碎离开，"锥心之痛"——情感高潮

## 写章时必须避开的特征
1. **楚正尧接班的细节**：楚正尧和秦钰的互动信息量低
2. **盛惜开车回家的过渡**：从医院到别墅的路程描写

## 写章可套用的公式
- 爽点公式：**前任被拒**——前任纠缠→女主明确拒绝→"到此为止"——痛快淋漓
- 虐点公式：**锥心之痛**——失去后才懂得珍惜，但为时已晚
- 悬念公式：**感情确认**——盛惜是否喜欢秦钰？本章未明确回答
- 情绪操控公式：**轻松→紧张→决绝→悲哀**——从秦钰的撒娇到姜霖的心碎

## 写章时用的叙事技巧
- 姜霖不知道秦钰的算计——他以为是盛惜变了心
- 盛惜不知道姜霖已经知道书信真相——两人各怀心事

- 盛惜对秦钰到底是什么感情？她没有直接回答
- 姜霖心碎后会做什么？读者担心他做傻事

- 用秦钰的撒娇制造轻松暧昧
- 用姜霖的狼狈形象制造反差
- 用盛惜的"污点"之说制造痛快感
- 用姜霖的"锥心之痛"制造唏嘘感
