# 情节指南：第92章

> 来源：分手了？秦少火速领证上位 第92章

## 写章目标
职场线深化+暗流涌动。展示盛惜入职后的第一天工作状态，通过losa交接、林副总不服、员工表忠心等细节，立体呈现职场权力格局，同时推进秦钰与盛惜的感情线（等她下班、共饮一杯水）。

## 写章节奏骨架
- 起始情绪：平稳/专注（losa带盛惜交接工作）
- 情绪强度变化：4→5→4→6→5
- 峰值位置：盛惜展现出专业能力让losa产生危机感
- 结束情绪：温暖/感慨（秦钰等她下班，两人聊天）

- 段落节奏：工作场景中等节奏，办公室对话快节奏
- 场景切换次数：3次（财务部办公室→盛惜办公室→秦钰办公室）
- 场景切换节奏：缓慢，前两个场景各占约40%篇幅

## 写章钩子布局
- 章首钩子：秦钰的行为被大家视为"秀恩爱"——暗示公开场合的亲密
- 章中钩子：林副总叫大家准时下班，公然对抗新领导——职场冲突升级
- 章末钩子：盛惜说"挺好，哪里都少不了勾心斗角"——暗示后续职场斗争

- 女员工："您才是我们部门的老大，我只听您的。"——赤裸裸的表忠心

## 写章时必须避开的特征
- 财务部具体业务内容和项目细节（只做宏观描述）
- 其他部门的权力结构和人际关系（本章聚焦财务部）
- 秦妍得知消息后的具体反应（留白处理，后续展开）

## 写章可套用的公式
- 爽点公式：对手试探→主角实力碾压→对手产生危机感 = 专业能力打脸
- 虐点公式：空降领导+下属公然对抗+孤立无援 = 职场初期困境
- 悬念公式：不服的副总监+表忠心的员工+识人不清 = 职场站队悬念
- 情绪操控公式：专注工作→被挑衅→识破投机→温暖收尾 = 专业+温暖并存

## 写章时用的叙事技巧
- losa不知道盛惜的真实能力，只当她是花瓶，交接时故意说专业内容想看她出丑——结果反被实力打脸
- 林副总不知道盛惜的身份背景，以为可以靠人多势众压制新领导——结果老板娘身份碾压一切

- 林副总的公然对抗暗示后续会有更大的职场冲突
- 女员工的表忠心让读者猜测盛惜会如何处理这种投机者
- losa的危机感和"笑容不自然"暗示她可能在交接中做手脚

- 用losa的"专业试探"制造紧张感
- 用盛惜的"听懂"和"记笔记"制造爽点
- 用林副总的"公然对抗"制造冲突感
- 用秦钰的"等她下班"制造温暖感
- 四段式情绪波动：紧张→爽→冲突→温暖
