# 情节指南：第86章

> 来源：分手了？秦少火速领证上位 第86章

## 写章目标
多线揭秘章。同时推进三条线：①秦钰去医院看望姜霖，两人罕见地坦诚对话，互相揭露对方不知道的真相；②盛惜在咖啡厅遇到丘嫚，得知秦钰曾在咖啡厅守望她公司；③林蓁查出林昭是秦钰的眼线。本章是信息密集型章节，通过多方视角拼凑秦钰暗恋的全貌。

## 写章节奏骨架
- 起始情绪：沉重（秦钰与姜霖对话）
- 情绪强度变化：6→7→8→9→7
- 峰值位置：丘嫚说"见到盛小姐，我就明白了"
- 结束情绪：震撼复杂（林蓁发现真相）

- 段落节奏：中速偏快，场景切换频繁，信息密度高
- 场景切换次数：4次（病房→咖啡厅→咖啡厅对话→林蓁打电话）
- 场景切换节奏：病房场景长（情感对话），咖啡厅场景快（信息揭露）

## 写章钩子布局
- 章首钩子：秦钰突然出现在姜霖病房，两人会说什么？
- 章中钩子：丘嫚暗示秦钰在咖啡厅看的是盛惜公司
- 章末钩子：林蓁发现林昭是秦钰的眼线

## 写章时必须避开的特征
1. **排除：秦钰与姜霖对话的全部内容** — 只展示关键片段，四小时对话浓缩为几个场景
2. **排除：丘嫚与秦钰的过往细节** — 只用"学弟"一笔带过，不展开暧昧
3. **排除：林昭被质问后的详细反应** — 电话场景快速收尾，不拖慢节奏

## 写章可套用的公式
- 爽点公式：多方信息汇聚 + 真相逐步揭露 = 拼图式爽感
- 虐点公式：暗恋者的自白 + 错过的五年 + 无意识的依赖 = 双向暗恋虐
- 悬念公式：A揭露B不知道的事 + B揭露A不知道的事 = 双向信息差
- 情绪操控公式：沉重对话 → 信息揭露 → 偶遇揭秘 → 眼线曝光 = 信息密度递增

## 写章时用的叙事技巧
- 秦钰不知道盛惜聊天时总有他的影子（姜霖告诉他）
- 姜霖不知道秦钰五年前告白失败的细节（秦钰告诉他）
- 盛惜不知道秦钰每天去咖啡厅是为了看她公司（丘嫚告诉她）
- 林蓁不知道林昭一直在给秦钰通风报信（她自己查出来的）

- 秦钰与姜霖四小时还聊了什么？只展示片段
- 盛惜得知这些信息后会怎么做？
- "当局者迷"——盛惜究竟爱谁？

- 前半段用秦钰与姜霖的对话制造沉重感，两人互相揭伤疤
- 中间用丘嫚的轻松语气制造转折，信息揭露的震撼感
- 结尾用林蓁的眼线曝光制造爆点，信息密度达到峰值
