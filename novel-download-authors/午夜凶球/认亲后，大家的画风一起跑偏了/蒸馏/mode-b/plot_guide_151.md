## 写章目标
**多线收束章 + 升学宴终章**。三场升学宴的最后两场收束：①林霆升学宴的余波——沈琳被流放；②谢玉露升学宴——主角团被公开感谢的社死现场。本章是"升学宴三连"的终章。

## 写章节奏骨架
```
爽感（沈琳被流放）→ 轻松（李观澜吐槽）
→ 尴尬（谢玉露升学宴被公开感谢）→ 社死（主持人念教材）
→ 搞笑（徐楠楠也社死）→ 轻松（谢家送包）
```
情绪走势：**爽→轻松→尴尬→社死→搞笑→轻松**，情绪起伏大。

- **前1/4**（1-31行）：沈琳被流放+李观澜打电话吐槽，节奏中等
- **中1/2**（32-84行）：谢玉露升学宴+被公开感谢+念教材，节奏逐渐加速到社死顶点
- **后1/4**（85-116行）：谢家送包+谢飞星互动，节奏轻快，轻松收尾

## 写章时必须避开的特征
1. **沈景明"摊牌"台词的完整结构**——「你唯一的用处」「别回来了」这套冷酷的权力宣言太有辨识度，照搬会暴露抄源文特征
2. **"日语洗脑教材"被公开念的社死场景**——「亚埋喽！西兹卡尼！」被主持人用央视标准声音念出来，这个社死场景太独特，照搬会暴露
3. **"恩将仇报"式的被感谢**——感谢致辞反而是社死惩罚，这个反转逻辑太独特，直接模仿会暴露

## 写章可套用的公式
1. **"流放"式惩罚**：不是暴力惩罚，而是送出国+给够生活费+"别回来了"，用距离和切割解决问题
2. **"被感谢=社死"反转**：公开感谢反而是惩罚（因为暴露了过去的黑历史），用反转制造喜剧
3. **"教材被公开"社死**：私下写的搞笑内容被公开处刑，经典的社死套路
4. **"误会"喜剧**：谢飞星点眉毛→李观澜以为骂她脑子有问题→实际是奶油，小误会制造笑点
5. **"跳过"的权力表达**：谢震元送包时跳过林霆，用"忽略"表达态度

## 写章时用的叙事技巧
1. **"三场升学宴"的节奏设计**：李萱→林霆→谢玉露，三场各有特色，本章完成最后两场的收束
2. **"社死"的渐进式**：从聚光灯→念教材→阵亡，社死程度逐级升级
3. **"冷酷vs温情"对比**：沈景明的冷酷流放 vs 谢家的温情感谢，两种家庭的对比
4. **"旁白吐槽"的密集使用**：「死了都要挖出来鞭尸」「恩将仇报」等旁白金句密集出现
5. **"小误会"收尾**：用谢飞星的眉毛事件这种小误会轻松收尾，不给读者留下沉重感
