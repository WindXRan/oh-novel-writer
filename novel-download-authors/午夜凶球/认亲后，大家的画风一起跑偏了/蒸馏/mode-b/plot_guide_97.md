# 情节指南：第97章

> 来源：认亲后，大家的画风一起跑偏了 第97章

## 写章目标
**配角支线推进章 + 反派衰落章**。两条线并行：①齐奶奶想撮合顾伟峰和张自珍，被顾伟峰婉拒；②朱文静去李家被拒、找李卫东摊牌失败，展示反派的逐步失势。两条线都围绕"张自珍离婚"后的余波展开。

## 写章节奏骨架
```
搞笑（齐奶奶催婚）→ 轻松（顾伟峰婉拒）
→ 压抑（朱文静被拒门外）→ 紧张（朱文静找李卫东）
→ 爽感（李卫东划清界限）→ 压抑（朱文静失势）
```
情绪走势：**搞笑→轻松→压抑→爽→压抑**，两条线情绪对冲。

- **前1/2**（1-50行）：齐奶奶催婚线，节奏轻快，以对话喜剧为主
- **后1/2**（51-99行）：朱文静线，节奏中等偏慢，展示反派困境

## 写章钩子布局
- **章末钩子**：朱文静眼神涣散盯着李卫东背影——暗示她不会善罢甘休
- **中段钩子**：顾伟峰"不想结婚"的坚定态度——与张自珍形成潜在CP空间

## 保留功能，换血肉
1. **齐奶奶催婚**：暗示顾伟峰追张自珍，顾伟峰以"她打人太猛"为由拒绝（1-48行）
2. **朱文静被孙老太拒之门外**：孙老太如见鬼般不让她进门（52-56行）
3. **朱文静找李卫东**：用老套路"是不是我的原因"试探，李卫东不按套路出牌（58-96行）
4. **李卫东划清界限**：承认"跟你是有点关系"，承诺供陈觅念书但不再无底线供给（72-94行）

- **顾伟峰的拒绝理由**（20行）：「看张壮士那个英雄气概，发起火来咱娘俩绑在一起都打不过吧」——用幽默化解催婚压力
- **齐奶奶的"气魄"**（16行）：「人家婚还没离完您就惦记上了您觉得合适吗？」——顾伟峰吐槽亲妈
- **朱文静的老套路失败**（68-72行）：她用"是不是我的原因"试探→李卫东直接说"跟你是有点关系"——打破她的预期
- **李卫东的觉醒**（88-90行）：意识到自己在朱文静身上投入超过正常水平，决定攒钱给孩子们留下什么——角色成长

- **齐奶奶**：「你觉得小张咋样？」——直接的催婚
- **顾伟峰**：「一切不以结婚为目的的恋爱行为都是耍流氓。」——用名言拒绝亲妈
- **朱文静**：「李哥，我听说你和嫂子要离婚？是不是我的原因？」——经典绿茶试探
- **李卫东**：「跟你是有点关系。」——不按套路出牌的爽感
- **齐奶奶**（内心）：「成个屁！」——老太太的内心吐槽

## 写章时必须避开的特征
1. **顾伟峰拒绝催婚的"张壮士"论**——「发起火来咱娘俩绑在一起都打不过」这个拒绝理由太独特幽默，照搬会暴露抄源文特征
2. **朱文静"以退为进"的完整话术链**——"是不是我的原因""这些年我拖累你们太多了""嫂子一定不高兴了吧"这套绿茶话术的递进太精细，照搬会暴露
3. **李卫东"不按套路"的回应方式**——直接承认"跟你是有点关系"打破朱文静预期，这个反套路设计太标志性，模仿会暴露

## 写章可套用的公式
1. **"催婚被拒"喜剧**：长辈催婚→晚辈用幽默理由拒绝→长辈放弃，轻喜剧套路
2. **"老套路失效"转折**：反派一直用同一招（以退为进试探），这次对手不按套路出牌，招式失效——展示局势变化
3. **"划清界限"场景**：角色经过反思后主动与暧昧对象划清界限，用对话完成关系重新定义
4. **"反派困境"展示**：从被拒门外→同事阴阳怪气→找靠山失败，层层叠加展示反派的衰落
5. **"双线对冲"结构**：A线（催婚喜剧）的轻松感冲淡B线（反派衰落）的压抑感，保持阅读体验平衡

## 写章时用的叙事技巧
1. **双线并行**：齐奶奶线和朱文静线交替推进，一喜一悲，节奏对冲
2. **"预期打破"**：朱文静的老套路失效，李卫东的回应打破她的预期，也打破读者的预期
3. **角色心理的"外化"**：齐奶奶的期待→失望、朱文静的试探→恐慌，都通过对话和微表情外化
4. **旁白的"吐槽"功能**：括号注释"大家不要觉得叫名字很亲密啊"——作者直接与读者对话
5. **配角的"功能性"**：顾伟峰不只是配角，他的"不想结婚"态度为后续与张自珍的关系留白
