# 钩子工程分析：第13章

> 来源：认亲后，大家的画风一起跑偏了 第13章

## 1. 段落钩子

（填入：作者怎么在段落结尾勾住读者往下读？用了什么技法？引用原文例句）

**技法：悬念中断 + 情感留白**

- **悬念中断**：第20段结尾"太吓人了。"戛然而止，读者迫切想知道什么梦能吓到李观澜。
- **情感留白**：第10段"两人你捅咕我一下我捅咕你一下，眼睛都快眨出花了。"用动作描写暗示朋友的担忧，引发读者共情。
- **提问式钩子**：第22段"什么梦能给你吓这样啊？"直接向读者提问，制造信息缺口。
- **对话中断**：第42段"我爸说……"故意中断，悬念感拉满，读者必须往下读。

**例句**：
- 第20段："不太想说。"李观澜幽幽道，"太吓人了。"
- 第42段："我爸说……"

## 2. 章首钩子

（填入：开头几段怎么抓住注意力？第一句话是什么类型的钩子？）

**技法：异常状态开场 + 悬念铺垫**

开篇直接呈现李观澜的"异常状态"：机械吃饭、答非所问、呆坐发愣。这种"人物失常"立即抓住读者注意力——她到底经历了什么？

**例句**：
- 第8段："吃饭时李观澜只是机械地把嘴填满，偶尔看着食物就那么呆呆地坐着。跟她说话也会回答，就是答得驴唇不对马嘴的。"
- 第12段："我感觉今天像是做了一个梦。"李观澜直勾勾地盯着天花板，突然喃喃轻语。

**钩子类型**：状态异常钩子（人物行为反常引发好奇）

## 3. 章末钩子

（填入：结尾怎么让读者必须翻页？是悬念、反转、情感、还是信息缺口？）

**技法：情感高潮后的情绪反转**

章末没有设置强悬念，而是用**情感共鸣**作为钩子。李观澜讲述父亲为她拼命的故事，情感达到高潮后，用"神来之笔"的幽默收尾，让读者在感动中带着微笑继续阅读。

**例句**：
- 第48段："两人本来听的很感动的，结果被李观澜的神来之笔逗得笑出了鼻涕泡儿。"

**钩子类型**：情感共鸣钩子（读者对角色产生深度共情后，自然想继续关注）

## 4. 情绪钩子

（填入：作者怎么让读者在意角色？用了什么情感绑定技法？）

**技法：逆鳞设定 + 父爱牺牲 + 成长忏悔**

- **逆鳞设定**：第30段"他俩就是我的逆鳞"——将父母定义为角色的情感底线，读者立即理解李观澜的价值观。
- **父爱牺牲**：父亲翻栅栏骂老师、醉酒回家、饭桌敬酒换钱——一系列细节构建出"为女儿拼命"的父亲形象。
- **成长忏悔**：第46段"我那天给自己甩了十几个大嘴巴"——角色的自我反思让读者产生深度共情。

**例句**：
- 第30段："谁那时候说我爸妈，我最讨厌有人说我爸妈不好，他俩就是我的逆鳞。"
- 第34段："我爸气得门都来不及走从幼儿园栅栏翻进来的。他恨得牙痒痒，一进来就骂老师……骂完抄起我就走。"
- 第46段："我那天给自己甩了十几个大嘴巴，把一脑袋黄毛剪得像鸡窝。"

## 5. 信息钩子

（填入：作者怎么制造「想知道答案」的欲望？藏了什么信息？）

**技法：信息缺口 + 逐步揭露**

- **核心悬念**：李观澜回家取充电器后到底发生了什么？为什么说"像是做了一个梦"？这个信息缺口贯穿全章，始终未揭晓。
- **次要悬念**：她做的梦内容是什么？为什么"太吓人了"？
- **部分揭露**：她承认之前讲的"大雨里走十几里地"是编的（第26段），但真正的经历仍未说。

**例句**：
- 第22段："什么梦能给你吓这样啊？李观澜就是回家去取充电器那节课开始不对劲儿的，回来路上还在公交车上睡觉了啊？"
- 第26段："我骗你的，哪有什么跑了大雨里跑了十几里地的事儿。"

**信息缺口保留**：全章结束，李观澜的真实经历仍未揭晓，读者必须翻页。

## 6. 反预期钩子

（填入：作者怎么打破读者预期？在哪里做了反转？效果是什么？）

**技法：人设反转 + 情感反转**

- **人设反转**：第26段，李观澜主动承认之前讲的感人故事是编的。读者原以为她是"受害者"，结果她也是"大忽悠"——角色更立体。
- **情感反转**：第46段，李观澜的"好姐妹"催她结账，被父亲撞见——从"叛逆少女"瞬间切换到"羞愧女儿"。
- **幽默反转**：第48段，感动的氛围被"笑出鼻涕泡儿"打破，情绪节奏张弛有度。

**例句**：
- 第26段："我骗你的，哪有什么跑了大雨里跑了十几里地的事儿。"
- 第44段："我的'好姐妹'催我结账来了，一口一个大姐大的，被我爸和他朋友看个正着。"
- 第48段："两人本来听的很感动的，结果被李观澜的神来之笔逗得笑出了鼻涕泡儿。"

**效果**：反预期让角色更真实、更有层次，避免了"卖惨"的单薄感。
