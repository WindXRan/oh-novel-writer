## 写章目标
**群体行动章 + 街头冲突章**。主角团赴京市聚会，林霆招待，吃饭时意外撞见李昭被约架。本章完成：①展示林霆的"地主之谊"；②李观澜旁观弟弟打架的荒诞喜剧；③持刀伤人事件升级，李观澜出手相救。

## 写章节奏骨架
```
轻松（林霆招待吃饭）→ 搞笑（价格吐槽）
→ 好奇（巷子里约架）→ 搞笑（菜鸡互啄）
→ 紧张（有人持刀）→ 爽感（李观澜大喊救命）
→ 悬念（李昭发现姐姐在场）
```
情绪走势：**轻松→搞笑→好奇→搞笑→紧张→爽→悬念**，节奏起伏大。

- **前1/4**（1-56行）：林霆招待+徐楠楠吐槽价格，节奏轻快
- **中1/2**（57-98行）：巷子里观战，节奏从慢到快——先旁观→菜鸡互啄→持刀升级
- **后1/4**（99-123行）：救人+收束，节奏快，悬念收尾

## 写章钩子布局
- **章末钩子**：李昭发现李观澜在场，"还不如被捅两下躺地上装死"——尴尬+悬念
- **中段钩子**：有人持刀扑向李昭——强悬念

## 写章时必须避开的特征
1. **"彩色大风车"的视觉比喻**——把两个少年扭打的脑袋比作"彩色的大风车"，这个比喻太独特且与角色（五颜六色头发）绑定，照搬会暴露抄源文特征
2. **李观澜"掏手机录像+蹲下继续看"的行为链**——把弟弟打架当戏看，这个荒诞的旁观者姿态太标志性，照搬会暴露
3. **黄毛被扎后的"还好扎的是我"反应**——这个荒诞的自我安慰逻辑太独特，直接模仿会暴露

## 写章可套用的公式
1. **"阶层差异"喜剧化**：用价格吐槽（298白粥）展示不同经济背景的角色碰撞，比直接叙述更生动
2. **"旁观者视角"看打架**：主角不参与战斗而是旁观+吐槽，用距离感制造喜剧效果
3. **"中二少年"打架描写**：用"先礼后兵""不讲武德"等夸张台词展示少年的稚嫩和可笑
4. **"意外撞见"推动情节**：吃饭时意外撞见弟弟打架，用巧合把主角拉入冲突
5. **"持刀升级"转折**：打架从拳脚升级到持刀，用危险升级制造紧张感

## 写章时用的叙事技巧
1. **"轻松→紧张"的节奏突变**：前半段吃饭喜剧→后半段街头暴力，节奏转换突然但自然
2. **"旁观者"叙事视角**：大部分场景从李观澜的旁观视角叙述，保持喜剧距离感
3. **"巧合"的合理化**：李昭学校附近吃饭→撞见打架，地理关联让巧合合理
4. **暴力的"喜剧化"处理**：打架过程用"大风车""录像""蹲下看"等元素消解暴力的严肃性
5. **"身份暴露"悬念**：李昭不知道姐姐在场→最后发现，用信息差制造尴尬笑点
