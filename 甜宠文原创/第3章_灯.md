# 第3章 灯

在茶厂干了半个月，林晚的手上长了一层茧。

称重的活确实比筛茶轻省，但也不轻松。每天要称上百袋茶叶，每袋五百克，秤砣拨来拨去，手腕酸得发僵。下班的时候她握拳都握不紧，手指头像是别人的手。

但她没换岗位。

沈泽每天都在。他在车间另一头，负责封袋和装箱。两个人隔着十几米的距离，偶尔对上视线，他会点一下头，然后移开。

他们没说过几句话。

有时候在食堂碰见，沈泽会在她对面坐下，但不怎么开口。他吃饭很快，吃完就走。有一次她注意到他的饭盒里只有白饭和咸菜，连个荤菜都没有。

她想问，但没问。

这天下午，林晚正在称重，刘师傅走过来拍了拍她的肩膀。

"你姑姑打电话来了。"刘师傅说，"让你下班早点回去。"

林晚心里咯噔一下。陈秀英从来不往厂里打电话，除非有事。

她问刘师傅说了什么，刘师傅摇头："没说，就让你早点回。"

剩下的两个小时林晚心神不宁。秤砣拨错了两次，被刘师傅骂了一顿。她一边道歉一边重新称，脑子里全是各种可能——姑父又喝多了？出了什么事？

下班铃一响她就跑了。

山路她已经走了半个月，闭着眼都能走。但今天她走得比平时快，几乎是小跑，到姑姑家的时候气都喘不匀了。

院门开着，她冲进去，看见陈秀英坐在屋檐下的板凳上，手里攥着手机，脸色不好。

"姑姑？"

陈秀英抬头看她，眼圈是红的。"回来了？"

"怎么了？"

陈秀英张了张嘴，没说出来，过了几秒才开口："你妈打电话来了。"

林晚的心沉了下去。"她说什么？"

"她说她想来看看你。"陈秀英的声音有点哑，"你继父不让她来。为这事两个人吵了一架。你妈说她在那边过得不好，继父嫌她老往你身上花钱。"

林晚站在原地，手指攥紧了裤缝。

"她说想把你接回去。"陈秀英看着她，"你怎么想？"

林晚没说话。

她不想回去。

回去住哪？继父的五金店楼上那个小阁楼？每天看继父的脸色过日子？母亲夹在中间左右为难？

"我不回去。"她说。

陈秀英叹了口气。"我就知道你会这么说。"

"我在这挺好的。"林晚在她旁边坐下，"厂里的活我能干，下个月就能转正了。"

"你一个姑娘家，总不能一直在茶厂干。"

"先干着再说。"

陈秀英没再劝。她从兜里摸出一根烟点上，抽了两口，突然说："你姑父知道了。"

"知道什么？"

"知道你妈改嫁的事。知道你继父不要你。"陈秀英的语气很平，"他在外面喝酒的时候听人说的。回来跟我闹了一场，说我把家里的钱都花在你身上了。"

林晚的手指收紧了。"姑姑，我没有花你的钱。我吃的用的都是我自己挣的。"

"我知道。"陈秀英掐灭烟，"但他不这么想。"

屋子里传出电视机的声音，老王在里面看球赛，偶尔骂一句脏话。

"你别理他。"陈秀英站起来，"他喝多了就嘴碎，明天酒醒了就忘了。"

林晚没说话。她坐在板凳上，看着院子里那几只鸡在刨土。天色暗下来了，山上的风凉飕飕地灌进院子。

晚饭的时候老王果然又喝了酒。他坐在沙发上，手里攥着啤酒瓶，盯着电视，嘴里嘟嘟囔囔的。林晚端着碗在饭桌前吃饭，陈秀英在厨房洗碗。

"你那个妈也是。"老王突然开口，声音含糊，"嫁了个开五金店的，把你扔到山里来。"

林晚放下筷子。

"你姑姑心软，什么人都往家里领。"老王灌了一口酒，"家里本来就紧巴，还多张嘴吃饭——"

"老王！"陈秀英从厨房探出头，"你少说两句。"

"我怎么了？我说的不是实话？"老王的声音大了起来，"她在这吃在这住，水电费不是钱？"

林晚站起来，把碗端到厨房。她的手在抖，碗磕在灶台上，发出一声脆响。

"你别听他的。"陈秀英按住她的手，"他喝多了。"

"我知道。"林晚说。

她回到自己的房间，关上门。

外面的声音还在继续。老王的嗓门越来越大，陈秀英在劝，劝不住。然后是什么东西摔在地上的声音，哗啦一声，像是啤酒瓶碎了。

林晚坐在床边，双手抱着膝盖。

她不想哭。但她的眼眶发酸，鼻子堵住了。她使劲咬住嘴唇，把眼泪逼回去。

手机响了。她拿起来一看——是母亲的电话。

她盯着屏幕看了五秒，按了拒接。

过了几秒，母亲发来一条消息："晚晚，妈妈想你了。你继父那边我会处理好的，你回来吧。"

林晚把手机扣在床上，没回。

外面的吵闹声渐渐小了。老王骂累了，或者醉倒了。陈秀英的脚步声从走廊经过，没有停。

林晚躺下来，把被子蒙在头上。

她想起父亲在的时候。那时候家里虽然不大，但安安静静的。父亲每天下班回来会给她带一根冰棍，夏天是橘子味的，冬天是红薯味的。她坐在沙发上看动画片，父亲在厨房做饭，母亲在阳台晾衣服。

那些日子好像上辈子的事了。

她不知道自己躺了多久，直到有人敲窗户。

不是敲门，是敲窗户。

她掀开被子坐起来，看见窗户外面站着一个人影。

是沈泽。

他站在窗外，手里拿着一个塑料袋，透过玻璃看着她。

林晚愣了两秒，下床走到窗边，把窗户推开了一条缝。

"你怎么在这？"

"路过。"沈泽说。他的声音很低，像是怕被人听见。"听见你家有动静，过来看看。"

林晚这才想起来，沈泽的住处就在山路上面一点，离姑姑家不远。老王闹出来的动静那么大，他肯定听见了。

"我没事。"她说。

沈泽看了她一眼，没说话，把手里的塑料袋从窗缝里塞进来。

林晚接过来，打开一看——是一盒创可贴和一管药膏。

"手上的茧子要护理。"沈泽说，"不然会裂口子。"

林晚低头看了看自己的手。手心的茧子确实有些地方裂了，渗着血丝，她一直没管。

"谢谢。"她说。

沈泽没应。他往后退了一步，站在院子里，月光照在他身上。他今天穿了件黑色的外套，拉链拉到下巴，显得更瘦了。

"你姑姑对你好不好？"他突然问。

"好。"林晚说。

"那个男的呢？"

林晚没回答。

沈泽也没追问。他转过身，背对着她，说了一句："要是他欺负你，你跟我说。"

然后他走了。

林晚站在窗边，看着他的背影消失在山路的黑暗里。她手里攥着那盒创可贴，塑料盒的棱角硌在掌心的茧子上，有点疼。

她把窗户关上，坐回床上。

打开药膏，挤了一点涂在手心。药膏凉凉的，涂上去之后裂口子的地方没那么疼了。她把十个手指头都涂了一遍，然后贴上创可贴。

躺在床上，她盯着天花板。那个水渍还在，还是像一朵云。

她想起了今天下午在厂里的事。称重的时候她不小心把一袋茶叶撒了，沈泽正好路过，蹲下来帮她一粒一粒捡起来。他的手指很长，捡茶叶的动作很快，指甲缝里还是嵌着洗不掉的茶渍。

她问他手上的疤怎么弄的，他说是小时候被开水烫的。她又问他为什么不上学了，他说没钱。

就这么两句话，再没别的了。

但他帮她捡了茶叶。

他还给她创可贴，给她药膏，问她姑父有没有欺负她。

她把被子拉到下巴，翻了个身，面向窗户。月光从窗帘缝里透进来，照在她贴着创可贴的手上。

她想，明天去厂里的时候，要给他带两个馒头。

不是因为别的，是因为他的饭盒里只有白饭和咸菜。

她闭上眼睛，很快睡着了。

这一次没有做梦。

第二天早上，她比平时早起了十分钟。陈秀英还没起来，厨房里冷锅冷灶的。她自己蒸了两个馒头，用塑料袋装好，塞进背包里。

出门的时候天还没亮透，山路灰蒙蒙的，雾气从山谷里漫上来，把远处的树都遮住了。她走得很快，到了厂门口的时候，大门刚开。

她在车间门口等了一会儿，看见沈泽从宿舍那边走过来。他还是那件灰色T恤，头发乱糟糟的，像是刚起来没多久。

"沈泽。"她喊了一声。

他停下来看她。

林晚从背包里掏出塑料袋，递过去。"给你。"

沈泽低头看了一眼，没接。

"我蒸多了。"林晚说，"不吃就浪费了。"

沈泽看了她一眼，接过去了。

"谢谢。"他说。

然后他转身进了车间。

林晚站在原地，看着他的背影。他走路的姿势还是那样，左脚比右脚慢半拍。

她进了车间，换上工作服，走到称重台前。

今天的茶叶是新到的，颜色比前几天的深，闻起来更香。她把秤砣拨好，开始称第一袋。

快到中午的时候，沈泽从车间那头走过来，手里拿着一瓶水，放在她的操作台上。

"喝水。"他说。

林晚抬头看他，他已经转身走了。

她拧开瓶盖喝了一口。水是凉的，带着一点点甜味。

她把水瓶放在操作台的角落，继续干活。

这一天她干得比平时快，下班的时候胳膊也没那么酸了。走出厂门的时候，她看见沈泽站在路边的老地方抽烟。

她走过去，在他旁边站定。

"明天还给你带馒头。"她说。

沈泽看了她一眼，把烟掐了。

"不用。"他说。

"为什么？"

"我有手有脚，能自己买。"

林晚笑了一下。"那我蒸多了怎么办？"

沈泽没回答。他把烟头扔在地上，用脚碾灭。

"你笑起来跟你姑姑不像。"他说。

林晚愣了一下，然后笑得更开了。"我姑姑不笑吗？"

"也笑，但跟你不一样。"沈泽说完，转身往山上走了。

林晚站在原地，看着他的背影。

这一次她没有追上去。

她站在厂门口，看着他的身影一步一步走远，消失在山路的拐弯处。然后她转身，往姑姑家的方向走去。

山路上没有路灯，但她已经不怕黑了。这条路她走了半个月，每一个拐弯、每一棵路边的树她都记得。

她走得不快，但很稳。

到家的时候，陈秀英在厨房炒菜，老王坐在沙发上看电视。今天他没有喝酒，茶几上放着一杯茶。

"回来了？"陈秀英从厨房探出头，"洗手吃饭。"

"好。"

林晚洗了手，坐在饭桌前。菜是腊肉炒蒜苗和一碗酸汤。老王端着茶杯喝了一口，没说话。

陈秀英坐下来，看了她一眼。

"手上的创可贴换过了？"陈秀英问。

"嗯，昨天换的。"

"谁给你的药膏？"

林晚夹了一筷子腊肉，没抬头。"厂里的人。"

陈秀英没再问。

吃完饭，林晚回到自己的房间。她把窗户推开，夜风灌进来，凉凉的，带着山里特有的草木味道。

她坐在窗边，把手伸出去。月光照在她的手心上，创可贴的边缘有点翘，她按了按。

远处的山黑成一片，只有天边的星星亮着。

她想，在这里待下去也挺好的。

有活干，有饭吃，有人给她创可贴。

她关上窗户，上床睡觉。

这一次她睡得很沉，连梦都没有做。
