# 《锈色月光》章纲 · 第4章—第80章

> **核心线**：讨债少女遇到跌落云端的厂长之子 → 相知 → 相爱 → 分离 → 真相 → 和解
> **情绪总弧线**：V形 + 延迟满足（前半段虐甜交织，中段谷底，后段真相爆发、治愈回升）
> **时间跨度**：2012年深秋 — 2013年深秋（约一年）

---

## 第一弧段 · 铺垫期（第4章—第10章）

**弧段功能**：建立日常、深化人设、铺设讨债主线矛盾、两人从陌生到熟悉
**情绪基调**：冷暖交替——底层生活的粗粝感中穿插微小温暖
**核心冲突**：温絮的讨债使命 vs 对沈决/奶奶生活的同情

### 第4章：纸盒里的星光
- 时间线：2012年10月下旬，温絮到达筒子楼第三天
- 核心事件：温絮帮奶奶糊纸盒，第一次看到沈决深夜从修理铺回来的背影
- 情节点：
  1. 温絮清晨被隔壁402室老太太的收音机吵醒，出门看到楼道里堆满纸盒
  2. 奶奶教她糊纸盒——浆糊的气味、手指被纸边割出细口，一个盒子挣八分钱
  3. 温絮算账：糊一千个才八十块，奶奶一个月糊两千个
  4. 傍晚温絮去楼道倒垃圾，看到沈决背着工具包从楼下上来，校服袖口沾满机油
  5. 两人在楼梯拐角对视，沈决点了个头没说话，进了403室，门关上前温絮看到屋内只有一张桌、一盏灯
- 目标情绪：心疼 + 微暖（底层生活的粗粝中透出一丝人情味）
- 章首钩子：意象预示——"筒子楼的走廊灯坏了三盏，只剩尽头那盏还亮着，像隧道尽头的出口。"
- 章尾钩子：温絮回到房间，从枕头下摸出母亲留下的信封，没有拆开。信封边角已经磨毛了。（期待度：中——信封里是什么？）

### 第5章：第三排靠窗
- 时间线：10月底，温絮入学第一天
- 核心事件：温絮转学到滨城三中，发现沈决是她的同桌
- 情节点：
  1. 温絮被班主任领进教室，安排在第三排靠窗的位置——旁边空着，桌角刻着"沈决"两个字
  2. 同学窃窃私语："那就是沈国栋的女儿……不对，是来讨债的吧？"
  3. 沈决迟到，从后门进来，看到温絮愣了一下，没说话坐下
  4. 课间温絮去接水，听到两个女生议论沈决："他爸跑了，他妈也走了，就剩他一个人住在那破楼里"
  5. 放学时沈决主动开口："你奶奶家在三楼尽头？我帮你搬过纸盒。"——语气平淡，像在说天气
- 目标情绪：陌生感 + 好奇（两个孤独的人在同一屋檐下的微妙距离感）
- 章首钩子：悬念对话——"'那个位置没人愿意坐。'班主任看了看温絮，'你坐那儿吧。'"
- 章尾钩子：温絮走出校门，回头看了一眼教学楼。沈决站在三楼走廊，正望着她这个方向。四目相对，他转身走了。（期待度：中——他在看什么？）

### 第6章：铁锈味的黄昏
- 时间线：11月初
- 核心事件：温絮第一次完整观察沈决的日常——放学后去废品站打工
- 情节点：
  1. 放学后温絮跟着沈决走了一段路，发现他没回家，而是去了火车站附近的废品站
  2. 她远远看着沈决把一捆捆废铁过秤、记账，动作熟练得不像十八岁的少年
  3. 废品站老板拍着沈决的肩："你爸欠的债，你一个人能还到什么时候？"
  4. 沈决没接话，拿了工钱转身就走，路过温絮时顿了一下——"你跟踪我？"
  5. 温絮没否认，只说了一句："我也在挣钱。"沈决看了她一眼，没再说话，走了
- 目标情绪：沉重 + 心疼（两个少年都在为生活挣扎的无力感）
- 章首钩子：反差场景——"学校的下课铃是欢快的电子音乐。沈决收拾书包的速度，像在赶末班车。"
- 章尾钩子：温絮回到筒子楼，看到403室的门缝透出灯光。她犹豫了一下，没有敲门。（期待度：弱——日常积累，为后续关系铺垫）

### 第7章：一颗橘子
- 时间线：11月中旬，期中考试前
- 核心事件：温絮用挣来的钱买了一袋橘子，悄悄放在沈决门口
- 情节点：
  1. 温絮帮隔壁老太太搬蜂窝煤，老太太给了她几个橘子
  2. 她想起沈决每天只在学校食堂打一份最便宜的素菜，犹豫后把橘子分出一半
  3. 趁楼道没人，她把橘子放在403室门口，用一张纸垫着
  4. 第二天早上温絮出门，发现门口放了一颗橘子——剥好的，用纸包着，没有留言
  5. 到了学校，沈决什么也没说，像往常一样趴在桌上睡觉。温絮看着那颗橘子，突然觉得这个秋天没有那么冷了
- 目标情绪：微甜 + 温暖（两个不善表达的人用笨拙的方式传递善意）
- 章首钩子：未完成动作——"温絮把橘子放下的时候，听到屋里传来咳嗽声。她的手悬在半空。"
- 章尾钩子：回声钩子——"温絮把那颗剥好的橘子掰开，一瓣一瓣吃了。很甜。她不知道的是，沈决在走廊那头，听着她吃橘子的声音，嘴角动了一下。"（期待度：中——双向的善意开始萌芽）

### 第8章：门缝里的催款单
- 时间线：11月下旬
- 核心事件：催债人再次上门，在沈决家门口贴了催款单，温絮目睹全过程
- 情节点：
  1. 温絮放学回来，看到两个穿皮夹克的男人站在403室门口拍门
  2. 沈决不在家——他还在废品站打工。男人把一张红色催款单贴在门上："告诉沈国栋，月底不还钱，就不是贴纸这么简单了"
  3. 温絮躲在楼梯拐角，心跳加速——她认出了其中一个人，就是第一天来时被她报警赶走的那个
  4. 男人走后，温絮把催款单撕下来，犹豫了很久，塞进了自己的口袋
  5. 沈决回来看到门上干干净净，什么也没说。但温絮注意到，他的手指在摸门框的时候停了一下
- 目标情绪：紧张 + 压迫（讨债威胁的真实感，温絮的内心撕裂）
- 章首钩子：倒计时——"催款单上写着：还款期限，11月30日。今天是11月22日。"
- 章尾钩子：温絮回到房间，把催款单展开看了很久。上面写的数字是六十万。她想起母亲信封里的东西——那个数字，刚好是三千二百。（期待度：强——两个数字之间有什么关联？）

### 第9章：奶奶的旧收音机
- 时间线：11月底
- 核心事件：奶奶讲述钢铁厂的往事，温絮第一次了解这个家庭的完整历史
- 情节点：
  1. 奶奶的收音机坏了，温絮想修但不会，想起沈决会修
  2. 沈决来帮奶奶修收音机，手法熟练——这台收音机是他小时候见过的
  3. 奶奶拉着沈决的手："你爸小时候也爱听这台收音机……"沈决的手僵了一下
  4. 奶奶讲起钢铁厂的黄金年代：三千人的大厂、热火朝天的车间、过年时发的整箱苹果
  5. "后来厂子倒了，你爸……唉，都是命。"奶奶叹了口气。沈决低着头，没说话
- 目标情绪：唏嘘 + 温暖（时代变迁下小人物的无奈，但人与人之间的温情仍在）
- 章首钩子：意象预示——"收音机里沙沙响，像从很远的地方传来的回声。奶奶说，这台收音机比沈决的年纪还大。"
- 章尾钩子：沈决修好收音机起身要走，奶奶叫住他："小决，你爸欠的钱……是厂里的事，不是你的错。"沈决站在门口，背对着她们，肩膀抖了一下。（期待度：中——沈决的过去正在慢慢揭开）

### 第10章：天台上的沉默
- 时间线：12月初，入冬
- 核心事件：温絮和沈决在天台上的第一次真正对话——关于各自的家庭
- 情节点：
  1. 温絮睡不着，上天台透气，发现沈决也在——他在听那台修好的收音机
  2. 两人并排坐着，沉默了很久。温絮先开口："你爸……真的欠了六十万？"
  3. 沈决没有回避："厂子倒了，他借了高利贷想救厂，没救成，人跑了。债全落在我头上。"
  4. 温絮想说什么，但咽了回去——她不能告诉他，自己也是来讨债的
  5. 沈决突然问："你呢？你为什么来滨城？"温絮说："我妈……让我来找一个人。"她没有说那个人就是他爸
- 目标情绪：压抑 + 共鸣（两个被命运抛弃的人在寒夜里找到了相似的灵魂）
- 章首钩子：神秘独白——"温絮想，如果她没有来滨城，没有住进这栋楼，没有遇见沈决——她是不是就不会这么难受了？"
- 章尾钩子：沈决站起来拍了拍裤子上的灰："明天冷，多穿点。"他走了。温絮看着他的背影，突然很想哭。不是因为冷。（期待度：强——感情萌芽的第一个信号）

### 🔸 弧段总结 · 铺垫期（第4—10章）

**情绪弧线**：冷 → 微暖 → 沉重 → 甜 → 紧张 → 唏嘘 → 压抑中共鸣

**已完成的铺设**：
- 人设深化：温絮的讨债使命、母亲信封的秘密；沈决的债务困境、独自撑家的坚韧
- 关系进展：从陌生到"会互送橘子"的邻居，再到天台上的交心
- 矛盾种子：六十万债务 vs 三千二百遣散费、催款单的威胁、奶奶的往事
- 伏笔：母亲信封（未拆）、催款单上的数字、奶奶口中的"厂里的事"

**五重驱动检查**：
- 压迫感：催债人上门、债务数字、沈决的困境
- 实力感：温絮的独立生存能力、沈决的修理手艺
- 认知颠覆：奶奶口中不一样的"沈国栋故事"
- 资源增值：两人的信任关系逐步建立
- 悬念增殖：母亲信封里是什么？六十万和三千二百的关系？

## 第二弧段 · 暧昧期（第11章—第20章）

**弧段功能**：感情升温、暧昧拉扯、讨债线暗中推进
**情绪基调**：暖色渐浓——冬天的寒意中，两个人互相取暖
**核心冲突**：温絮越来越喜欢沈决，但讨债使命像一根刺扎在心里

### 第11章：物理题里的温柔
- 时间线：12月上旬，期末复习
- 核心事件：沈决帮温絮讲物理题，两人第一次近距离接触
- 情节点：
  1. 温絮物理跟不上，期中考了六十二分，被班主任叫去谈话
  2. 回到座位发现桌上多了一张纸——沈决的字迹，把电磁感应的知识点用最简单的方式列了出来
  3. 放学后沈决破天荒没去打工，在教室里给她讲题。窗外下着小雪，教室只有他们两个人
  4. 讲到一道题时两人的手碰到了一起，沈决像被烫了一下缩回去，温絮低头假装没注意
  5. 讲完题沈决收拾东西要走，温絮叫住他："谢谢你。"沈决说："你底子不差，就是没人教。"——语气平淡，但温絮听出了别的意思
- 目标情绪：心动 + 微甜（青涩的暧昧感）
- 章首钩子：闪前碎片——"后来温絮想起那个雪天，总觉得物理题里的电磁场，像某种看不见的引力。"
- 章尾钩子：温絮回到筒子楼，看到403室的灯亮着。她站在门口，手里攥着那张写满公式的纸，心跳很快。（期待度：中——暗恋的萌芽）

---

### 第12章：那盏不亮的灯
- 时间线：12月中旬
- 核心事件：温絮房间的灯坏了，沈决来修，两人在黑暗中对话
- 情节点：
  1. 温絮房间的灯泡闪了几下灭了，她不敢换——电线老化，怕触电
  2. 她去敲403的门，沈决拿着工具箱跟她过来
  3. 沈决踩着桌子换灯泡，温絮在下面扶着，他的影子投在墙上，很大
  4. 灯泡换好了，但沈决说电线老化得厉害，最好全部换掉。"明天我买线来。"
  5. 温絮要给钱，沈决没接："你帮我奶奶搬过纸盒。"——他第一次提起这件事
- 目标情绪：温暖 + 安心（被人照顾的感觉）
- 章首钩子：未完成动作——"灯灭的那一秒，温絮听到隔壁传来收音机的声音。奶奶还没睡。"
- 章尾钩子：沈决走后，温絮看着新换的灯泡，很亮。她突然想起母亲离开前说的话："去找奶奶，奶奶会照顾你。"——可照顾她的，不只是奶奶。（期待度：中——依赖感开始建立）

### 第13章：早餐的温度
- 时间线：12月下旬
- 核心事件：温絮开始每天给沈决带早餐，沈决从假装不知道到默默接受
- 情节点：
  1. 温絮发现沈决每天早上不吃早餐就来学校，第一节课总是趴在桌上——不是困，是饿
  2. 她开始每天早起半小时，在早餐摊买一个煎饼果子，放在沈决桌上
  3. 前三天沈决没动。第四天，煎饼果子不见了。第五天，桌上多了一瓶矿泉水
  4. 同学开始议论："温絮是不是喜欢沈决？""沈决那种人有什么好喜欢的，穷得叮当响"
  5. 沈决听到议论，脸色变了一下。温絮想解释，沈决说："别理他们。"——声音很低，但温絮听到了
- 目标情绪：甜 + 微涩（暗恋的小心翼翼，外界的压力）
- 章首钩子：悬念对话——"'你为什么每天给我带早餐？'沈决终于问了。温絮早料到他会问，准备好的答案却没说出口。"
- 章尾钩子：温絮说："因为你帮我修了灯。"沈决看着她，眼睛里有什么东西闪了一下。他没再问。（期待度：中——两人心知肚明但都不说破）

### 第14章：竞赛名单
- 时间线：12月底，元旦前
- 核心事件：学校物理竞赛选拔，沈决被推荐但交不起报名费，温絮偷偷帮他交了
- 情节点：
  1. 物理老师宣布省级竞赛选拔，每个班推荐两人，沈决是其中之一
  2. 报名费两百块，沈决听到数字后沉默了。放学后他去找物理老师，说自己不参加了
  3. 温絮听到了对话，第二天一早去教务处帮沈决交了报名费——用的是她帮奶奶糊纸盒挣的钱
  4. 沈决发现报名表上自己的名字，去教务处问，才知道有人替他交了
  5. 他在教室门口拦住温絮："你疯了？两百块你得糊多少纸盒？"温絮说："你值得。"
- 目标情绪：心疼 + 感动（为对方付出的冲动，以及付出背后的重量）
- 章首钩子：倒计时——"报名截止日期是12月28日。今天是12月27日。"
- 章尾钩子：沈决攥着报名表，指节发白。他没有把钱还给温絮——因为他知道，还了她也不会收。他只说了一句："我会考好的。"（期待度：中——承诺的重量）

### 第15章：天台交心
- 时间线：元旦夜
- 核心事件：元旦夜两人在天台看烟花，沈决第一次主动谈起自己的父亲
- 情节点：
  1. 元旦学校放假，筒子楼里冷冷清清。温絮在天台看到远处的烟花
  2. 沈决也上来了，手里拿着两罐橘子汽水——"元旦快乐。"
  3. 两人靠在天台的水泥护栏上，看烟花在远处炸开又熄灭
  4. 沈决突然说："我爸走的那天晚上，也放了烟花。不是元旦，是厂子宣布破产那天。有人庆祝，有人哭。"
  5. 温絮没有说话，只是把手里的橘子汽水递给他。沈决接过去，喝了一口："我爸说他去借钱，会回来的。他没回来。"
- 目标情绪：酸涩 + 共鸣（两个人都在承受不属于自己的重量）
- 章首钩子：意象预示——"烟花的声音从远处传来，像什么东西碎了。温絮分不清那是庆祝还是告别。"
- 章尾钩子：烟花放完了，天台暗下来。沈决说："温絮，你有没有想过——如果有一天你知道了什么关于我的事，你会怎样？"温絮的心猛地一沉。（期待度：强——他在暗示什么？还是试探？）

### 第16章：心事
- 时间线：1月初，寒假前
- 核心事件：温絮发现自己已经喜欢上了沈决，但讨债的使命让她不敢靠近

### 第22章：温柔笔迹

  3. 她去药店买了退烧药，又去菜市场买了小米，熬了粥。沈决迷迷糊糊不肯喝，她一勺一勺喂
  4. 半夜沈决烧得说梦话："爸……你别走……爸……"温絮握着他的手，眼泪掉在他的手背上
  5. 天亮时沈决退烧了，睁开眼看到温絮趴在床边睡着了。他没有叫醒她，只是把被子往她身上拉了拉
- 目标情绪：心疼 + 温暖（最脆弱的时刻，最真实的依赖）
- 章首钩子：闪前碎片——"温絮后来想，那是她第一次听到沈决说'爸'这个字。也是唯一一次。"
- 章尾钩子：温絮醒来时发现身上多了一条被子，沈决坐在桌前做题，像什么都没发生过。但桌上多了一碗粥——是她熬的那碗，他热过了。（期待度：中——双向的温柔）

### 第29章：生日蛋糕
- 时间线：5月底，温絮生日
- 核心事件：沈决给温絮过了人生中第一个生日——用废铁做了一个小蛋糕摆件
- 情节点：
  1. 温絮的生日是5月28日。她没有告诉任何人——连奶奶都忘了
  2. 放学后沈决叫住她："今天你生日？"温絮愣了："你怎么知道？"沈决说："你入学登记表上写的。"
  3. 他从书包里掏出一个东西——用废铁片焊接的小蛋糕摆件，上面歪歪扭扭刻着"生日快乐"
  4. "废品站的铁片，我打磨过了，不割手。"沈决把东西塞给她，"没有钱买真的。"
  5. 温絮捧着那个铁蛋糕，很重，但她的手一点也不累。她说："这是我收到过最好的礼物。"——声音有点哑
- 目标情绪：感动 + 甜（最朴素的礼物，最重的心意）
- 章首钩子：悬念对话——"'你生日是什么时候？'温絮问。沈决说：'没有。我不过生日。'——他说这话时，眼睛没有看她。"
- 章尾钩子：温絮把铁蛋糕放在窗台上，和橘子汽水的瓶子并排。月光照在上面，铁皮反射出一点光——像真的蜡烛。（期待度：中——甜蜜中的心酸）

### 第30章：暴风雨前的平静
- 时间线：6月初，夏天
- 核心事件：高考前最后一个平静的周末，两人去了海边——温絮想在一切崩塌前留住这个瞬间
- 情节点：
  1. 高考倒计时一个月。周末沈决破天荒没有去打工，说："带你去个地方。"
  2. 两人坐了一个小时的公交车，到了滨城唯一的海边。海水是灰色的，但温絮觉得很好看
  3. 他们在海边坐了一下午，沈决给她讲小时候的事——爸爸带他来海边捡贝壳，那时候厂子还在
  4. 温絮听着，心里很难受。她想：如果有一天他知道了一切，他还会带她来海边吗？
  5. 回去的路上温絮靠在沈决肩上，像寒假那次一样。但这次她没有睡着，她睁着眼，看着窗外的路灯一盏一盏往后退
- 目标情绪：平静 + 酸涩（暴风雨前的宁静，读者知道好日子不多了）
- 章首钩子：意象预示——"天边有乌云，但还没有下雨。温絮想，暴风雨来之前，总是最安静的。"
- 章尾钩子：回到筒子楼，温絮在门口叫住沈决："沈决。"他回头。温絮说："如果有一天……你恨我了，你会告诉我吗？"沈决皱了皱眉："我为什么要恨你？"温絮笑了笑："没什么。"（期待度：强——暴风雨即将来临的预感）

### 第31章：奶奶倒下了
- 时间线：6月中旬，高考前两周
- 核心事件：奶奶突发脑梗被送医院，温絮需要钱但没有
- 情节点：
  1. 深夜奶奶突然倒在地上，温絮听到动静冲过去，发现奶奶说不出话，嘴歪了
  2. 她叫了救护车，沈决帮忙把奶奶抬下楼。筒子楼没有电梯，他背了六层
  3. 医生说是脑梗，需要住院。押金五千块，温絮翻遍口袋只有三百
  4. 沈决去窗口交了钱——他打工攒的所有积蓄，一共四千七
  5. 温絮蹲在走廊里哭，沈决坐在她旁边："钱的事我来想办法。你先别哭。"——他的声音也在抖
- 目标情绪：绝望 + 心疼（被生活逼到绝路的无力感）
- 章首钩子：危机升级——"奶奶倒下的那一刻，温絮觉得整栋楼都在晃。不是地震，是她的世界在塌。"
- 章尾钩子：温絮在医院走廊坐了一夜。天亮时她做了一个决定——她要拿到那笔钱。不管用什么方式。（期待度：强——危机推动她走向极端选择）

### 第32章：卖身契
- 时间线：6月下旬
- 核心事件：催债人找到温絮，提出交易：帮他们从沈决那里拿到房产证，他们帮她拿到母亲的遣散费
- 情节点：
  1. 温絮在学校门口被催债人拦住——不是上次那几个，是另一个，穿西装，很客气
  2. "温小姐，我知道你来滨城是为了什么。三千二百块，对吧？"温絮的心沉了下去
  3. "我们做个交易。沈决住的那套房子，房产证在他手里。你帮我们拿到，我们给你三万二——你母亲的遣散费，一分不少。"
  4. 温絮说："你们想让他流落街头？"男人笑了："他迟早要搬的。房子是厂里的，不是他的。"
  5. 温絮没有答应，也没有拒绝。她回到医院，看到奶奶躺在病床上，输液管一滴一滴地落
- 目标情绪：撕裂 + 绝望（被逼到墙角，两个选择都是错的）
- 章首钩子：悬念对话——"'温小姐，你有一天时间考虑。'男人把名片递给她，'明天这个时候，我在这里等你。'"
- 章尾钩子：温絮把名片攥在手里，攥了很久。名片上写着一个名字和一个电话号码。她没有打。但她也没有扔。（期待度：强——她会怎么选？）

### 第33章：讨债人的选择
- 时间线：6月底
- 核心事件：温絮拒绝了催债人的交易，但发现沈决已经开始怀疑她
- 情节点：
  1. 温絮没有去赴约。她把名片撕了，扔进医院的垃圾桶
  2. 她回到筒子楼，在楼梯口碰到沈决。他看了她一眼："你去医院了？奶奶怎么样？"
  3. 温絮说还在观察。沈决点点头，转身要走，突然说："今天有人来找我，问我认不认识一个叫温絮的女孩。"
  4. 温絮的心跳漏了一拍："谁？"沈决说："不知道。穿西装的。"他看着温絮，"你认识？"
  5. 温絮摇头。沈决看了她很久，没有再问，转身上了楼。但温絮注意到，他关门的声音比平时重
- 目标情绪：紧张 + 不安（谎言开始出现裂缝）
- 章首钩子：闪前碎片——"温絮后来想，如果那天她说了实话，一切会不会不一样？但她没有。她选择了沉默。"
- 章尾钩子：温絮站在403室门口，犹豫了很久，抬手敲了门。门开了，沈决站在里面，看着她。她张了张嘴，说出口的却是："你明天有空吗？帮我给奶奶带点东西。"（期待度：强——她在错失最后的坦白机会）

### 第34章：真相
- 时间线：7月初，高考结束
- 核心事件：沈决在温絮的枕头下发现了母亲的信——知道了她来滨城的真正目的
- 情节点：
  1. 高考结束了。温絮去医院看奶奶，沈决帮她收拾房间——他想把她乱堆的书整理一下
  2. 他翻开枕头，看到了那个信封——已经拆开了。信纸上的字很工整，是他父亲的字
  3. 他读完了信。三千二百块。遣散费。"你妈在厂里工作了八年，沈国栋只发了十分之一。"
  4. 他把信放回原处，把枕头摆好。然后坐在温絮的床上，看着窗外，坐了很久
  5. 温絮回来看到沈决坐在她的房间里，脸色很白。"你怎么了？"沈决站起来："没什么。我走了。"——声音很轻，像怕惊动什么
- 目标情绪：窒息 + 恐惧（秘密暴露的瞬间，一切要崩塌了）
- 章首钩子：闪前碎片——"温絮后来想，那天她出门前应该把信藏好。但她没有。因为她从来不觉得沈决会翻她的东西。——她错了。"
- 章尾钩子：沈决走出402室，回到403室，关上门。他靠在门上，滑坐在地上。他想笑，但笑不出来。原来她也是来讨债的。（期待度：极强——真相暴露，读者等待爆发）

### 第35章：我喜欢你
- 时间线：7月初
- 核心事件：沈决质问温絮，温絮在崩溃中说出"我喜欢你"——但沈决已经无法相信
- 情节点：
  1. 第二天温絮去找沈决，发现他不在家。她在天台找到了他——他在听收音机，但收音机没有开
  2. "你什么时候知道的？"沈决没有回头。温絮愣了。"信。我看到了。"沈决的声音很平
  3. 温絮想解释，沈决打断她："你来滨城，是为了讨债。你接近我，是因为我是沈国栋的儿子。"
  4. "不是——"温絮的声音在发抖，"我不是故意的……我一开始不知道……后来……"她哭了出来，"后来我喜欢你了。"
  5. 沈决站起来，看着她："你喜欢他笑了一下，比哭还难看，"你喜欢我的方式，就是骗我？"
- 目标情绪：撕裂 + 绝望（最残忍的质问，最无力的辩解）
- 章首钩子：未完成动作——"温絮站在天台门口，看到沈决的背影。她想转身走，但她的脚不听使唤。"
- 章尾钩子：沈决从她身边走过，没有看她。温絮抓住他的袖子，他甩开了。"别碰我。"——三个字，像刀。（期待度：极强——关系彻底破裂，读者心碎）

### 第36章：高烧梦呓
- 时间线：7月中旬
- 核心事件：温絮病倒了，高烧不退。沈决没有来看她，但402老太太说他半夜来敲过门
- 情节点：
  1. 温絮淋了一场雨，当天晚上就烧起来了。奶奶还在医院，402老太太来照顾她
  2. 她烧得迷迷糊糊，喊了沈决的名字。402老太太叹了口气："那孩子来过。"
  3. "半夜来敲我的门，说听到你在咳嗽。我让他进来，他站在门口看了一眼，没进来。"
  4. "他让我给你煮姜汤。我说我不会，他就自己煮了，端过来放在门口。我开门的时候，他已经走了。"
  5. 温絮喝着姜汤，眼泪掉进碗里。她想：他还是在乎她的。但他不肯原谅她
- 目标情绪：心碎 + 微暖（被拒绝后的一丝温暖，更让人心疼）
- 章首钩子：意象预示——"雨下了一整夜。温絮想，这场雨是不是替她哭的？"
- 章尾钩子：温絮退烧后出门，看到403室的门关着。她敲了门，没有回应。她又敲了一次，还是没有。她靠在门上，说了一句："对不起。"门里面没有声音。（期待度：强——和解的希望渺茫）

### 第37章：醋意暗涌
- 时间线：7月下旬
- 核心事件：周琳开始频繁出现在沈决身边，温絮看着但不能说什么
- 情节点：
  1. 高考成绩出来了，沈决考上了省城的大学。温絮也考上了——但在另一个城市
  2. 周琳来找沈决庆祝："沈决，我们考上同一个城市了！以后可以一起走。"
  3. 沈决没有拒绝，也没有答应。但温絮注意到，他跟周琳说话时，眼睛是看着她的
  4. 温絮想上前说点什么，但她的脚钉在原地。她有什么资格说？她是一个骗子
  5. 放学后温絮一个人走回筒子楼，经过那条铁轨——他们牵手走过的那条。铁轨上长满了草，像很久没人走过了

- 章首钩子：反差场景——"周琳的笑声很甜。温絮想起自己第一次见到沈决时，他连笑都不会。"
- 章尾钩子：温絮在铁轨上走了很久，走到天黑。她想，如果她沿着铁轨一直走下去，是不是可以走到一个没有秘密的地方？但她走不到。（期待度：中——失去的痛苦）

### 第38章：生日蛋糕（下）
- 时间线：7月底，沈决生日
- 核心事件：温絮给沈决过了一个生日——用糊纸盒挣的钱买了一个小蛋糕。沈决没有拒绝
- 情节点：
  1. 温絮从入学登记表上查到沈决的生日：7月28日。她用糊纸盒挣的最后一点钱，买了一个最小的蛋糕
  2. 她在403室门口等了很久。沈决回来看到她和蛋糕，站在原地没有动
  3. "生日快乐。"温絮把蛋糕递过去。沈决没有接。"你不用这样。"
  4. 温絮说："我知道你恨我。但今天是你生日。"她把蛋糕放在门口，转身要走
  5. "等一下。"沈决叫住他。他拿起蛋糕，打开盒子，看着上面歪歪扭扭的"生日快乐"——和他给她做的铁蛋糕一样的字迹。他没有说话，但温絮看到他的眼睛红了

- 章尾钩子：沈决端着蛋糕进了屋，关门前说了一句："谢谢。"门关上了。温絮站在走廊里，听到里面传来收音机的声音——很老的那首歌。（期待度：强——最后的温柔，分离在即）

### 第39章：蝉鸣散去
- 时间线：8月初
- 核心事件：温絮离开滨城前最后一次去天台，发现沈决也在。两人没有说话，但温絮把一封信留在了403室门口
- 情节点：
  1. 温絮的火车票是明天。她收拾好行李，把奶奶从医院接回来，托付给402老太太
  2. 最后一个晚上她去了天台——沈决也在，像第一次那样坐在水泥护栏上
  3. 两人没有说话。蝉鸣声很响，像在替他们告别。温絮想说很多话，但一个字也说不出来
  4. 沈决先站起来，经过她身边时停了一下："路上小心。"——他知道了
  5. 温絮回到楼下，在403室门口放了一封信。信里写着她来滨城的全部经过，和那句一直没说出口的话
- 目标情绪：离别 + 遗憾（无声的告别，最重的遗憾）
- 章首钩子：意象预示——"蝉鸣声从远处传来，像在哭。温絮想，蝉的生命只有一个夏天。她和沈决的夏天，也要结束了。"
- 章尾钩子：第二天早上温絮离开筒子楼时，403室的门关着。她没有敲门。走到楼梯口时，她回头看了一眼——三楼走廊的灯坏了三盏，只剩尽头那盏还亮着。和她来的时候一样。（期待度：极强——离别，读者期待后续发展）

### 第40章：空房间
- 时间线：8月中旬
- 核心事件：温絮离开后，沈决一个人坐在403室里，读了那封信
- 情节点：

  4. 温絮说："好。"她想：这一次，她不会再松手了
  5. 两人继续往前走。远处是省城的天际线，阳光照在上面，很亮
- 目标情绪：希望 + 温暖（面向未来，携手同行）
- 章首钩子：意象预示——"银杏叶落了一地，像铺了一层金黄色的地毯。温絮想，这条路上，以后会有很多落叶。但有人陪她一起踩。"
- 章尾钩子：温絮和沈决走到路的尽头。前面是城市，是未来。温絮想：她来滨城是为了讨债。但她留下来的，是因为爱。（期待度：中——主题升华）

### 第79章：信（终）
- 时间线：11月
- 核心事件：温絮写了一封信——这次不是寄给沈决的，而是寄给自己的。信上写着她来滨城的全部经历
- 情节点：
  1. 温絮在宿舍里写信，室友问她写给谁。她说："写给以后的自己。"
  2. 信上写着：她来滨城的第一天，天台上的少年，修好的收音机，一颗橘子，物理题里的温柔，铁轨上的牵手，三千二百块，二十封信，橘子汽水的约定
  3. 她写了沈决——冷硬的、温柔的、沉默的、笑着的。她写了自己——害怕的、勇敢的、愧疚的、坚定的
  4. 信的最后她写："如果你看到这封信，说明你已经走出来了。那些事都过去了。但你不要忘。因为那些事让你成为了现在的你。"
  5. 她把信折好，放进铁蛋糕的底座里。拧紧。这一次，里面又有了信
- 目标情绪：释然 + 力量（与过去和解，拥抱未来）
- 章首钩子：闪前碎片——"温絮后来想，那封信是她写给自己的情书。"
- 章尾钩子：温絮把铁蛋糕放在枕头旁边。她闭上眼，想起沈决说过的话："路上小心。"——她想，她会的。但这次，她不是一个人走。（期待度：中——象征意义的闭环）

### 第80章：锈色月光
- 时间线：12月，除夕
- 核心事件：除夕夜温絮和沈决在新家的阳台上看烟花——这次有奶奶的遗像、有橘子汽水、有彼此
- 情节点：
  1. 除夕夜沈国栋做了一桌子菜。温絮、沈决、沈国栋、402老太太，四个人围在一起
  2. 饭后温絮和沈决站在阳台上。远处有烟花，像那年元旦一样。但这次，他们不是在天台，是在新家
  3. 沈决从口袋里掏出两瓶橘子汽水。"给你。"温絮接过来。他每年除夕都请她喝，一次也没忘
  4. 温絮喝了一口，很甜。她说："沈决。"他嗯了一声。温絮说："新年快乐。"沈决说："新年快乐。"
  5. 烟花放完了。温絮靠在沈决肩上，看着夜空。她想：筒子楼没了，铁轨没了，天台没了。但她还在，他还在。这就够了
- 目标情绪：温暖 + 希望（苦尽甘来，珍惜当下）
- 章首钩子：意象预示——"除夕的烟花比任何一年都多。温絮想，是不是因为值得庆祝的事，越来越多了？"
- 章尾钩子：温絮在日记本上写了最后一行字——"今天是除夕。沈决请我喝了橘子汽水。和那年一样甜。"她合上日记本，放在铁蛋糕旁边。窗外是沈决房间的灯光。两盏灯，隔着一面墙，亮了一整夜。和那年一样。（期待度：圆满——全书终章，回环呼应第20章）
