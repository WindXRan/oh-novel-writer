# 新书概念

## 基本信息

- **书名**：学姐从不恋爱，直到遇见我
- **作者**：（仿写自隔胳呜呜）
- **类型**：都市日常 / 校园恋爱 / 单女主 / 1v1
- **核心卖点**：日常轻松 + 小甜文 + 无系统 + 不重生 + 扮猪吃虎
- **目标字数**：200-300万字（长篇连载）
- **目标平台**：番茄小说

## 故事梗概

林知序高中暗恋了三年的女孩，甚至考上了跟女孩一样的大学，可女孩却在他考上后反悔了。

悲痛之下，林知序化身音乐创作人，在短视频平台发布原创歌曲，没想到大学还没开学，他写的歌突然爆火，他竟然火了……

上大学后，林知序原以为自己再也不会谈恋爱，却偶然相识了高冷学姐。

林知序决定，要偷偷跟学姐谈恋爱，然后惊艳所有人。

等他和高冷学姐手拉手出现在学校里的时候，全校都震惊了。

音乐系冰山女神温时雨的身边竟然出现了男人？

这时，暗恋三年的女孩却发现自己心里是有他的，于是再次展开了对林知序的追求。

"抱歉，小学弟已经是我的人了。"温时雨强势的说道。

……

"后来呢？林知序先生，您跟知名音乐制作人温时雨是怎么相爱的呢？"

听到此话，林知序摸了摸脑袋，看着那个绝美的身影，轻声笑道："她偷偷给我写的歌编了曲……"

## 核心设定

### 男主：林知序

- **年龄**：18岁，大一新生
- **专业**：数字媒体技术
- **外貌**：身高180，长相清秀帅气，但性格内向害羞
- **性格**：嘴笨、容易害羞、不擅长跟女生说话，但内心细腻、真诚
- **特长**：音乐创作（词曲创作、吉他、钢琴）
- **背景**：父母都是警察，牺牲在任务中，由小姨抚养长大，表哥是锦城大学的教授
- **网名**：「晚风」（音乐平台账号，粉丝不知道他是大学生）
- **暗恋经历**：高中暗恋三年的女孩，考上同一所大学后被反悔拒绝

### 女主：温时雨

- **年龄**：21岁，大三学姐
- **专业**：音乐表演（主修钢琴）
- **外貌**：身高168，黑色长直发，容貌绝美，身材高挑
- **性格**：高冷、话少、工作狂，但内心有柔软的一面（怕打雷）
- **身份**：音乐系系花、校学生会文艺部部长
- **特长**：钢琴、编曲、音乐制作
- **背景**：父母离异，从小跟母亲长大，母亲是音乐学院教授
- **网名**：「雨声」（音乐平台上的编曲师，与「晚风」是网友搭档，但双方不知道对方真实身份）

## 故事弧线

### 第一卷：仙女学姐（第1-50章）

- **核心事件**：入学误会 → 认识学姐 → 发现学姐是网友搭档 → 暗恋开始
- **情绪弧线**：失落 → 惊喜 → 尴尬 → 心动 → 甜蜜

### 第二卷：偷偷喜欢（第51-150章）

- **核心事件**：军训日常 → 学生会纳新 → 合作创作 → 感情升温
- **情绪弧线**：暗恋 → 试探 → 甜蜜 → 误会 → 和好

### 第三卷：官宣恋情（第151-300章）

- **核心事件**：表白 → 官宣 → 面对前任追求 → 甜蜜日常
- **情绪弧线**：紧张 → 幸福 → 困扰 → 坚定

### 第四卷：携手同行（第301-500章）

- **核心事件**：音乐事业 → 毕业 → 求婚 → 未来
- **情绪弧线**：奋斗 → 成长 → 感动 → 圆满

## 章纲事件更换原则

源文第1章：男主进局子（误会私闯民宅）
新书第1章：男主被误认为跟踪狂（在校园里被保安拦下）

源文第2章：男主在图书馆码字，遇到学姐
新书第2章：男主在琴房录歌，遇到学姐

源文第3章：室友介绍美女排行榜
新书第3章：室友介绍音乐系系花

**核心原则**：只保留情绪弧线和冲突类型，具体事件必须完全不同。
