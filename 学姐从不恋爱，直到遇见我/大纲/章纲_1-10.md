# 章纲：第1-10章

## 第1章 被当成跟踪狂了
- 源文对应：源文第1章 进局子了
- 源文事件：洛野去教师公寓，被苏白粥误认为小偷报警，被抓到派出所
- 新书事件：林知序第一天到锦城大学，在校园里迷路，被保安误认为跟踪女生的可疑人员，被带到保安室盘问。发现表哥沈砚清出国了，教师公寓被租给了一个女生。
- 情绪弧线：懵逼 → 委屈 → 无奈 → 恍然大悟
- 钩子：发现教师公寓住着一个神秘女生
- 场景：锦城大学校园、保安室
- NPC：林知序、保安、沈砚清（电话中）

## 第2章 琴房里的学姐
- 源文对应：源文第2章 爱吃火锅
- 源文事件：洛野在图书馆码字，遇到苏白粥，雨中送伞，发现她是网友"爱吃火锅"
- 新书事件：林知序去音乐楼找琴房录歌，误入一间正在练琴的琴房，发现是那天在校园里遇到的高冷学姐。学姐正在弹钢琴，他被琴声吸引，两人短暂交流后离开。
- 情绪弧线：好奇 → 惊艳 → 尴尬 → 心动
- 钩子：学姐弹的曲子，竟然是他之前发在网上的原创曲
- 场景：音乐楼琴房
- NPC：林知序、温时雨

## 第3章 室友们
- 源文对应：源文第3章 男士外套
- 源文事件：室友们到齐，讨论美女排行榜，发现苏白粥是校花，沈乔被女生追着要微信
- 新书事件：室友们到齐，周子昂（健身达人）、陆言琛（校草级帅哥）、张一鸣（话痨）各有特色。张一鸣开始介绍音乐系系花温时雨的传说，说她是冰山女神，从不跟男生说话。
- 情绪弧线：兴奋 → 好奇 → 震惊（发现学姐就是系花）
- 钩子：林知序发现那天在琴房遇到的学姐，就是传说中的冰山系花
- 场景：九栋618寝室、三食堂
- NPC：林知序、周子昂、陆言琛、张一鸣

## 第4章 军训出名了
- 源文对应：源文第4章 军训
- 源文事件：军训开始，王大锤被罚跑步，李昊阳被女生送脉动，洛野在才艺表演时唱歌
- 新书事件：军训第一天，张一鸣在站军姿时睡着被罚跑步。休息时间，陆言琛被女生围住要微信。林知序被室友怂恿上台唱歌，一首原创歌曲惊艳全场，被人拍下传到表白墙。
- 情绪弧线：疲惫 → 搞笑 → 惊艳 → 得意
- 钩子：温时雨在表白墙上看到了林知序唱歌的视频
- 场景：军训操场
- NPC：林知序、周子昂、陆言琛、张一鸣、教官

## 第5章 仙女学姐的外套
- 源文对应：源文第5章 仙女学姐
- 源文事件：军训结束，洛野喝醉后遇到苏白粥，苏白粥把外套披在他身上
- 新书事件：军训结束后聚餐，林知序喝醉后在教学楼迷路，遇到刚从迎新晚会彩排回来的温时雨。温时雨穿着礼裙，美得像仙女，她把外套披在醉醺醺的林知序身上就离开了。
- 情绪弧线：兴奋 → 迷糊 → 惊艳 → 惆怅
- 钩子：第二天醒来，林知序发现身上多了一件带着香味的女士外套
- 场景：教学楼门口
- NPC：林知序、温时雨

## 第6章 迎新晚会
- 源文对应：源文第6章 新生晚会
- 源文事件：新生晚会，苏白粥表演节目
- 新书事件：迎新晚会，温时雨作为音乐系代表表演钢琴独奏。林知序在台下认出她就是那天在琴房遇到的学姐，也是给他披外套的仙女。晚会结束后，两人在后台偶遇，温时雨认出了他。
- 情绪弧线：期待 → 惊艳 → 紧张 → 尴尬
- 钩子：温时雨说了一句"原来是你"
- 场景：学校礼堂
- NPC：林知序、温时雨、室友三人

## 第7章 网友"雨声"
- 源文对应：源文第7章（源文中游戏开黑情节）
- 源文事件：洛野和室友们一起打游戏，展示游戏实力
- 新书事件：林知序在寝室录歌，准备发到网易云。他联系网友搭档"雨声"帮忙编曲，"雨声"回复很快，两人在线上合作默契。林知序不知道的是，"雨声"就是温时雨。
- 情绪弧线：专注 → 愉快 → 期待 → 甜蜜（不知情的甜蜜）
- 钩子：温时雨收到"晚风"发来的新歌demo，发现旋律有点耳熟
- 场景：九栋618寝室
- NPC：林知序、温时雨（线上）、室友三人

## 第8章 学生会纳新
- 源文对应：源文第8章（学生会相关情节）
- 源文事件：学生会纳新，洛野加入学生会
- 新书事件：学生会文艺部纳新，林知序被张一鸣拉着去报名。面试时，部长竟然是温时雨。温时雨看到林知序的简历上写着"擅长词曲创作"，表情微微变化，最终让他通过了面试。
- 情绪弧线：紧张 → 惊讶 → 期待 → 欣喜
- 钩子：温时雨面试时问了一句"你就是'晚风'？"
- 场景：学生活动中心
- NPC：林知序、温时雨、张一鸣

## 第9章 第一次部门会议
- 源文对应：源文第9章（学生会工作相关）
- 源文事件：苏白粥安排工作任务
- 新书事件：文艺部第一次部门会议，温时雨安排校园歌手大赛的筹备工作。林知序被分配到节目审核组，需要跟温时雨对接。两人第一次正式工作接触，温时雨依旧高冷，但对林知序的音乐才能表现出认可。
- 情绪弧线：认真 → 紧张 → 尴尬 → 小确幸
- 钩子：温时雨在会议结束后，单独留下林知序，问他"你那首《雨天》是在哪里学的？"
- 场景：学生活动中心会议室
- NPC：林知序、温时雨、文艺部成员

## 第10章 校园歌手大赛
- 源文对应：源文第10章（校园活动相关）
- 源文事件：校园活动，洛野展示才艺
- 新书事件：校园歌手大赛海选，林知序报名参加。他唱了一首原创歌曲，评委席上的温时雨表情复杂——这首歌的编曲，是她在网上帮"晚风"做的。海选结束后，温时雨找到林知序，两人第一次面对面坦白网友身份。
- 情绪弧线：紧张 → 自信 → 震惊（发现真相）→ 甜蜜
- 钩子：温时雨说"原来你就是'晚风'……我应该早点猜到的"
- 场景：学校礼堂
- NPC：林知序、温时雨、室友三人
