# 核心设定表

## 基本信息

- **书名**：重生回到小县城，开局每天三十万
- **题材/类型**：都市重生·县城神豪·基建种田
- **目标平台**：番茄小说
- **预计字数**：150-200万字
- **目标读者**：18-35岁男性，大城市打工人，有"回乡"情结

## 一句话梗概

前世在大城市996猝死的建筑师林远，重生回到老家临水县，绑定"县域振兴系统"——全县每人每天给他十块钱。为了每天的进账翻十倍、百倍，他用前世经验和无限资金，把一个连三甲医院都没有的十八线小县城，建成让全世界疯狂涌入的超级大都会。

## 主角设定

- **姓名**：林远
- **年龄**：28岁（重生时）
- **前世身份**：大城市建筑师，连续加班四天后猝死在工位上
- **重生时间点**：2019年，大学毕业3年后
- **核心特质**：稳重、有远见、重感情
- **金手指/核心能力**：县域振兴系统（按人口发钱）+ 前世20年记忆（知道未来趋势）
- **弱点/缺陷**：前世太拼导致猝死，今世要学会平衡工作和生活
- **核心动机**：不想再走前世的老路，要在老家过好日子，顺便改变家乡

## 世界观骨架

- **时代/背景**：2019年中国，一个叫临水县的十八线小县城
- **核心设定**：县城常住人口3万，产业空心化，年轻人外流严重
- **力量体系**：无（都市现实题材）
- **社会结构**：县城人情社会，关系网重要，体制内是铁饭碗

## 核心冲突

- **主线矛盾**：林远要建设县城 vs 各种势力阻碍（地头蛇、官僚、资本）
- **终极阻碍**：县城人口增长到100万的系统终极目标
- **情感线**：林远与家人、朋友、爱情的关系

## 系统规则

| 维度 | 规则 |
|------|------|
| **系统名称** | 县域振兴系统 |
| **触发条件** | 重生当天，手机屏幕黑屏→金色字体浮现→系统绑定 |
| **发放规则** | 每日凌晨自动到账，金额 = 县域常住人口 × 10元 |
| **初始金额** | 约30万/天（临水县常住人口3万） |
| **使用限制** | 仅限县域内消费；不能网购外地商品；不能转入理财；不能带出县外 |
| **成长路径** | 人口增长→日入金额增长（3万→30万/天，10万→100万/天，100万→1000万/天） |
| **隐藏目标** | 常住人口突破50万→系统全面升级（解锁跨县投资权限） |
| **终极目标** | 常住人口突破100万→"县域崛起"特别奖励 |

## 三阶段风格

| 阶段 | 章节范围 | 风格 | 核心爽点 |
|------|----------|------|----------|
| 前期 | 1-50章 | 轻松喜剧 | 有钱花不出去的反差爽 |
| 中期 | 51-150章 | 扮猪吃虎 | 法律+资金+信息差碾压 |
| 后期 | 150+章 | 写实温情 | 乡村振兴+小人物群像 |

## 递进式反派设计

| 层级 | 反派类型 | 解决方式 | 章节范围 |
|------|----------|----------|----------|
| L1 | 地头蛇/商业对手 | 收购+法律 | 51-80章 |
| L2 | 地方势力/高利贷 | 法律+资金 | 81-110章 |
| L3 | 基层官僚/城管 | 投诉+媒体 | 111-130章 |
| L4 | 中层官员/世家 | 利益交换 | 131-150章 |

## 核心爽点循环

```
循环1（消费喜剧）：有钱花不出去→建设消费场景→消费体验提升→新消费场景匮乏
循环2（产业建设）：发现产业机会→建厂→创造就业→人口回流→系统增收
循环3（合法碾压）：对手挑衅→法律+资金+信息差碾压→声望跃升→更强对手出现
循环4（基建种田）：县城基础设施落后→投资建设→县城面貌改变→外部认可
```
