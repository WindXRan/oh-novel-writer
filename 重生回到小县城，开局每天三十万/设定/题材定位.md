# 题材定位

## 核心梗三层递进设计

### 一代：主题/中心思想

**"用资本改变家乡，让小地方不再小"**

- 主题1：乡村振兴——不是为了赚钱而建设，而是为了让更多人能在家门口找到出路
- 主题2：人生意义——前世为了大城市拼命，今世要找到真正值得为之奋斗的东西
- 主题3：人情温暖——县城的烟火气和人情味，是大城市给不了的

### 二代：题材核心

**"县城神豪+基建种田"**

- 核心吸引力：无限资金在有限场景中的碾压感
- 核心卖点：有钱花不出去的反差喜剧+建设家乡的成就感
- 核心情绪：掌控感+成长快感+道德满足感

### 三代：核心情绪

**"扮猪吃虎+弥补遗憾"**

- 情绪链条：被低估→展示实力→震惊所有人→改变家乡
- 遗憾弥补：前世猝死→今世重生→不再错过家人和朋友

## 核心梗定义

**核心梗 = 重生者带着前世记忆 + 按人口发钱的系统 + 只能在县城花 + 建设家乡**

读者最想看的：一个在大城市混不下去的人，回到老家后，用无限资金把破县城变成理想国。

## 微创新与差异化设计

| 差异化点 | 三本书的做法 | 合体版的创新 |
|----------|-------------|-------------|
| 主角身份 | 创业失败者/打工人/重生者 | 重生者+建筑师背景（懂建设） |
| 系统规则 | 固定金额/按人口×1/按人口×100 | 按人口×10（平衡写实和爽感） |
| 风格 | 单一风格 | 三阶段递进（喜剧→碾压→温情） |
| 反派设计 | 平铺/递进 | 五层递进+每层解决方式不同 |
| 长期驱动 | 无/人口增长 | 人口增长飞轮+里程碑奖励 |

## 冲突网络设计

### 纵向矛盾（层级递进）

```
L1：地头蛇/商业对手（51-80章）
    ↓ 解决方式：收购+法律
L2：地方势力/高利贷（81-110章）
    ↓ 解决方式：法律+资金
L3：基层官僚/城管（111-130章）
    ↓ 解决方式：投诉+媒体
L4：中层官员/世家（131-150章）
    ↓ 解决方式：利益交换
L5：高层博弈（150+章）
    ↓ 解决方式：坦诚对话+共赢
```

### 横向矛盾（日常冲突）

- 家人不理解→催婚压力→亲戚质疑
- 县城商业生态落后→消费场景匮乏→产业空心化
- 外出务工者→留守儿童→空巢老人

### 交叉矛盾（纵横交织）

- 地头蛇+家人：地头蛇威胁主角家人
- 官僚+产业：官僚刁难主角的建设项目
- 世家+爱情：世家子弟追求女主角

## 事业线设计

### 主线：县城建设

```
第1阶段（1-50章）：个人消费+考察县城
第2阶段（51-100章）：建厂+创造就业
第3阶段（101-150章）：基建+产业升级
第4阶段（151-200章）：区域整合+县城崛起
```

### 支线：产业发展

- 服装厂（第一桶金）
- 餐饮连锁（消费升级）
- 房地产（城市建设）
- 教育医疗（民生改善）

## 爱情线设计

### 女主角设定（待定）

- 身份：县城本地人/外来创业者
- 特点：独立、有主见、不拜金
- 与主角的关系：从误解到理解到相爱
- 冲突点：女主不理解主角为什么要在县城建设

### 爱情线节奏

- 前期：相遇→误解→小互动
- 中期：理解→支持→暧昧
- 后期：表白→相爱→共同奋斗

## 题材核心卖点检验清单

- [x] 读者翻开最想看什么？→ 重生者如何用系统改变县城
- [x] 每个剧情单元是否在展示核心梗？→ 消费喜剧+产业建设+合法碾压
- [x] 核心梗爽点模式能否循环？→ 四个循环交织运行
- [x] 有无挂羊头卖狗肉？→ 全程围绕县城建设展开
- [x] 全文每章是否满足三要素中至少一个？→ 每章有期待点/爽点/过渡
