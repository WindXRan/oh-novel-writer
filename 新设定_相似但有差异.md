# 新设定：相似但有差异

## 原书核心设定（保留）
- 主角：穿越者+长生系统
- 系统：徒弟修炼，师父共享，永远比徒弟高一手
- 师父人设：懒癌+健忘
- 徒弟来源：捡来的婴儿
- 核心矛盾：等徒弟
- 结构：单元式，每卷一个徒弟

## 新版设定（相似但有差异）

### 主角：沈忘
- **身份**：穿越者，绑定「忆海系统」活了三世（每世约300年）
- **系统**：「忆海系统」——收徒解锁记忆碎片，徒弟越强，师父想起的前世越多
- **人设**：健忘、话少、外冷内热
- **核心矛盾**：活了三世却忘了自己是谁，每次收徒都像第一次

### 徒弟：楚辞（原名阿辞）
- **身份**：灭门遗孤，身负「诅咒」（与师父前世记忆有关联）
- **性格**：沉默寡言、倔强、重情义
- **核心特点**：每次师父解锁记忆碎片，诅咒就会减轻一点

### 核心爽点（保留原书公式）
1. **等待焦虑**：主角等待某件事很久，终于实现
2. **系统激活**：金手指系统触发，带来希望
3. **捡到宝**：获得徒弟/伙伴，开启新篇章
4. **扮猪吃虎**：主角看着废，实际深不可测
5. **师徒情深**：师徒之间的温情互动
6. **轻松幽默**：大量搞笑日常

### 差异化设计（与原书不同）
| 维度 | 原书 | 新版 |
|------|------|------|
| 主角身份 | 穿越者+长生系统 | 穿越者+忆海系统（三世轮回） |
| 系统机制 | 徒弟强→师父强 | 收徒解锁记忆碎片 |
| 师父人设 | 懒癌+健忘 | 健忘+话少+外冷内热 |
| 徒弟来源 | 捡来的婴儿 | 捡来的少年（灭门遗孤） |
| 核心矛盾 | 等徒弟 | 忘记自己是谁 |
| 动物 | 鸡/蟑螂 | 蛤蟆/老鼠 |
| 功法 | 劈柴剑法 | 挑水功/扫地掌 |
| 地名 | 桃花山/桃花观 | 忘川山/忘忧居 |
