# 卷纲·第一卷：窗口期（Ch1-60）

## 本卷目标

陈渡从穿越新人到能在渡劫审批科独立处理积案，完成第一个标志性胜利：证明三方会签不是必经程序。

## 爽点节奏

每5章一个中型爽点，每10-15章一个大爽点：

| 章节 | 爽点类型 | 核心事件 |
|------|---------|---------|
| Ch1-3 | 开篇三章 | 考场反杀→贬职机缘→第一个案子 |
| Ch4-5 | 制度突破 | 地府调档成功，批下玄真子 |
| Ch6-7 | 权力反杀 | 天条第十条撤销暂停令 |
| Ch8-10 | 信息差碾压 | 蟠桃园黑账+瑶池特供曝光 |
| Ch11-13 | 权力博弈 | 王母召见（待定） |

## 情绪弧线

穿越的困惑 → 考试的高压 → 被贬的落差 → 发现机缘的希望 → 
首次成功的振奋 → 被打压的压抑 → 反杀的痛快 → 
新危机（断粮）的不安 → 调查的紧张 → 王母召见的恐惧

## 人物弧线

- **陈渡**：从"按规矩办事"到"用规矩打人"——学会主动出击
- **张道陵**：从装睡到睁眼——Ch6 给册子，Ch10 坐起来，Ch15+ 可能重新出山
- **小石头**：从吐槽旁观到主动参与——Ch10 收下抽屉钥匙
- **孙悟空**：从陌生人到自己人——Ch5 批审批→Ch8 送酒站台
