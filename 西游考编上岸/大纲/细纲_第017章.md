# 细纲·第17章：花果山的回信

- **核心事件**：花果山的猴子开始排队来天庭办事；每只猴子拿着一份补充材料，全是毛茸茸的爪印
- **章首钩子**：视觉反差 — 南天门外站着一排猴子，从西天门排到南天门。增长天王站在门口不敢动——上次孙悟空撞塌的柱子还没修好
- **爽点**：猴子们不是来闹事的，是来"办事"的——每只猴子都有一份渡劫申请补充材料。增长天王想拦，领头的老猴掏出天条抄件："你拦一下试试。"
- **章尾钩子**：悬念揭示 — 太白金星路过看了一眼，对陈渡说："赵公明今天在凌霄殿拍了桌子。说花果山的猴子扰乱天庭秩序。明天早朝，他要提紧急议案。"陈渡："那我明天也去。"
- **期待度**：强
- **字数目标**：2100字
- **涉及角色**：陈渡、孙悟空（不出现但幕后）、小石头、太白金星、增长天王、花果山老猴
- **情绪变化**：压抑→猴子的幽默→增长的紧张→太白金星的消息→决定去早朝
