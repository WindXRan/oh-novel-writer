# 细纲·第16章：审计的反击

- **核心事件**：功德司审计处提前完成审计——结论是渡劫审批科的功德收支存在严重问题，要求追回所有功德。但陈渡发现审计报告的漏洞
- **章首钩子**：冲突升级 — 审计处来人，把一份厚达八十页的审计报告放在陈渡桌上
- **爽点**：陈渡翻开报告发现一个致命错误——审计范围包括了渡劫审批科成立之前的时间段。渡劫审批科成立前的东西根本不归他管
- **章尾钩子**：两难抉择 — "这份报告如果上报凌霄殿，你们审计处就是出具虚假报告。"审计仙吏脸色变了。但陈渡不确定要不要赶尽杀绝——把对方逼到墙角，下次对方也会把他逼到墙角
- **期待度**：中
- **字数目标**：2100字
- **涉及角色**：陈渡、审计处仙吏、小石头、张道陵
- **情绪变化**：压力→查看报告→发现漏洞→反击的爽快→要不要赶尽杀绝的犹豫
