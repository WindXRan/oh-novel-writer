# 细纲·第8章：猴子来喝茶

- **核心事件**：孙悟空来喝茶送猴儿酒；太白金星摊牌天庭香火衰减+蟠桃会审计方向
- **章首钩子**：反差场景 — 孙悟空当天晚上就到了，"没撞柱子"，走的是西天门
- **爽点**：猴儿酒泡桂花茶→太白金星亮出一千年香火收支表→"你不是一个人，你有猴子"→"查蟠桃会"
- **章尾钩子**：行动承诺 — 孙悟空让每只猴子都写补充材料，"反正猴子多，写死他们"。小石头在审计通知背面加了一笔——画了个猴子
- **期待度**：强
- **字数目标**：2200字
- **涉及角色**：陈渡、孙悟空、太白金星、小石头
- **情绪变化**：被断粮的低落→孙悟空到来的温暖→太白金星摊牌的震撼→方向明确→团队成型
