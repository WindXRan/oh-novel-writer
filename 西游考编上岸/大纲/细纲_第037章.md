# 细纲·第37章：灰烬里的东西

- **核心事件**：仓库烧了——但陈渡从灰烬里找到了比合同更有价值的东西
- **章首钩子**：意象预示 — 陈渡站在烧塌的仓库前。满地黑灰。风一吹——灰里露出一片玉简。没烧化
- **爽点**：玉简不是合同——是张福海的私人账本。记的不是供应价——是回扣。每年供应合同签完，张福海返还赵公明三成。八百年——累计一百四十二万功德
- **章尾钩子**：隐藏含义 — 账本最后一页有一行字——不是张福海的笔迹。是赵公明的。"福海吾甥：账不可留。阅后即焚。"——他没烧
- **期待度**：极强
- **字数目标**：2100字
- **涉及角色**：陈渡、赵明、太白金星
- **情绪变化**：绝望→希望→震惊→致命证据到手
