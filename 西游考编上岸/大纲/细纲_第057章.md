# 细纲·第57章：赵霆的选择

- **核心事件**：赵霆恢复原身后——面临选择：回雷部？留功德司？还是离开天庭？
- **章首钩子**：反差场景 — 赵霆坐在耳房里。穿的不是雷部旧袍了——是凡间的粗布衣。三百二十年没穿过凡间的衣服。"我要走了。"
- **爽点**：赵霆拒绝回雷部——"我当了二十八年卧底。当够了。"拒绝留功德司——"这地方关了我三百年。"他选择去凡间——"福陵山那头猪——当了三百年。我想当回人。"
- **章尾钩子**：承诺 — 赵霆走到门口。回头——"第三千零一个人——你别急着查。那个人不是敌人。但也不是朋友。等你站稳了——他会来找你。"
- **期待度**：强
- **字数目标**：2000字
- **涉及角色**：陈渡、赵霆、张道陵、小石头
- **情绪变化**：不舍→理解→赵霆告别→留下线索
