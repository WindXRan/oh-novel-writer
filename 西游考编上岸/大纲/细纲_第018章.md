# 细纲·第18章：早朝

- **核心事件**：陈渡第一次参加凌霄殿早朝；赵公明提案限制渡劫审批权；陈渡当庭出示雷部原始报告——功德司篡改审查结论
- **章首钩子**：场景反差 — 凌霄殿早朝，百官分列。陈渡穿着洗得发白的青色官袍站在最后面，旁边的人都在看他——没人认识他
- **爽点**：赵公明念完提案→玉帝问"渡劫审批科的人来了吗"→陈渡从最后面走出来→出示两份雷部报告→全场死寂
- **章尾钩子**：权力反转 — 玉帝看了两份报告，说了四个字："暂缓审议。"赵公明的脸白了。但陈渡知道，他只是赢了这一回合，不是赢了这场仗
- **期待度**：极强
- **字数目标**：2500字
- **涉及角色**：陈渡、赵公明、玉帝（首次出场）、太白金星、满朝文武
- **情绪变化**：紧张→孤身→拿出证据→全场震惊→短暂胜利→清醒
