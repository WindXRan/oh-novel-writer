# 细纲·第15章：雷部的声音

- **核心事件**：陈渡去雷部调报告，发现雷部也不干净——有人收了功德司的好处，在审查报告上动过手脚
- **章首钩子**：场景反差 — 雷部在天庭最东边，门口雷声震天。进去后比渡劫审批科还安静
- **爽点**：雷部值班的雷将是个直性子，"这份报告我见过。原始版本确实是'合格'。后来功德司来函，说审错了，让重审。重审结果就变成了'不合格'。"
- **章尾钩子**：证据到手 — 雷将把原始报告和修改后的报告都复印了一份给陈渡。"但你要想清楚——拿着这两份东西走出去，你就是功德司的敌人。""已经是了。"
- **期待度**：强
- **字数目标**：2000字
- **涉及角色**：陈渡、雷部值班雷将
- **情绪变化**：警惕→雷将直率的意外→拿到证据的振奋→走出雷部的坚定
