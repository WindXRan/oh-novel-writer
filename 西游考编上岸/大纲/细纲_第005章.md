# 细纲·第5章：第一个来办事的神仙

- **核心事件**：玄真子审批批下，雷部震惊；孙悟空带着1300份猴子渡劫申请上门；功德司巡察处来问责
- **章首钩子**：权力反转 — 雷部天将亲自驾云送批复，"四百七十年的积案让一个入职七天的新人批了"
- **爽点**：孙悟空掏出一千三百多份申请→陈渡翻出前任批注"生灵涵盖所有活物"→逐张签字→巡察处被天条怼回去
- **章尾钩子**：悬念揭示 — 玉真子的档案被单独收进抽屉，"此人与玄真子同门，玄真子拒绝赵公明后，玉真子次年被拒"
- **期待度**：强
- **字数目标**：2200字
- **涉及角色**：陈渡、小石头、张道陵（梦话）、孙悟空、巡察处仙吏
- **情绪变化**：被认可→被猴子的信任打动→签字签到手抬不起来→被巡察→反杀→发现新线索
