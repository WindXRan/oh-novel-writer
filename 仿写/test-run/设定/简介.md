# 简介

## 标准版

林穗穗醒了过来，发现自己回到了十八岁——知青下乡的第一天。

上辈子，她嫌周沉穷、嫌他闷、嫌他配不上城里的千金大小姐。直到那个沉默寡言的男人为了保护她死在前线，她才抱着那枚血迹斑斑的铜板哭得撕心裂肺。

这辈子，她一下火车就直奔柳河公社，找到了那个还在土坯房里啃窝窝头的少年。

"周沉，我要嫁给你。"

少年手里的窝窝头掉了，耳根红透，声线却冷得像冰："……你疯了。"

可从此以后，村头兵哥的眼神再也离不开那个城里来的小知青。他以为是自己动了不该动的心，却不知道，上辈子他也曾为她赴汤蹈火、至死不渝。

## 悬念版

你有没有恨过一个人？

恨他穷、恨他闷、恨他像块捂不热的石头。

可有一天他死了——为了保护你死的。

林穗穗抱着周沉的遗物哭了一整夜，醒来时，知青专列刚开进柳河公社。

那个前世冷硬如铁的军装男人，此刻正站在村口，手捧搪瓷缸，面无表情地看热闹。

他还不认识她。

**可他欠她一条命，她欠他一个真相。**

重生一世，她是来还债的。可那个沉默寡言的兵哥，为什么看她的眼神，像是已经等了她很久？

## 甜宠版

柳河公社的糙汉子们最近发现：周沉那小子走了狗屎运。

城里来的漂亮知青林穗穗，今天给他送饭，明天给他补衣裳，后天干脆堵在村口喊"我要嫁给你"。

周沉："……你图我什么？"

林穗穗偷偷想：图你上辈子为我挡的子弹，图你默默攒了一辈子的钱全写了我的名字。

可她嘴上只说："图你长得好看呀。"

冷面兵哥再也绷不住，一把将她搂进怀里，哑着嗓子："……这可是你说的，反悔都没用了。"

前世沉默的守护，今生热烈的奔赴。暗恋两辈子的男人，终于在这一世，收到了那封迟到的回信。
