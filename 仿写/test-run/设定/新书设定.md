# 新书设定

## 书名候选
1. 《七零军婚：重生冷情硬汉少年时》（推荐）
2. 《七零娇宠：重生冷漠丈夫少年时》
3. 《重回七零：嫁给那个沉默的少年》

## 类型定位
- 年代穿越｜重生｜军婚｜男暗恋｜甜宠

## 核心卖点（保留的成功因子）

| 源文成功因子 | 新书保留方式 |
|---|---|
| 重生救赎悲剧丈夫 | 女主穿越到七零，丈夫前世因保护她牺牲，她要改变其命运 |
| 男暗恋（起始好感拉满） | 男主上辈子暗恋女主一辈子，死后才知；重生后初始好感度已满 |
| 性格反差CP（明媚×偏执） | 保留：城里娇气大小姐 × 沉默冷硬乡下兵哥 |
| 身份反转 | 前世：女主天之骄女→家道中落，男主穷小子→部队首长 |
| 亡夫遗恨 | 保留"他为我而死，我才发现爱他"的核心遗憾 |
| 双洁HE | 保留双洁设定 |

## 人物设定

### 女主：林穗穗
- **身份**：城里来的下乡知青，纺织厂厂长之女，娇气会撒娇
- **性格**：明媚活泼，嘴上不饶人但心软，有点小嗲精属性
- **目标**：这辈子让周沉好好活着，别再为她挡枪
- **与原主差异**：源文苏夏是高冷大小姐→改造成更接地气的娇气小太阳；源文开场已结婚→新书开场未婚，更有拉扯空间

### 男主：周沉
- **身份**：七零年代农村兵哥，家里穷得叮当响，沉默寡言
- **性格**：外表冷硬如铁，内心偏执深情，占有欲极强但不外露
- **关键人设标签**：冷硬兵哥｜偏执忠犬｜自卑隐忍｜暗恋成真
- **与原主差异**：源文许霁青是偏执病娇感→改为更有年代感的沉默硬汉；身份从寒门学子→农村兵哥，更符合时代背景

### 关键配角
- **林父**：纺织厂厂长，宠女但重男轻女，反对下乡
- **赵红梅**：同期知青，嫉妒女主，制造矛盾
- **周母**：农村寡妇，老实巴交，喜欢女主但怕高攀
- **陈大壮**：同村混混，觊觎女主，推动冲突

## 世界观设定

### 时代背景
- 1970s，知青下乡运动末期，改革开放前夜
- 城乡二元对立明显，农村生活艰苦

### 地理/空间设定
- **南方小镇柳河公社**：青山绿水，民风淳朴但物资匮乏
- **知青大院**：女主和其他知青的集体住所
- **周家土坯房**：男主家，穷但有烟火气
- **部队驻地**：男主参军后的场景（后期）

### 主要势力/组织
- **柳河公社**：农村基层，大队长说了算
- **知青办**：管理知青分配和返城名额
- **部队**：男主成长线，从新兵到尖子兵

### 核心设定（独特规则/系统）
- 女主是**身穿**重生（非魂穿），保留前世记忆，但身体回到18岁
- 男主**无前世记忆但潜意识有本能亲近**（眼神会不自觉地找她）
- "铜板信物"：前世男主牺牲前留给女主的定情信物，新书中成为关键道具

### 与源文的差异化
- **时代背景不同**：现代校园→七零年代知青下乡，新鲜感强
- **相识方式不同**：源文同桌→新书知青分配同村
- **男主类型不同**：病娇学霸→沉默硬汉兵哥
- **冲突主线不同**：源文是校园+家族→新书是生存+返城+军婚，更厚重
- **情感阻力不同**：源文是身份差距→新书是城乡差距+时代鸿沟
