# 风格指南：第1章

## 写章时叙事声音
用第三人称有限视角（女主POV），语气在刻薄自嘲和脆弱无措之间反复横跳。女主心里话带点赌气的阴阳怪气，但涉及创伤记忆时立刻收住，变成短促的陈述句，不抒情不煽情。写章时保持这种"嘴硬心软"的叙事声音——女主骂自己"没心没肺惯了"，下一段却写她忘不掉那个人的温度。

## 写章时对话写法
- 句子长短：对话用极简短句，催眠/治疗场景的对话控制在2-7个字内，不做任何修饰
- 口语化程度：中（日常对话偏口语，但催眠师的提问要保持职业性的冷淡感）
- 角色区分：催眠师对话是"功能导向的引导式"（"苏小姐，现在我想带你回到一个地方"），女主的回答是"破碎的不完整句"（"……蓝色。"），男主的话是"近乎沉默的单字"（"不会。"）。仿写时让不同角色的对话功能完全错开

## 写章时场景描写
- 详略程度：细腻但有留白。一个场景只挑2-3个感官细节写，不写全。例：写催眠环境只写"躺椅，暗光，温暖蓬松的羊绒毯"三个名词短语，不加视觉细节、不加声音、不加气味
- 环境与情绪：用环境细节外化情绪，不写内心独白。例：写女主的恐惧不写"她很害怕"，写"她漂亮的眼睛通红，抽泣着，竭力地往他怀里钻。手也攥得很紧，一掌心的汗，细腻湿软。"——全部是动作和身体细节，没有直接的情绪标注

## 写章时转折手法
- 场景切换方式：硬切，不加过渡词。从催眠对话直接跳到"许霁青去世三年，苏夏失眠了三年"——句号一断，话题全换。仿写时所有场景转换都用句号或空行完成，不用"然而""这时""突然""她想起"这类过渡语
- 切换节奏：干脆。一章内可以来回切3-4次时空，每次只用一个句号做跳板

## 写章时节奏控制
- 段落长度：长短极端交替，最短1行成段，最长不超过10行。不允许连续3个段落长度相近
- 高潮段：用碎片断句加速。例：坠机段落写"警报声。刺耳的警报声。""喀拉喀拉。"——单句或拟声词独立成段，制造感官冲击。仿写时紧张场景至少插入3个单句段
- 舒缓段：用长句中的逗号拖慢节奏。例：回忆救援那段用逗号串起一串事实（"本来的衣物外裹着一层熟悉的男款防寒服。衣服是穿上去的，拉链拉到最顶"），让读者感觉时间在缓慢流动

## 写章时用词偏好
- 口语化程度：中偏低（女主内心活动偏口语，叙述语言偏书面但有自然瑕疵）
- 修辞偏好：比喻用得克制但精准，多用"像……"的结构，比喻的核心意象经常是"湿冷的/黏腻的/可怖的温柔"方向。例："冰冷而黏湿，像是沉水中纠缠不散的藻丝""他像一把破碎却牢不可破的冰镐"
- 特色高频词：仿写时高频使用"似乎""像是""也许是""不知是……还是……"制造模糊感；用"明明/偏就/倒也"这类口语化转折副词制造人感

## 写章时情绪表达
情绪全部用动作和身体反应外化，不写内心独白式的"她感到XX"。写恐惧时写"瞳孔骤然放大""身体微微颤抖""咽了咽口水"；写愧疚时写"迟来的良心和畏惧互相滋养"这种自嘲式的判定句，而不是直接写"她后悔了"。写到极致悲伤时，用事实陈述代替抒情渲染。例：许霁青死亡的真相不是用煽情语言写的，而是写"除了一双手，许霁青全身的骨头几乎都断了"——一个冰冷的事实，比任何抒情都有力。

## 值得模仿的写法
1. **名词短语开篇造氛围**：写章时用2-3个名词短语独立成段开篇，不交代时间地点人物，直接切入氛围。例："躺椅，暗光，温暖蓬松的羊绒毯。"——仿写时写"炕沿，搪瓷缸，窗纸上结的霜花。"保留技法，换内容。

2. **"事实钉"收束法**：在情绪最浓的时刻，用一个冷静的事实陈述来收尾，不抒情不总结。例：在第1章最后，写完了许霁青用血画SOS、全身骨头断裂的惨烈画面之后，不写任何"她多么感动""她决心补偿"的抒情，只写"他像一把破碎却牢不可破的冰镐，就那样撑着她。"——一个比喻句定住全章。仿写时在高度情绪化的段落末尾，用一个不带情感修饰词的事实比喻句或动作句收住，让读者自己消化情绪。
