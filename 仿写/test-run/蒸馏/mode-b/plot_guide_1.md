# 情节指南：第1章

## 写章目标
写章时完成开局定调——让女主携前世记忆穿越到七零年代，通过她的视角立下"这辈子让他好好活着"的核心使命。

## 写章节奏骨架
- 情绪曲线：写章时从高烧初醒的恍惚层层下拉，经历前世死亡回放，最终在见到少年周沉时收束为决心
- 场景切换次数：写章时做4次场景切换（女主在知青大院醒来→闪回前世死亡场景→切换回当前知青点→村口远眺），切换方式用高烧昏迷/记忆闪回/感官触发
- 段落节奏：写章时按慢热铺垫→闪回爆发→收束蓄势的节奏分配字数（15%→60%→25%）

## 写章钩子布局
- 章首钩子：写章时用感官细节勾起好奇心——"炕热得像烙铁"或"搪瓷缸子磕在木板上的声响"，让读者立刻代入七零年代氛围
- 章中钩子：写章时让前世记忆闪回逐层揭露——"飞机坠落"→"他扑过来保护她"→"他在雪地上用血画SOS"→"他最后叫了她的名字"，每揭露一层推高情绪
- 章末钩子：写章时让女主看到远处田埂上一个沉默少年的背影，心跳如擂鼓——"是他"→"他还活着"→"这辈子她绝不让他死"，用情感承诺卡住读者翻页冲动

## 写章时必须避开的特征（至少2条）
1. 写章时避开"催眠""心理咨询/治疗师"等所有现代心理治疗场景——女主的记忆闪回应由生理性触发（高烧/撞击/极度情绪波动），而非专业催眠
2. 写章时避开"校服""教室""课代表""同桌"等校园场景——首次见到青年周沉的场景在村口田埂或公社大院里，而非教室。女主从记忆回收到现实的对焦方式是通过"泥土味""鸡叫声""粗布衣裳的触感"来完成，而非黑板上的电子钟

## 写章可套用的爽/虐/悬念公式
- 虐点公式：写章时用前世记忆逐层闪回（初见时的冷漠→婚姻中的压抑→死亡瞬间的牺牲→最后的遗言），每层的情绪强度递增，到"他死了"达到顶峰
- 爽点公式：写章时让记忆闪回的最后一层→睁开眼睛，意识到自己回到了过去+他还活着=情绪急转弯，从绝望到希望的爆发
- 悬念公式：写章时让女主带着完整前世记忆醒来，但读者不知周沉是否也有记忆（他没有）——章末女主看到周沉背影时，他转头的一瞬，读者和女主同时屏住呼吸

## 写章时用的叙事技巧
- 信息差设计：写章时让女主知道前世一切（周沉为她而死），周沉此时完全不知她是谁；读者知道女主有记忆但不知道周沉是否有记忆（他没有，这个信息在后续章节才完全确认）
- 情绪操控：写章时用"倒计时"式节奏——从苏醒→回忆→坠机→牺牲→醒来，让读者经历完整的情绪过山车，最后以女主"这一次我一定让你活着"的决心收尾，留给读者暖意而非压抑
- 感官锚定法：写章时用粗糙的棉被触感、搪瓷缸的磕碰声、土炕的烧炕味、公鸡打鸣——每处感官细节都在提醒女主的身体已回到七零年代，强化穿越的真实感
- 视角差制造虐感：写章时利用女主内心对周沉已爱得撕心裂肺（前世记忆），但在章末远眺时，周沉只是淡淡扫了她一眼就转身走了——他还不认识她，这种"我爱你但你不知道"的视角差天然带虐
