# 章纲：第23章

## 第23章 苏家旧伤

- 核心事件：苏彦冷漠离开后，苏念接到母亲电话得知她因想念女儿而生病住院。苏念内疚自责，陆衍舟用"有些人不值得你为他们难过"安慰她。两人驱车前往苏家老宅，却在门口被管家拦下，称苏彦有令不许苏念进门。
- 因果逻辑：苏彦冷漠离开→母亲生病住院→苏念内疚→陆衍舟安慰→驱车前往苏家→被管家拦下
- 触发逻辑：源文盛霄质问→新苏彦冷漠离开；源文母亲脚崴→新母亲生病住院；源文姚管家拦门→新管家拦门
- 节奏模式：紧张(7)→内疚(8)→缓解(5)→失落(6)→希望(4)
- 钩子：管家犹豫片刻后小声说"夫人说...让她进来吧"，苏念能顺利进门吗？
- 爽点来源：被排斥后获得意外接纳+男主温暖安慰=情感释放爽点
- 原创元素：苏家老宅墙上挂着苏念小时候画的全家福，但被苏彦用布盖住了（视觉化呈现兄妹隔阂，源文没有）
