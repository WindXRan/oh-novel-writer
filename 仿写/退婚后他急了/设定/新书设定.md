# 新书设定

## 书名候选
1. 《退婚后他急了》（推荐）
2. 《放手后他却疯了》
3. 《替身她不干了》

## 类型定位
- 标签组合：女频/现代言情/甜宠/追妻火葬场/先婚后爱/契约婚姻

## 核心卖点（保留的成功因子）
| 源文成功因子 | 新书保留方式 |
|---|--|
| 开篇即冲突：订婚宴/婚车事件强钩子 | 保留"订婚宴上未婚夫追白月光"开局，第一页建立强烈情绪 |
| 替身觉醒爽感：女主幡然醒悟不再当备胎 | 保留觉醒核心，但改为"当众被甩"式觉醒，羞辱感更强 |
| 闪婚契约关系：冲动领证→先婚后爱 | 保留契约结婚框架，改为商业联姻式协议（互惠谈判），增加理性博弈感 |
| 火葬场追妻：前男友后悔追妻 | 保留并强化火葬场，前男友从"觉得女主离不开他"到"发现彻底失去后崩溃" |
| 新欢更宠：男主表面不羁实则深情 | 保留"反差宠"模式，男主从冰山到融化，人设改为"禁欲系→宠妻狂魔" |
| 闺蜜吐槽担当 | 保留毒舌闺蜜角色，增加事业线互动，闺蜜+合伙人双重身份 |

## 人物设定

### 女主：林念
- 身份：独立花艺品牌"一念"创始人兼首席花艺师；27岁，孤儿院出身，靠自己打拼
- 性格：外柔内韧，看人通透但重感情；被骗一次后绝不回头，断舍离极快；表面温柔好说话，原则问题寸步不让
- 目标：品牌三年内进入一线商圈；找到真心相待而非权衡利弊的人
- 与人设差异：源文盛惜是暗恋10年的被动型、恋爱脑觉醒；林念是5年正常恋爱被辜负、清醒女主主动选择翻篇——不是"终于不爱了"而是"老娘不伺候了"

### 男主：顾深
- 身份：顾氏医疗集团CEO；30岁，白手起家（父亡母改嫁，靠自己考上名校→创业）；海城公认的冷面阎王
- 性格：表面冷峻疏离、话少气场强；实际极度厌恶被人掌控，内心有被抛弃创伤；一旦认定一个人会毫无保留地宠
- 关键人设标签：禁欲冰山/反差宠/事业脑/控制欲强/守男德
- 与人设差异：源文秦钰是阳光纨绔型、玩世不恭、明骚张口就来；顾深是内敛冷峻型、情绪不外露、闷骚用行动宠——一个负责"说"，一个负责"做"

### 关键配角
- 江城：前男友，江氏集团太子爷，29岁。把女友当家族联姻的"合适选项"，白月光回国后摇摆不定，失去后才疯狂追妻火葬场
- 沈心怡：白月光，豪门千金，珠宝设计师，善解人意型绿茶，表面温柔实则心机
- 叶笙：闺蜜兼工作室合伙人，32岁，时尚买手，毒舌清醒、人间清醒弹幕机
- 顾明城：男主父亲（已故），曾是顾氏创始人；男主母亲改嫁豪门，是男主内心创伤来源
- 林叔：孤儿院院长，60岁，女主的半个父亲

## 世界观设定

### 时代背景
现代都市，2025年前后，海城（滨海一线都会）。数字化社交、高端商业、奢侈品消费盛行。

### 地理/空间设定
- 海城CBD商圈：男女主公司所在地，主要事业线场景
- "一念"花艺工作室：女主品牌根据地，位于老城区改造艺术街区
- 江城老宅：江家祖宅，传统豪门做派，压抑感强
- 顾深公寓：顶层大平层，冷色调极简风，映射男主性格
- 悦澜湾：海城顶级度假酒店，关键情节发生地

### 主要势力/组织
- 顾氏医疗集团：男主执掌，医疗科技巨头，市值百亿
- 江氏集团：前男友家族企业，传统制造业，走下坡路
- 沈氏地产：白月光家族，地产起家，与江家有商业往来
- "一念"花艺工作室：女主品牌，小而美的独立创业

### 核心设定（独特规则/系统）
1. 契约婚姻协议：男主为夺回顾氏控制权需要"已婚"人设，女主为工作室拉投资需要资源背书——双方签订12个月契约婚姻，互不干涉私生活，到期和平离
2. "一物换一物"原则：每次男主帮女主解决商业危机，女主用花艺作品/品牌联名回馈，形成独特的亲密互动模式
3. 火葬场多段递进：江城追妻分三阶段——试探性挽回（被拒）→狼狈下跪（自尊崩塌）→事业破产式追悔（彻底失去）

### 与源文的差异化
- 女主从"被动暗恋"变为"主动选择"：源文盛惜觉醒是被迫的，林念是发现被背叛后主动翻篇
- 相识方式不同：源文是路边晕倒被秦钰捡到；新书是商业晚宴上女主被前男友当众放鸽子，男主开口解围→谈判式契约
- 契约动机不同：源文秦钰是被催婚+帮女主出口气；新书顾深是夺权需要婚姻背书，女主需要资本资源——互利谈判
- CP互动模式不同：源文秦钰主动追、明面宠；新书顾深深情不外露、行动派宠（先冷淡后真香）

| 检查项 | 源文 | 新书（必须不同） |
|--------|------|----------------|
| CP互动模式 | 秦钰主动调戏、明显宠妻 | 顾深前期疏离冷漠、暗中护妻，后期冰山融化、行动宠（做得多说得少） |
| 女主独特身份 | 普通上班族/前未婚妻 | 花艺品牌创始人/手艺人 |

| 男主核心创伤 | 无（单纯被宠坏的纨绔） | 少年丧父、母亲改嫁→信任缺失、渴望被无条件选择 |
