# 情节指南：第10章

## 写章目标
让陆深和林溪有一次近距离、不说话的互动——通过"修自行车链条"这个动作，让陆深第一次对林溪展现出主动的、不留痕迹的照顾，林溪在事后回味中感知到那份笨拙的温柔。

## 写章节奏骨架
- 情绪曲线：从上学路上的平淡→自行车坏掉的烦躁→蹲在路边弄不好的窘迫和无助→陆深出现并蹲下帮忙的意外→他修好就走留下余韵→林溪独自回味的微甜
- 场景切换次数：2次（上学路上→学校附近小路→继续上学）
- 段落节奏：前1/3铺垫（骑车出门→掉链子→自己弄不好）→中间1/3核心互动（陆深出现→修链条→对话）→后1/3余韵（看着他走远→骑车继续上路→进校门）

## 写章钩子布局
- 章首钩子：用"链条掉了"的瞬间声响抓住注意力——咔的一声，脚蹬空转，自行车猛地一冲
- 章中钩子：陆深蹲下来的动作没有问"要不要帮忙"，他直接把书包放地上就伸手了——这种"不问只做"的作风第一次在林溪面前展现
- 章末钩子：林溪在看着陆深走远后，低头看见车锁上多了一小截铁丝——他顺手帮她紧了链条，没告诉她。这一小截铁丝是钩子，暗示"他还会继续帮她"

## 保留功能，换血肉
| 保留的功能（可复用） | 新书必须换掉的具体内容 |
|-------------------|---------------------|
| 女主在某个场景中感到无助/跟不上节奏 | 换场景：蒲雨物理课听不懂方言→林溪上学路上自行车掉链子自己修不好 |
| 男主意外的出现解决了女主的困境 | 换方式：原溯用考试满分+让老师改方言→陆深直接蹲下来动手修链条，一句话不多说 |
| 男主展现"比女主预期的更强"的一面 | 换展现方式：原溯学霸属性（物理试卷满分）→陆深动手能力（修车熟练） |
| 女主对男主产生好奇/改观 | 换改观理由：从"他其实很聪明"→"他好像没那么冷漠" |
| 留下一个"下次还会再有交集"的暗示 | 换暗示方式：原溯答应讲题→陆深说"太松了，回头紧一下"（暗示还会有下次接触） |

## 写章时必须避开的特征
1. 写章时避开任何教室/课堂场景——本章全程发生在上学路上，不进入校园
2. 写章时避开"学霸帮你讲题"的桥段——陆深不是学霸人设，不在本章展露任何智力上的东西
3. 写章时不要写陆深说太多话——本章他的台词应控制在3句以内，且每句不超过8个字
4. 写章时不要让林溪说"谢谢"以外的感谢话——她应该有点拘谨、不知所措

## 写章可套用的爽/虐/悬念公式
- 甜点公式：陆深蹲下来时没有事先问"需不需要帮忙"——他直接行动，这种"不多说只做"的作风本身就是角色的无声表白
- 悬念公式：章末林溪发现车锁螺丝被拧紧了——陆深走之前顺手帮她紧的，但他什么都没说。这个细节让读者和女主一起"愣了一下"，然后慢慢回味
- 虐点公式（轻虐）：林溪蹲在路边弄了十分钟也没弄好，手全是黑油，经过的人没人停下来——建立"没人帮她"的孤立感，衬托陆深出现的分量

## 写章时用的叙事技巧
- 视角锚定：全程跟随林溪的视角，陆深的动作和表情只通过她的眼睛观察——不写陆深的内心活动
- 细节制造余韵：陆深修车时的手指（指节上的创可贴、粗大的关节、指甲缝里的机油印）——他身上永远带着修船厂的痕迹，这是人设的细节锚点
- 动作代替对话：陆深装好链条后站起来，拍了拍手上的油，捡起书包——这一连串动作比说话更有力
- 留白技巧：陆深走之后不要写林溪的内心独白，只写她看到的东西（他混进人流、车锁上的铁丝、自己满手的黑油）——让读者自己填充她的感受
- 感官锚定：贯穿全章的触觉回忆——手上黑油的黏腻感，指缝里的机油味，捏链条时冰凉的金属触感
