# plot_guide_13：台风

## 源文结构（第13章）
- A：蒲雨早起买包子
- B：路上喂许岁然（闺蜜互动）
- C：教室里等原溯，书包里的包子渐渐凉了
- D：原溯来了，两人拉锯（我要放/他说不用/她塞桌斗里）
- E：原溯最后还是吃了，说"两清了"

## 映射逻辑
| 源文节拍 | 新书处理 | 映射类型 |
|---------|---------|---------|
| A 早起买包子 | 台风登陆，雨越下越大，同学陆续被家长接走 | 场景替换：食物准备→天气变化 |
| B 喂许岁然(闺蜜) | 林溪借教室电话打给外婆，外婆说雨大别回 | 情感替代：闺蜜投喂→外婆关怀 |
| C 等原溯来，包子凉 | 等雨停，教室里只剩她一个人 | 等待焦虑：等的人不同，心情相似 |
| D 拉锯：吃/不吃 | 拉锯：一把伞不够/一起走太挤 | 核心推拉：源文→食物；新书→送伞 |
| E 他吃了/"两清了" | 他说"顺路"，半边肩膀湿透，送到门口就走 | 释然余韵：吃完就走→送到就走 |

## 本章功能
- 全书弧线阶段：温暖→微酸（强度6）
- 推进：台风天意外交集，让陆深第一次主动靠近林溪（非偶遇模式）
- 陆深人设强化：嘴上冷淡，行动上"送到底"的作风
- 为后续"陆深看到林溪有危险会出手"做铺垫
- 下雨=全身湿透=不可避免的肢体接近感=暧昧升温

## 情绪曲线
1. 台风天放学（不安）→ 2. 同学陆续被接走（孤单）→ 3. 外婆来电话说别回（失落）→ 4. 教室暗下来，雨声大（焦虑）→ 5. 陆深出现（意外）→ 6. 共伞拉锯（紧张/心跳）→ 7. 路上风雨中靠近（悸动）→ 8. 到家全身湿透（余韵/暖）

## 关键画面
- 窗外雨幕如帘，操场的旗杆被风吹得弯成弧形
- 走廊尽头有人撑伞跑过，水花四溅
- 林溪在空教室收拾书包，走廊脚步声由远及近
- 陆深站在门口，校服湿了一半，手里拎着两把伞（只有一把是自己的）
- 伞不够大，陆深把伞往她那边倾，他右肩完全暴露在雨中
- 风大时伞被吹翻，两人同时伸手去扶，手指短暂碰到
- 到家门口，林溪回头看——陆深半边肩膀全湿透了，还在滴水
- 他说完"到了"转身就走，林溪想说什么，他的背影已经没入雨幕

## 对话要点
- 陆深的台词：很少，每句不超过6个字（"你没伞？"、"走吧"、"顺路"、"到了"、"嗯"）
- 林溪的台词：比陆深多但也不多
- 对话节奏：源文"吃不吃"拉锯 → 新书"走不走"拉锯

## 注意
- 长句不能超65字
- 短句(<15字)至少48句
- 单句成段保持19处左右
- 连接词不超过3个
- 句首多样性要高（不要超过16%的句首重复）
- 不要写陆深的内心活动——全程林溪视角
- 不要写"谢谢你"以外的感谢话
- 陆深说"顺路"的时候不要太自然——他要有一个微不可察的停顿
