# 新设定：更具体、更有趣

## 核心设定

### 主角：林九
- **身份**：前世是魔道宗主，被正道围杀后转世，但转世失败，卡在"半死不活"状态
- **系统**：「续命系统」——每收一个徒弟，寿命延长100年；徒弟死了，寿命减200年
- **人设**：贪生怕死、抠门、话痨、但关键时刻靠谱
- **核心矛盾**：想续命就得收徒弟，但徒弟容易死，死了反而减寿命

### 徒弟：石小满
- **身份**：被正道追杀的魔道遗孤，身上有「魔种」，会被正道修士感应到
- **性格**：倔强、重情义、但命硬（怎么都死不了）
- **核心特点**：每次快死的时候都会莫名其妙活过来（魔种保护）

### 核心爽点
1. **续命焦虑**：师父怕死，但徒弟总作死
2. **反向保护**：师父保护徒弟，徒弟保护师父，互相保护
3. **抠门日常**：师父抠门到极致，但对徒弟很大方
4. **话痨吐槽**：师父话痨，徒弟话少，形成反差

### 差异化设计
| 维度 | 原书 | 新版 |
|------|------|------|
| 主角身份 | 穿越者+长生系统 | 转世魔道宗主+续命系统 |
| 系统机制 | 徒弟强→师父强 | 收徒弟续命，徒弟死减命 |
| 师父人设 | 懒癌+健忘 | 贪生怕死+抠门+话痨 |
| 徒弟来源 | 捡来的婴儿 | 主动找上门的魔道遗孤 |
| 核心矛盾 | 等徒弟 | 怕徒弟死 |
| 动物 | 鸡/蟑螂 | 蛤蟆/老鼠 |
| 功法 | 劈柴剑法 | 挑水功/扫地掌 |
