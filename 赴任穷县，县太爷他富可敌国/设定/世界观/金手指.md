# 金手指：县令银库系统

## 基本规则
每日卯时（早上六点），系统自动向宿主发放白银。

### 发放公式
**安远县在册人口 × 0.5两 = 每日发放白银**

| 当前人口 | 每日到账 |
|---------|---------|
| 28,000人 | 14,000两 |
| 50,000人 | 25,000两 |
| 100,000人 | 50,000两 |

白银直接出现在宿主指定的密室银箱中，凭空生成，外人无法追溯来源。

## 三层限制

### 限制一：用途限定
白银只能用于**县政相关支出**：
- 基建：修路、筑堤、建房、开渠
- 军备：招兵、买马、造兵器、修城墙
- 教育：建学堂、请先生、资助生员
- 民生：赈灾、济贫、买粮种、兴水利
- 商业：开官办工坊、招商补贴、建市场
- 行政：衙门修缮、差役俸禄、办公耗材

不可用于：个人消费、馈赠私礼、贿赂、私蓄。

### 限制二：县域限定
白银只能在**安远县范围内使用**。
- 在本县采购物资 ✓
- 从外地请工匠来本县干活 ✓
- 资助本县人去外地办事 ✓
- 直接在外地消费 ✗
- 转移银两到外地 ✗

### 限制三：当日清零
当日未使用金额次日清零，不累计。
**必须当天花完——花钱本身就是一个挑战。**

## 终极目标
当安远县**在册人口达到100,000人**时，触发"封疆"特别奖励。
