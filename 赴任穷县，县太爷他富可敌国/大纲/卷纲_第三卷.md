# 卷纲_第三卷：势力扩张

> Ch71-120 | 约10万字 | 升级卷

## 本卷目标

雷万钧退场后，苏敬之成为新的核心对手。沈舟的格局从县级升级到州级——拿下驿路、整合产业、办学、应对州府势力。最终苏敬之覆灭，孙守正被边缘化，沈舟成为区域经济引擎。

## 情绪弧线

```
余韵（Ch71-77）→ 新生（Ch78-85）→ 暗流（Ch86-95）→ 爆发（Ch96-110）→ 升级（Ch111-120）
```

## 人物弧线

- **沈舟**：从"县内称雄"到"辐射周边"。面对更强大的对手（苏家），从碾压式消费转向政治智慧和博弈。
- **苏敬之**：从暗线浮出→公开对抗→覆灭。比雷万钧更阴险（不靠暴力靠体制渗透）。
- **方若水**：在州级博弈中发挥更大作用，成为沈舟最倚重的战略参谋。
- **陈巧儿**：管理能力升级，从账房到多产业财务管理。
- **苏晚**：从相亲对象变为真正的盟友（透露苏敬之底牌）。

## 关键反转

**Ch90**：苏敬之表面帮沈舟解决麻烦，实际在设局夺产业。沈舟利用方若水的法律武器+周思远的数据分析，当场揭穿苏敬之的布局，反将一军。
