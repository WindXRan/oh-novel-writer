# 卷纲_第一卷：初入云江

> Ch1-30 | 约6万字 | 建制卷

## 本卷目标

沈舟从一无所有的穿越者，变成拥有核心团队（陈大山、陈巧儿、周思远、方若水）和第一个产业（大通货栈）的云江县知县，与雷万钧完成首次正面交锋并初步立威。

## 情绪弧线

```
困惑（Ch1-3）→ 试探（Ch4-8）→ 发现（Ch9-11）→ 布局（Ch12-20）→ 反击（Ch21-27）→ 蓄势（Ch28-30）
```

## 人物弧线

- **沈舟**：从"穿越了怎么办"的迷茫 → 有了战略方向的务实县令。学会三点：先摸底再动手、用法律武器、建团队而非单打独斗。
- **陈大山**：从"大人别瞎搞"的劝退派 → 主动帮沈舟跑腿打探的执行者。
- **陈巧儿**：从米铺记账的小姑娘 → 月钱八两的县衙大账房，进入工作状态极快。
- **周思远**：从被革职窝在破庙教书 → 重新出山做幕僚，才华得到施展。
- **方若水**：从独立讼师 → 与沈舟建立深度合作，知府关系是关键筹码。
- **雷万钧**：从笑面虎式试探 → 发现沈舟不好对付 → 主动和解实为布局。

## 关键反转

**Ch25**：雷万钧主动请沈舟喝茶和解，注销欠税。看似沈舟赢了，实际雷万钧在试探沈舟的底线和后台。章尾方若水来信："雷万钧下午去了州府，待了两个时辰。"——和解是假的，他在摸底。
