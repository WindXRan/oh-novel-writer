# 卷纲_第二卷：站稳脚跟

## 本卷目标

沈舟完成大通货栈改造开业，商业版图从货栈扩展到多个产业，与雷万钧的全面战争以三线围剿收场，雷万钧众叛亲离退场。苏敬之暗线开始浮出。

## 情绪弧线

```
蓄势（Ch31-35）→ 扩张（Ch36-42）→ 受阻（Ch43-50）→ 破局（Ch51-58）→ 压制（Ch59-67）→ 碾压（Ch68-70）
```

## 人物弧线

- **沈舟**：从"有第一个产业"到"有商业体系"。学会用利益捆绑团队、用法律约束对手、用银子碾压一切阻碍。
- **雷万钧**：从嚣张试探 → 多线受挫 → 众叛亲离 → 被迫退场。最后的疯狂反扑被三线截断。
- **方若水**：从法律顾问升级为战略参谋，知府关系在关键时刻发力。
- **赵守成**：Ch43前后自荐入伙，成为货栈运营核心。
- **苏敬之**：只在暗线中出现（酒局、信件、代理人），为第三卷铺垫。

## 关键反转

**Ch60**：雷万钧联合州府力量做最后反扑，计划从三个方向打压沈舟。但沈舟提前布局——陈志远反水交出行贿录音、方若水锁死法律路径、赵守成切断商业命脉。三线同时收网，雷万钧措手不及。
