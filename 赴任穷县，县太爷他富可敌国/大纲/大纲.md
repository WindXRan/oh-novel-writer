# 全书大纲

> 对标：《每天六千万，只能在县城花？》（全书198章深度拆解版）
> 题材：架空古代·县令基建·系统爽文
> 平台：番茄小说
> 预计：198章，约40万字

---

## 核心设定

### 金手指
- 系统：县令银库系统
- 公式：在册人口 × 0.5两/日，当日清零不累计
- 限制：仅限县政用途，仅限云江县
- 终极目标：10万人口→触发"封疆"特别奖励

### 节奏模式：「进攻-受阻-破局」三拍循环

```
进攻（2-5章）：沈舟提出目标、部署行动
受阻（2-4章）：反派/官僚/意外设置障碍
破局（1-3章）：用银子碾压/法律武器/政治智慧解决
```

每个循环约5-10章，全书约25-30个循环。

### 反派洋葱模型（四层递进）

```
第一层：雷万钧（盐铁地头蛇）——Ch21-77
第二层：苏敬之（州府世家/官盐走私）——Ch74-110
第三层：孙守正（州刺史/体制内腐败）——Ch45-134
第四层：刘昌平（邻县知县/竞争者）——Ch148-198
```

每打掉一层，露出下一层更强大的敌人。保证198章冲突不重复。

### 数字节奏（爽感锚点）

| 阶段 | 银两量级 | 代表数字 |
|------|---------|---------|
| Ch1-10 | 十四万两/十天 | 14,000两/天、300两（粮种农具）、600两（驿站） |
| Ch11-30 | 数十万两 | 1,000两（大通货栈）、12两（欠税暗雷）、20两（讼师费） |
| Ch31-70 | 百万两级 | 竞拍保证金、改造费用、运营资金 |
| Ch71-120 | 数百万两 | 驿路投资、州府博弈金额 |
| Ch121-198 | 千万两级 | 路网、县学、医馆、跨县投资 |

## 卷级大纲

### 第一卷：初入云江（Ch1-30，约6万字）
- 功能：建制卷——人设+班底+第一个项目+亮出反派
- 核心事件：绑定系统→摸排县情→买物资→修驿站→见知府→招幕僚→收购大通货栈→对抗雷万钧→酒局亮底牌
- 起始状态 → 结束状态：孤身知县 → 有了班底+第一个产业+和雷万钧正面交锋

### 第二卷：站稳脚跟（Ch31-70，约8万字）
- 功能：对抗卷——与雷万钧全面战争+商业版图扩张+官场关系建立
- 核心事件：货栈开业→竞拍地块→雷万钧挖坑→省府视察→饭局争锋→三线围剿→雷万钧众叛亲离
- 起始状态 → 结束状态：初步站稳 → 商业版图初成+雷万钧势力瓦解

### 第三卷：势力扩张（Ch71-120，约10万字）
- 功能：升级卷——反派升级（雷万钧→苏敬之）+格局升级（县级→州级）+核心里程碑
- 核心事件：雷万钧退场→苏敬之浮出→驿路争夺→办学→收购产业→苏敬之覆灭→上面来人
- 起始状态 → 结束状态：县内称雄 → 辐射周边+州府格局

### 第四卷：县域崛起（Ch121-198，约16万字）
- 功能：收官卷——权力更替+跨县博弈+民生工程+封疆奖励
- 核心事件：新知府空降→全域路网→县学→医馆→跨县虹吸→刘昌平归附→十万人口→封疆
- 起始状态 → 结束状态：十万人口达标 → 触发"封疆"奖励

## 七条副线

1. **反派更迭线**：雷万钧(Ch21-77) → 苏敬之(Ch74-110) → 孙守正(Ch45-134) → 刘昌平(Ch148-198)
2. **政商关系线**：无根基 → 互利合作 → 双核心 → 主导地位
3. **商业帝国线**：粮种 → 货栈 → 盐铁 → 驿路 → 路网 → 手工业
4. **核心团队线**：孤身 → 陈大山 → 陈巧儿+周思远 → 方若水 → 赵守成 → 顾怀远
5. **县域竞争线**：独善其身 → 邻县冲突 → 虹吸效应 → 邻县依附
6. **舆论民心线**：百姓观望 → 尝到甜头 → 民心所向 → 舆论危机 → 朝廷正名
7. **家庭个人线**：穿越迷茫 → 立足 → 亲情考验 → 权力考验 → 封疆大吏
