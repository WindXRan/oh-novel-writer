# story-blurb · 网文书名+简介生成

## 功能

根据源文设定和新书设定，生成：
1. 网文书名（不与源文对标）
2. 网文化简介（口语化、有节奏、有钩子）
3. 封面（可选，需OpenAI key）

## 核心原则

### 简介风格要求

**必须做到**：
- 口语化，像真人写的，不是AI生成的
- 短句+长句交替，有节奏感（短句打节奏，长句铺信息）
- 有钩子（悬念/反转/冲突），结尾必须留悬念
- 有标签（开头亮标签：【穿书+年代+偏执男主】之类）
- 人称正确（与源文一致，通常第三人称）
- 200-300字，不要太长
- 结尾留悬念，不要剧透结局
- **代词密度控制**：100字内"她"≤3、"他"≤3

**必须避免**：
- AI感（"悬着的心终于死了"、"好家伙"、"两眼一黑"、"这波操作"等网络烂梗）
- 太端着、太文艺（"她心想"、"命运的齿轮开始转动"）
- 太长、太啰嗦
- 与源文简介结构雷同
- 与源文书名对标
- 剧透结局
- **代词轰炸**：连续出现"她/他/她/他"，读起来像乒乓球来回弹

**代词密度控制技法**（简介最常踩的坑）：
- **用名字替代**：不说"他扣住她的手腕"，说"魏长河扣住她的手腕"
- **省略主语**：不说"她想要什么，他就不择手段"，说"想要什么，魏长河就不择手段"
- **用称呼/外号轮换**：那男人、姓魏的、继兄、黑市商人……交替使用
- **合并动作**：不说"她利用完想踢开他，他扣住她的手腕"，说"利用完想踢开人，手腕反被扣住"
- **改写句式**：把"她……他……她……他"的对仗句拆掉，换成独立短句或动作描写

**简介结构公式**（5种钩子类型，每版选一种）：
1. **悬念型**：穿书/重生背景 → 原著命运 → 觉醒 → 男主出场 → 章末金句
2. **冲突型**：标签句 → 三角关系 → 核心矛盾 → 男主占有宣言
3. **情感型**：场景切入（冬天/雨天/深夜） → 情感记忆 → 双向拉扯 → 暗示执念
4. **反转型**：背景 → 困境 → 男主递条件 → 发现门被焊死
5. **口语型**：吐槽式开场 → 现实打脸 → 男主出场 → 对话收尾

### 书名风格要求

**必须做到**：
- 网文感强（参考番茄/七猫热榜）
- 有记忆点
- 不与源文书名撞梗

**必须避免**：
- 与源文书名相似度过高
- 太文艺、太小众
- AI味书名（"重生之XX"、"我在XX当XX"、"被XX后我XX了"等模板句式）

**书名公式**（混搭出新意）：
1. **人设+处境**：《七零娇气包被疯狗盯上了》《年代文作精她不想当炮灰》
2. **动词+悬念**：《回城后，黑市大佬赖上我》《重生七零，疯狗男主他非要负责》
3. **称呼+关系**：《穿书七零：偏执大佬的掌心娇》《继兄他不肯放手》
4. **反差感**：《炮灰女配她躺赢了》《作精在年代文当团宠》

## 流程

### Step 1：分析源文

读取源文信息：
- 书名
- 简介
- 核心设定（穿书/替嫁/重生等）
- 男女主人设
- 核心冲突
- 大结局方向（HE/BE/开放式）

### Step 2：分析新书

读取新书设定（concept.md）：
- 核心设定差异
- 男女主人设
- 核心冲突
- 目标读者
- 大结局方向

### Step 3：生成书名

生成5个候选书名，要求：
- 不与源文书名撞梗
- 网文感强
- 有记忆点

### Step 4：生成简介

生成5个候选简介，要求：
- 人称与源文一致（通常第三人称）
- 开头亮标签
- 短句+长句交替
- 有钩子（悬念/反转/冲突）
- 200-300字
- 口语化，像真人写的
- 结尾留悬念
- **代词自检**：100字内"她"≤3、"他"≤3，超标则用名字/省略主语/句式改写降下来

### Step 5：生成封面（可选）

调用 story-cover skill 生成封面prompt。

## 简介风格参考

### 好简介特征（源文风格）

```
【已签实体】
[偏日常/细水长流/先同居后爱/双洁/甜宠]

一朝穿越，宋云绯成了假冒失忆总裁女友的女配。
原书中，女配不小心开车撞了宴金集团的总裁，因怕坐牢，趁他失忆假冒他的女友。
不仅骗他是个穷小子，还带着他去了偏远城市躲起来，对他各种PUA。
后来男主被家人找到，男主刚恢复记忆，结果未婚妻要跟别人结婚了，他开始追妻火葬场。
至于宋云绯这个恶毒女配，男人秋后算账，不仅逼迫她打掉孩子，还坐了五年牢，出狱后又惨死在街头。
穿越过来的宋云绯一睁眼，看着刚送外卖回来的男人，两眼一黑，悬着的心终于死了。
没办法，宋云绯只能将错就错。
一边跟楚靳寒过着贫穷的日子，一边准备跑路。
然而，终于等到跑路那天，男人却愤愤地将她抓了回来。
"骗完我就想跑？"
宋云绯震惊，好家伙，这人什么时候偷偷恢复记忆的？
```

**特点**：
- 口语化，有网感
- 短句+长句交替
- 有钩子（"骗完我就想跑？"）
- 有反转（男主恢复记忆）
- 结尾留悬念

### 坏简介特征（AI感太重）

```
陈茉替姐出嫁那天，顾衍恨她恨得咬牙切齿。
三年后，他车祸失忆了。
忘了恨，忘了她是谁，每天冷着脸给她做饭、接她下班、雨天骑电动车来接她。
陈茉心想：完了，这人失忆后比失忆前还可怕。
```

**问题**：
- "陈茉心想"太文艺
- "完了，这人失忆后比失忆前还可怕"有点AI感
- 缺少具体细节

### 坏简介特征（代词轰炸）

```
宋明锦谁都不想嫁。继兄严卫东老实但无趣，黑市商人魏长河疯得让人害怕。可魏长河比任何人都懂她——她想要什么，他就不择手段捧到她面前。当她利用完想踢开他时，他扣住她的手腕贴在自己脸上："宋明锦，我这条命是你捡的，你就得负责到底。"
```

**问题**：
- 7个"她/他"密集排列，读起来像打乒乓球
- "她想要什么，他就不择手段捧到她面前"——3个代词挤一句
- "当她利用完想踢开他时，他扣住她的手腕"——又是3个

**修改后**：
```
宋明锦谁都不想嫁。继兄严卫东老实但无趣，黑市商人魏长河疯得让人害怕。可魏长河比任何人都懂她——想要什么，姓魏的就不择手段捧到面前。利用完想踢开人，手腕反被扣住贴在他脸上："宋明锦，我这条命是你捡的，你就得负责到底。"
```
- 代词从7个降到2个，节奏更利落
