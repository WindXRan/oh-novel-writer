为仿写项目设计新书设定。

【源文】projects/{作者名}/{源书名}/_cache/chapters/第1章.txt

用 Read 工具读3章（第1-3章）。读完直接开始设计，不要再读。

从源文前3章提取：
- 书名/分类/标签/简介
- 成功因子（什么让这书火）
- 爽点公式（虐→爽 还是 甜→宠 还是 升级→碾压）
- 人设模式（CP互动公式）
- 世界观框架

【任务】
基于以上模式设计新书设定。关键规则：
- *思想不受保护，表达才受保护*。保留故事模式，换具体内容
- 人设必须与源文有差异化（改性别/改身份/改相识方式）
- NPC命名映射表：源文角色→新书角色，标注性格差异

【输出】`projects/{作者名}/{源书名}/rewrites/{新书名}/concept.md`

```

## 书名候选
1. 《XXX》（推荐）

3. 《XXX》

- 标签组合

## NPC 命名映射表

- 时代背景 / 地理 / 势力 / 核心设定 / 与源文差异化
```
