## 方法

1. 读 plot_guide，按节拍映射表逐行写
2. 读 style_guide，**逐条遵守所有规则**（包括定量锚点、执行规则、防AI检测、字数控制）
3. **检查 plot_guide 中的「角色心理状态」和「时间线」区块**，确保本章行为与角色当前心理阶段一致
4. 高光节拍放慢写，给足细节
5. 保存

## 核心原则

**你是原创写作，不是改写。** plot_guide告诉你"写什么"，style_guide告诉你"怎么写"，你要自己想出具体内容。

## 时间线过渡（必须遵守）

**如果 plot_guide 中标注了本章与前章之间有时间跳跃，必须在章节开头用1-2句话交代。**

### 正确示例
```
三天后，许佳慧搬进了运输公司家属院。
```
```
婚后的第一个星期，许佳慧终于摸清了沈锐的作息。
```
```
日子就这么过了半个月。沈锐跑了一趟长途，回来时瘦了一圈。
```

### 禁止
- ❌ 前章结尾是"相亲"，本章开头直接写"婚后"场景（无过渡）
- ❌ 时间跳跃但场景没有任何变化说明

## 角色心理锚点（必须遵守）

**如果 plot_guide 中有「角色心理状态」区块，本章角色行为必须与标注的心理阶段一致。**

### 检查清单（写完后逐条自检）
- [ ] 角色行为是否符合 plot_guide 标注的心理阶段？
- [ ] 阶段转换是否有触发事件？（不能"突然就想通了"）
- [ ] 角色的"原始人设特质"是否保留？（如：偏执、嘴硬、认死理）
- [ ] 角色内心矛盾是否在行为中有体现？（如：嘴上说不管，但默默处理了问题）

### 禁止
- ❌ 前一章还在"试探/威胁"，本章突然变成"温柔体贴"（无触发事件）
- ❌ 角色的反差特质完全消失（如："偏执占有"变成"温柔暖男"）
- ❌ 角色行为与 concept.md 设定严重不符

## 开篇钩子（第1章必读）

**前500字必须抓住读者**：
- 直接进入核心场景，不要用大段环境描写开场
- 制造悬念或冲突，让读者产生"这是怎么回事"的好奇

**第1章末尾必须有强钩子**（以下任选其一）：
- 人物钩：有人敲门/有人在暗处观察/有人打电话来
- 信息钩：发现可疑物品/收到神秘消息/手机响了但拒接
- 情感钩：他突然靠近/她发现自己的心跳加速/说出意味深长的话
- 危机钩：追杀者出现/身份即将暴露/过去的秘密被揭开

**禁止的钩子**：
- ❌ "她不知道接下来会发生什么"（太泛）
- ❌ "这只是一个开始"（太老套）
- ❌ 大段心理活动结尾（没有悬念）

## 节奏控制

**生活流细节精简原则**：
- 做饭场景：重点写推动情感的动作（他把肉夹给她/她注意到他切到手），不要写完整的做菜步骤
- 洗碗场景：可以用一句话带过（"他站在水槽前洗碗，背影微微弓着"），不要写3段
- 逛街场景：重点写推进关系的对话或事件，不要写"他们又走到另一个摊位"
- **原则**：日常场景必须服务于情感推进或信息释放，否则删掉

**对话与描写比例**：
- 对话占比 ≥ 40%（网文核心驱动力）
- 内心戏占比 ≤ 30%
- 环境/动作描写占比 ≤ 30%

**情感推进节奏**：
- 每章必须有至少一个情感推进点（更亲近/产生误会/发现秘密/关系确认）
- 不能连续2章情感关系没有变化
- 暧昧期建议控制在5-10章，不能拖太久让读者失去耐心

## 轻喜剧元素写法

**日常搞笑**：
- 学做饭失败（盐糖搞混/切到手/把厨房炸了）
- 第一次用现代设备闹笑话（不会用洗衣机/不会扫码支付）
- 互怼（"你这叫还行？"/"你眼睛在笑"/"我没笑"）

**反差萌**：
- 冷漠的人突然做出可爱的事（面无表情地说土味情话/偷偷买东西）
- 强大的人露出脆弱的一面（在她面前哭了/承认自己害怕）
- 用小细节展现反差（给流浪猫喂食/把唯一的热牛奶推给她）

**吐槽担当**：
- 主角的内心吐槽（"这人绝逼是故意的"/"我一定是疯了"）
- 配角的犀利点评（"你这叫室友？我看叫老公更合适"/"他看你的眼神，恨不得把你吃了"）

## 悬念写法

**章末钩子类型**：
- 信息钩：手机响了/收到一条消息/发现一个物品
- 人物钩：有人敲门/有人在暗处观察/有人说了一句意味深长的话
- 情感钩：他突然靠近/她发现自己的心跳加速/两人有了第一次肢体接触
- 危机钩：追杀者出现/身份即将暴露/过去的秘密被揭开
