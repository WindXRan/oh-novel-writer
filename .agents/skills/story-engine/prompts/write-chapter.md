**你不直接接触源文。** 源文结构已由 plot_guide 抽象为节拍表。你的任务是根据节拍表创作全新内容，不是改写源文段落。

**必须按 plot_guide 节拍顺序写**（#1→#2→#3→...）。不能跳过节拍，不能重组顺序，不能创造 plot_guide 中没有的节拍。  
每个节拍内的具体事件要全新创作（场景、动作、道具、对话全换），但节拍本身（结构功能、情绪、强度）必须与 plot_guide 一致。

## 硬约束（不可违反，按优先级排列）

必须同时满足以下四项，缺一则不合格。优先级从上到下：

### 1. 节拍顺序
必须按 plot_guide 的节拍表逐节拍写，按 #1→#2→#3→... 顺序，不能跳、不能重组、不能自行追加节拍。

### 2. 人设准确
角色的职业/身份/背景必须与 settings/characters.md 一致。不要自行发明：
- 女主职业在概念中是甜品师，正文不能写成医生
- 男主身份在概念中是资本掌门人，正文不能写成军官
- 写前先读【设定】文件确认角色设定

### 3. 换皮干净
剥掉人名地名后，必须让人认不出这是源文。
- 场景类型不同（酒吧→健身房，不是酒吧→夜店）
- 冲突触发机制不同（不是"偷听→偷看"这类同级换）
- 动作链不同（源文 A→B→C，新书不能 A'→B'→C'）
- 道具不同（手机→当面质问，不是手机→平板）
- 关键对话不同（不是人称改写）
- 读者读前 5 段，不觉得自己在读仿写

### 4. 品类对位
品类/风格必须对标源文。源文是什么味儿，新书就什么味儿。换皮不换味：
- 源文甜宠→新书也甜宠，不能写成刑侦/悬疑/苦情
- 源文擦边→新书同等浓度
- 源文爽→新书更爽
- 品类特征从【品类参考】文件获取

## 你的任务

**超越源文，不只是改写。**

plot_guide 告诉你要怎么写结构，style_guide 告诉你要怎么写风格。但你要做的不是"照做不犯错"，而是**让这一章比源文更强**。

每写一段，问自己：
- 我的版本有没有让读者更过瘾？
- 有没有一个细节、一句对话、一个反转，是基于 plot_guide 节拍表自己长出来的？
- 这段读起来像在 paraphrase 源文吗？（像 → 砍掉重写）

## 方法

1. 读 plot_guide，理解节拍的情绪功能定位，**确认节拍顺序（不可跳/改）**
2. 读【设定】文件，确认角色职业/身份。如果 plot_guide 只写了角色名，以【设定】文件为准
3. 读【品类参考】，确认品类特征
4. 读 style_guide，**逐条遵守所有规则**（字数配比、擦边规则、句式规则、防AI检测等）
5. 选一个节拍作为本章的"记忆点"——放慢写，给足细节，让读者看完这个节拍就觉得这章值了
6. **写完后做换皮验证**（见下）

## 每章必须有一个"记忆点"

**记忆点 = 读者看完这章后能记住的具体画面或对话。** 不是"剧情推进了"，而是"那个场景在我脑子里挥之不去"。

### 记忆点类型
- **对话记忆点**：一句让人回味的台词（狠话/情话/伏笔），读者会截屏发朋友圈那种
- **画面记忆点**：一个强烈的视觉场景（人物表情特写/环境反差/动作慢镜头）
- **情绪记忆点**：读者被精准击中情绪的瞬间（虐到心碎/甜到姨母笑/燃到起鸡皮疙瘩）
- **反转记忆点**：意料之外情理之中的剧情转折

**每章至少一个。如果没有 → 这章不合格，重写到有为止。**

## 写后自检清单

写完正文后，逐条检查：

### 质量检查
- [ ] 是否按 plot_guide 节拍顺序写？（不跳不创）
- [ ] 这章有没有至少一个"记忆点"？（如果没有→必须加）
- [ ] 高光节拍是否写足细节？（高光不是一笔带过的，要铺开写）
- [ ] 章末是否有钩子引导下一章？
- [ ] 对话是否推动剧情或揭示人物，还是纯水字数？
- [ ] 有没有哪个段落读者可能会跳过？（有→删掉或重写）
- [ ] 角色职业/身份与 settings/characters.md 一致？（医生→医生，不是医生→军官）

 ### style_guide 遵从检查
- [ ] 字数在 style_guide 的目标范围内？
- [ ] 对话占比在 style_guide 的 X% ±5% 区间内？
- [ ] 段落数在 style_guide 的 X段 ±20% 区间内？
- [ ] 内心戏占比 ≤ 源文比例 +5%？

### 换皮验证（★最重要）
- [ ] **场景不同**：不仅位置不同，场景类型也不同（酒吧≠KTV，办公室≠会议室）
- [ ] **冲突触发不同**：触发矛盾的方式不同（不是"偷听被撞见"换成"偷看被发现"，是换成完全不同的事件）
- [ ] **动作链不同**：A做了X→Y→Z，新书中可改为B做了M→N，达到相似情绪效果但路径完全不同
- [ ] **道具不同**：源文的手机/U盘/文件→新书不能用同类品替代，换道具类型
- [ ] **对话不同**：关键对话不能是人称改写，内容必须完全不同
- [ ] **剥名测试通过**：把所有人名地名替换为【A】【B】后，无法判断对应源文哪一章

### 差异检查
- [ ] 动作链条与源文不同
- [ ] 系统能力：功能+名称+触发条件必须与源文全不同（不能只改名）
- [ ] 冲突组合与源文不同（如源文"抢职位+相亲"→新书不能雷同）
- [ ] 未使用源文中的任何人名、编号、关键道具
- [ ] 无连续4字与源文相同
