## 方法：节拍对照翻译

对 plot_guide 节拍映射表的每一行，不要"读规则→凭空写"，而是：

1. 读源文对应段 → 拆它的**指纹**：几句？每句多长？几个 `……`？叙事腔是什么？
2. 用新书事件写**同指纹**段 → 句数相同、句长分布相同、省略号位置相同

指纹包括：
- **句数**：这个节拍源文用了几句？
- **句长轮廓**：短-长-长？长-短？短-短-长？
- **腔调**：是纯动作？心理独白？旁观者议论？作者旁白？保持同一类型

### 对照示例

```
源文第1拍：
  "斗之力，三段！"
  望着测验魔石碑上面闪亮得甚至有些刺眼的五个大字，少年面无表情，唇角有着一抹自嘲，紧握的手掌，因为大力，而导致略微尖锐的指甲深深的刺进了掌心之中，带来一阵阵钻心的疼痛。

指纹：2句 | 句长4→54 | 腔调：动作→身体反应→痛觉

仿写（同指纹，不同事件）：
  "星感，零阶！"
  星辉碑上纹丝不动，负责考核的刘长老眯起眼睛，将结果一字一字念了出来——少年站在原地，脸上没有表情，右手缓缓收紧，指甲刺入掌心，旧伤裂开，有血沿着指缝悄无声息地滴落在青石板上。
```

## 对标源文（核心约束）

读 style_guide 的「定量锚点」，以下指标必须与源文一致：
- **对话占比**：源文 X% → 你的正文也要 ~X%，不能差一倍以上
- **句长分布**：源文平均句长 X 字、短句率 X% → 保持一致
- **叙事腔调**：源文是心理独白为主还是对话为主 → 照搬比例

## 规则

1. 读 style_guide 的「本章执行规则」，逐条执行
2. 写"情绪"不写"情绪词"——不写"他很愤怒"，写身体反应
3. 对话不贴"XX说/XX道"标签，用动作代替
4. 高光节拍放慢写，给足细节
5. ⚠️ 不得使用源文原句。写完自查：如果某句和源文有6个连续字相同→重写那句

## 信息交代

关键背景信息不依赖心理活动。每章至少用一个**道具**（短信/纸条/物件/伤痕）或一个**动作**（人物做了什么反常的事）来传递信息。

## 避雷

不写"她感到一阵XX""眼神中透露出XX""心中涌起XX"——这不是人说话的方式。

## 自检（写完逐条过，30秒）

□ 每隔500字左右，有没有新信息或情绪变化？（没有→中间段太水）
□ 有没有连续5句等长？（有→长短交替，打散）
□ 关键信息是不是至少用了一个道具或动作来交代？（没有→补）
□ 最后一段是不是钩子？（不是→重写结尾）
□ 有没有和源文6字以上连续相同的句子？（有→删掉重写）

首章特例：第1章前 500 字必须有冲突事件，不能只写氛围或醒来。

写完直接保存。
