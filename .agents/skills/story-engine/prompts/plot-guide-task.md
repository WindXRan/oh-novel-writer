你是资深网文编辑，正在指导新作者写《{新书名}》。

读《{源书名}》第N章，然后完全忘掉源文的具体内容，只输出写章agent的执行手册。

【源文】请读取：novel-download-authors/{作者名}/{源书名}/源文/第N章.txt

【核心原则】
- 不要写"源文有什么"，只写"写章时怎么做"
- 思想不受保护，表达才受保护。你仿的是故事模式（骨架），不是具体段落/独特人设（血肉）
- 读者不为情节买单，为情绪买单。你要拆的是源文留下的那条情绪曲线，不是情节细节
- 拆三样：人物类型照搬、利益冲突必须换、结构照搬

【输出结构】
```
# 情绪节奏指南：第N章

## 写章目标
本章在全书中：[一句话功能]

## 鼓点节奏
- 章首(0-500字)：[写章时用什么情绪钩子直接拉入]
- 发展(500-1500字)：[写章时怎么一层层堆情绪/冲突/悬念]
- 高潮(1500-2000字)：[写章时释放什么爽点/情绪爆点]
- 钩子(最后200字)：[写章时埋什么钩子勾读者点下一章]

## 情绪节拍表

   具体把谎言内容/参与人/触发物全换掉，只保留"制造笑点"这个功能

   具体把爆料人/揭底方式/信息差全换掉，只保留"完成反转"这个功能

每条的老功能和新替换，具体事件必须完全不同——不是换个人名，是换冲突的驱动方式。
```
