## 换皮检验 ⚠️（必须逐项自检）

**第一步**：分别写源文和新书的一句话概括（剥掉人名地名）
```
源文概括：[谁]因为[什么冲突]，做了[什么事]，结果[怎样]
新书概括：[谁]因为[什么冲突]，做了[什么事]，结果[怎样]
```

**第二步**：逐项对比骨架
| 对比项 | 源文 | 新书 | 是否同构？ |
|--------|------|------|-----------|
| 冲突本质 | XX冲突 | XX冲突 | 必须不同 |
| 事件结构 | A→B→C | A→B→C | 必须不同 |
| 人物关系模式 | X关系 | Y关系 | 必须不同 |
| 经典桥段 | XX桥段 | YY桥段 | 必须不同 |

**桥段检查**：如果源文使用了经典桥段，新书必须用完全不同的桥段替代。常见经典桥段包括但不限于：
- 开篇醒来、拒绝亲密、睡中拥抱、醉酒照顾
- 英雄救美、误会分手、失忆重逢
- 骚扰解围、借车困境、房屋漏水
- 雨中送伞、电动车暧昧

**场景设计检查**：
- 源文用过的场景类型，新书尽量换
- 如果必须保留同类场景，具体事件和交锋方式必须完全不同

**第三步**：台词比对
- 逐句比对源文和新书的台词
- **6字以上连续匹配 = 违规**，必须重写该台词
- 即使换了人名，句式结构相同也算违规

**第四步**：判断
- 3项骨架全部不同 + 台词无雷同→合格
- 任意1项同构或台词违规→换皮，重做该节拍

**违规示例**：
- 源文"母亲担心早恋→发现打架→介入→女孩装可怜→母亲反思"
- 新书"母亲担心成绩→发现争吵→介入→女孩装无辜→母亲反思"
- 冲突本质同构（都是身份冲突），事件结构同构（担心→发现→介入→装→反思）→换皮，重做

**台词违规示例**：
- 源文：「"你怎么能这样对妈妈说话？"」
- 新书：「"你怎么能这样跟妈妈讲话？"」
- 6字连续匹配（"你怎么能这样"）→违规，重写

## 角色高光时刻 ⭐（每章至少一个）

每章必须设计至少一个让人记住的场景。从以下类型中选 ≥1 个，标注发生在哪个节拍：

| 类型 | 说明 | 示例 |
|------|------|------|
| **甜** | 两人之间不经意的心动瞬间 | 他把她做的糊鸡蛋全吃了；她睡着了他给她盖被子 |
| **虐** | 不得不隐瞒/伤害对方的时刻 | 嘴上说分手，手在发抖；骗他出门，自己在屋里哭 |
| **笑** | 轻松幽默的桥段 | 女主逞强做饭差点烧厨房；男主面无表情说土味情话 |
| **反差** | 角色做出人意料的事 | 一直冷漠的人突然护短；一直怂的人突然硬刚 |
| **细节** | 一个小动作/物件承载情感 | 他把唯一的热牛奶推给她；她偷偷存着他写的小票 |
| **打脸** | 被嘲笑后实力碾压 | 众人嘲笑废物，主角一招震全场 |
| **装逼** | 低调展示实力 | 淡淡一句"还行"，实际碾压所有人 |
| **绝境翻盘** | 濒临失败时逆转 | 快死时觉醒新能力/援兵赶到 |
| **推理突破** | 关键线索串联 | 众人迷惑时主角看穿真相 |
| **热血** | 激发斗志的宣言/行动 | "这一战，我接了！" |

**格式**：类型 + 发生在第几个节拍 + 一句话描述 + 为什么读者会记住

```
## 本章高光
- 类型：甜 | 节拍#5 | 陆止渊把她做的糊鸡蛋全吃了，说"还行" | 读者记住：嘴上嫌弃身体诚实
```

## 爽点要求 ⭐（网文核心）

**每章必须有至少一个"小胜"或"钩子"**：
- **小胜**：女主/男主在本章获得的小胜利（怼人成功、展示实力、获得信息、化解危机）
- **钩子**：章末悬念，让读者想看下一章

**节奏要求**：
- 内心戏不超过本章30%，其余必须是对话/动作/外部冲突
- 女主必须有主动性（做决定、采取行动），不能全程被动挨打
- 每3章至少一个小高潮（打脸/反转/情感爆发）
