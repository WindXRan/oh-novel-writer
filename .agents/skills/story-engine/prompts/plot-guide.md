## 仿写三要素（必须遵守）

| 要素 | 规则 | 做法 |
|------|------|------|
| **人物** | 类型可照搬 | 战神/神医/扮猪吃虎等通用设定直接用，concept.md 定名 |
| **利益冲突** | 必须更换 | 源文是身份冲突→新书用利益冲突/信息差冲突/道德冲突 |
| **结构** | 照搬骨架 | 源文写心理→你也写心理；源文推进冲突→你也推进冲突；源文写钩子→你也写钩子。但动作和反应全部换 |

**核心**：只仿骨架（结构+情绪曲线），不仿血肉（具体事件）。情节必须完全不一样。

---

## 数据

- 源文字数：{源文字数}字（已统计，直接使用）
- 新书目标字数：{目标字数}字（固定2000字，番茄标准）

> 字数统计标准：所有可见字符（汉字+标点+数字+字母），不含换行和空格。

## 任务

1. 读 concept.md，记住固定角色名
2. 读源文第{N}章
3. 逐段拆源文：标结构功能 + 冲突类型 + 情绪 + 强度
4. 按情绪节拍表，填入全新事件——**冲突类型不同，动作反应全换，结构功能一致**
5. 按每个节拍在源文中的字数占比 × {目标字数}，分配新书字数。**各节拍字数之和必须等于 {目标字数}**
6. 观察源文第{N}章释放了哪些信息、没释放哪些信息 → 填入「信息释放对标」

## 源文→新书 节拍映射表 ⭐

| # | 结构功能 | 源文：冲突（类型） | 源文：动作 | 源文：反应 | 情绪 | 强 | 新书：冲突（类型≠源文） | 新书：动作（全换） | 新书：反应（全换） | 字数 |

### 冲突类型参考
身份冲突 / 利益冲突 / 信息差冲突 / 情感冲突 / 目标冲突 / 道德冲突 / 时间压迫 / 空间限制

### 结构功能参考
开场钩子 / 建立困境 / 推进冲突 / 心理活动 / 转折点 / 释放爽点 / 埋钩子 / 场景过渡

⚠️ 规则：
- "冲突类型"必须换（源文身份→新书利益/信息差/道德，不能相同）
- "社交场景"必须换（源文饭局→新书不能饭局；源文逛街→新书不能逛街；源文请客→新书不能请客）
- "动作"和"反应"必须全换（不能只改人名地名，事件本身要不同）
- "结构功能"和"情绪+强度"照搬（一一对应，不可改）

## 换皮检验 ⚠️

把新书所有节拍剥掉人名地名后，用一句话概括：
→ （概括）
→ 读者看到这句话能想起源文吗？能→换皮，重做。不能→合格。

## 角色高光时刻 ⭐（每章至少一个）

每章必须设计至少一个让人记住的场景。从以下类型中选 ≥1 个，标注发生在哪个节拍：

| 类型 | 说明 | 示例 |
|------|------|------|
| **甜** | 两人之间不经意的心动瞬间 | 他把她做的糊鸡蛋全吃了；她睡着了他给她盖被子 |
| **虐** | 不得不隐瞒/伤害对方的时刻 | 嘴上说分手，手在发抖；骗他出门，自己在屋里哭 |
| **笑** | 轻松幽默的桥段 | 女主逞强做饭差点烧厨房；男主面无表情说土味情话 |
| **反差** | 角色做出人意料的事 | 一直冷漠的人突然护短；一直怂的人突然硬刚 |
| **细节** | 一个小动作/物件承载情感 | 他把唯一的热牛奶推给她；她偷偷存着他写的小票 |

**格式**：类型 + 发生在第几个节拍 + 一句话描述 + 为什么读者会记住

```
## 本章高光
- 类型：甜 | 节拍#5 | 陆止渊把她做的糊鸡蛋全吃了，说"还行" | 读者记住：嘴上嫌弃身体诚实
```
