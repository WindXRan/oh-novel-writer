## 任务

读源文第{N}章，提取**让源文好看的关键技法**。不要写长篇分析，只输出 writer 能直接用的指令。

**核心目标：让仿写和源文一样好看，甚至更好看。**

---

## 输出结构

```
# 第{N}章 风格速查

## 源文亮点（writer 必须对标）

**记忆点**：源文最让人记住的1-2个瞬间是什么？（对话/细节/反转）
- 例："真乖。" / "去买盒避孕套。" / "Chanel新款50万"
- 仿写必须有同等量级的记忆点

**笑点**：源文有没有让人笑的地方？怎么制造的？
- 例："还没在一起？你不会暗恋人家吧？" — 反差式幽默
- 仿写每章至少1个笑点

**细节**：源文用了哪些具体细节（品牌/价格/物件/动作）？
- 例：铂金袖扣、Chanel新款、五十万、劳斯莱斯幻影
- 仿写必须用同等量级的具体细节，不能泛泛写"品质很好"

**节奏**：源文节奏快还是慢？哪里快哪里慢？
- 例：开头快（直接进入事件），中间慢（家庭互动细节），结尾快（悬念收尾）
- 仿写必须对标源文节奏

## 执行规则（核心，只保留最重要的）

### 必须做
1. **短句为主**：单句≤15字，连续3句不超20字。参考源文："真乖。""疯了。""疼。"
2. **动作展现情绪**：不写"她感到紧张"，写"她手指攥紧床单"。参考源文："他喉结滚了滚""她手心出汗"
3. **对话有记忆点**：每章至少1句让人想截图的对话。参考源文："去买盒避孕套""还没在一起？你不会暗恋人家吧？"
4. **细节要具体**：用品牌名/价格/物件名，不泛泛。参考源文："Chanel新款50万""铂金袖扣""劳斯莱斯幻影"
5. **每章至少1个笑点**：用反差/吐槽/意外制造。参考源文："你不会暗恋人家吧？""杨大小姐别气"
6. **男主内心戏≥20%**：男主必须有对女主的内心评价+对自己行为的反思。参考源文："只知道她皮肤很白，腰很细""他必须承认，此刻心下那丝罕见的心虚"

### 禁止做
1. **禁止AI路标词**：首先/其次/然后/最后/与此同时/此外/值得注意的是
2. **禁止直抒情**：感到无比/心中涌起/不由得/不禁 → 用动作替代
3. **禁止泛泛描写**："品质很好""长得很帅""气氛暧昧" → 用具体细节
4. **代词密度控制**：每100字内"她"≤3、"他"≤3，超过即违规
   - 用名字替代：「他扣住她的手腕」→「沈司南扣住她的手腕」
   - 省略主语：「她醒了，他还在睡」→「醒了。身边人还在睡。」
   - 称呼轮换：那男人、姓沈的、神外医生……交替使用
   - 合并动作：「她跑向门口，他从身后跟上来」→「人往门口跑，身后脚步声逼近」

### 字数控制
- 目标：{目标字数}字，范围：{目标字数_min}~{目标字数_max}字
- 不足：加细节/对话/动作
- 超标：删冗余/精简

## 本章情绪节奏
开篇（X情绪）→ 发展（X情绪）→ 高潮（X情绪）→ 结尾（X情绪）
```
