## 任务

### 第一步：分析源文风格

仔细阅读上方所有源文章节名，自行总结风格特征：
- 平均长度、长度范围
- 句式偏好（问句？省略句？对话体？短句？）
- 称呼词（宝贝/宝宝/乖宝 等）
- 人称视角（我/你/他）
- 语气基调（甜？虐？欲？撩？冲突？悬念？）
- 标点习惯（？……！。）
- 独特规律（如"谢京辞"频繁出现、角色名直接入标题等）

### 第二步：生成新标题

根据你分析出的风格，为每个仿写章节生成新标题。

**必须遵守**：
1. 长度分布与源文一致（如源文平均8字，你也要8字左右）
2. 句式比例与源文一致（如源文40%是问句，你也要约40%问句）
3. 称呼词风格与源文一致
4. 语气基调与源文一致
5. 标点习惯与源文一致
6. 标题内容必须反映该章节的核心冲突/情绪/事件

**禁止**：
- 使用源文人名/地名
- 使用 AI 味重的表达（"从XX到XX""他自XX来""XX的力量"）
- 风格偏离源文（源文是对话体你就不能写名词短语）

## 输出格式

只输出新标题，每行一个，格式：
```
第N章 标题文本
```

不要加任何解释、编号、markdown标记。
