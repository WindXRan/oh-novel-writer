## 描写技巧

### D1-肖像描写
**结构**：神情 + 发型/面容 + 年龄对比 + 状态反差
**规则**：
- 不写静态属性（"他有一头黑发"），写动态状态（"满头黑色短发失去了光泽"）
- 用对比制造反差：往日意气风发 vs 当前死灰
- 旁观者视角收尾：「[旁观者]心想，[角色]已经[判断]」
**正向**：「父亲神情肃穆地骑在马 上，满头棕色长发在风中飞扬。他修剪整齐的胡子里冒出几缕白丝，看起来比三十五岁的实际年龄要老些。」
**违规**：「他长得很帅，眼睛很大，有一头黑发。」→ 纯静态描述，缺乏动态和情感

### D2-五感描写
**结构**：前提概要 + 五感之一 + 场景描写 + 语言（可选）+ 感觉陈述
**规则**：选一个感官为主，不要五感全上
**五感正向**：
- 嗅觉：「社区带着警察刚一推开门，众人连忙捂住了鼻子，老单身汉的床底下传出一阵阵恶臭」
- 触觉：「林钰顿时耳鸣目眩，脸上火辣辣的，又疼又麻」
- 视觉：「衔远山，吞长江，浩浩汤汤，横无际涯」
- 听觉：「宿舍的水龙头失修已久时不时会渗水滴下，全身的精力都被滴水声缠上」
- 味觉：「一股腥臊味在味蕾上炸开，席卷了五脏六腑」
**违规**：「房间里很臭，很难受。」→ 缺乏感官细节和身体反应，直接告诉读者感受

### D3-物象排列
**结构**：人物行为 + 接续物象 + 环境刻画
**规则**：
- 物象用短词并列，制造画面密度
- 形容词+名词的结构：「通红的辣椒，翠绿的油菜，肥美的鸭子」
- 物象服务于情绪：欢快场景用明亮物象，压抑场景用灰暗物象
**正向**：「炼体道门，法术神通，符箓咒文，心经，密宗偏门，奇门遁甲。缩地成寸，撒豆成兵。真言，手决。更甚人生果嫁接法，蟠桃熏烟御寒术。凡所应有，无所不有。」
**违规**：「秘籍有很多，各种各样的都有。」→ 无具体物象，无画面密度

### D4-连贯动作描写
**结构**：一系列带有动词的连贯描写，形成动作流
**规则**：
- 动词要具体：不说"走过去"，说"贴着墙边缘小心挪动"
- 动作链条不断：A→B→C，中间不插议论
- 可在动作流末尾加一句身体反应（汗珠、喘息等）
**正向**：「市长一板一眼，一家一家看过来……还爬上了农家的小楼，掀开农民储粮、储米、盛油、盛酒的缸盖，然后伸手进去摸一摸……末了，市长还随手抓一把稻谷出来，挑几颗放进嘴里咬咬……」
**违规**：「市长检查了农民的储粮，发现情况还不错。」→ 动作被叙述替代，缺乏连续性

## 对话技巧

### 对话标签
**规则**：
- 不用"他说""她道""XX说/XX道"
- 用动作标签或无标签直接引语
- 对话前后加动作描写，不要干巴巴说话
**正向**：「雯压低声道："在那边等我。"忙不迭地拉上窗直奔下楼。」
**违规**：「"你还好吗？"他说道。」→ 干巴巴的对话标签

### 对话口语化
**规则**：
- 有打断、抢话、没说完的情况
- 心里想的话用意识流写法，不加引号
- 角色吵架时用短句对轰，不要解释性台词
**正向**：「"你放手！""你先放手！"」
**违规**：「"请你放开我的手，我对此感到非常不满。"」→ 过于书面化的对话

## 句式技巧

### 短句占比 ≥ 35%
**规则**：
- 用于情绪爆发、测试宣布、简评
- 连续3句以上超过20字 → 违规
**正向**：「"斗之力，三段！"」「转身。离开。」
**违规**：「萧炎缓缓转过身去，他的目光扫过那些嘲讽的面孔，嘴角露出一丝苦涩的自嘲。」→ 连续长句，无短句节奏

### 段落节奏
**规则**：
- 大部分段落只有一句话（特别是对话段落）
- 叙事段落不超过三句
- 对话和叙事交替出现，不要连续超过三个叙事段
**正向**：
```
"阿雯，出来呀！"
雯听到楼下的呼喊，拉开窗户，黎明的拂晓分外清冷。
圣司骑着自行车停靠在路边，朝着二楼挥手。
是他！雯欣喜万分。
```
**违规**：连续5段叙事，中间无对话打断 → 节奏拖沓

### 情绪外化
**规则**：用动作/神态/身体感受表达情绪，禁止直接写情绪词
**正向**：「紧握的手掌，因为大力，而导致略微尖锐的指甲深深的刺进了掌心之中」「脸红到脖子根」「手都在抖」「眼眶微热」
**违规**：「他感到无比愤怒」「心中涌起一股杀意」「不禁感到失落」→ 直接情绪词

### 环境描写密度
**规则**：
- 每章不超过3处，每处不超过1句
- 仅放在人物出场或场景转换时
- 连续3段大段环境描写 → 违规
**正向**：「人头汹涌的广场」「青衣少女快速跑出」（各1句，分散在不同位置）
**违规**：连续3段写「天空如何、地面如何、周围如何」→ 环境堆砌

## 代词密度控制

**规则**：每100字内"她"≤3、"他"≤3。超过即代词轰炸，读起来像乒乓球来回弹。

**技法**：
- **用名字替代**：「他扣住她的手腕」→「魏长河扣住她的手腕」
- **省略主语**：「她想要什么，他就不择手段」→「想要什么，魏长河就不择手段」
- **称呼轮换**：那男人、姓魏的、继兄、黑市商人……交替使用
- **合并动作**：「她利用完想踢开他，他扣住她的手腕」→「利用完想踢开人，手腕反被扣住」
- **改写句式**：「她……他……她……他」的对仗句拆掉，换成独立短句或动作描写

**正向**：「宋明锦想躲，躲不掉。想要什么，姓魏的不择手段捧到面前。利用完想抽身，手腕被扣住贴在他脸上。」（代词仅1个）
**违规**：「她想要什么，他就不择手段捧到她面前。当她利用完想踢开他时，他扣住她的手腕。」（代词6个）

## 比喻技巧

### 比喻规则
**规则**：
- 比喻要具体可感，不要抽象
- 用日常物象比喻，不要用生僻物象
- 比喻不超过1句，嵌入叙事中
**正向**：「对手如'前仆后继'般试图延缓阿甘的脚步」「象憋足了的水，遇见个出口就要激冲出去」
**违规**：「他的心情像天气一样阴沉。」→ 过于抽象的比喻
**违规**：「她美得像一朵花。」→ 陈词滥调

## 物象排列模板（可直接套用）

### 修真秘籍版
```
[角色]作为[身份]，[行为]，[心情]。
[物象1]，[物象2]，[物象3]，[物象4]，[物象5]。
[更甚的物象]，[更甚的物象]。凡所应有，无所不有。
```

### 嫁妆/礼物版
```
[容器]里东西真不少。[数量]+[物品1]及[物品2]。
[物品3]、[物品4]、[物品5]、[物品6]。
[珍品描述]，泛着[质感]。侧边还[附加物]：[具体内容]。
```

### 集市/场景版
```
[位置]的就是[排列方式]的[场景]。
[感官描述]，[物象1]，[物象2]。
一旁[人物]干[行为]着，[肖像]，[特点]。
```
