# 情节结构技巧库（plot 用）

> 来源：@旅行的巨人 高级仿写系列。生成 plot_guide 时按需注入对应技巧。
> 每条必须有：结构/模板 + 正向示例 + 违规示例

---

## 人物入场方式（4种，按需选用）

### T1-正面描写入场
**结构**：肖像描写 + 经历描写 + 其他人物侧面描写
**适用**：重要角色首次登场，需要建立读者印象
**模板**：
```
[角色]神情[情绪词]地[位置动作]，[头发/面容特征]。
[年龄/身份对比句]。[当前状态]，与往日[对比场景]相比，
很难想象得到是同一个人。[旁观者]心想，[角色]已经[判断句]。
```
**正向**：「父亲神情肃穆地骑在马 上，满头棕色长发在风中飞扬。他修剪整齐的胡子里冒出几缕白丝，看起来比三十五岁的实际年龄要老些。」
**违规**：「林钰是一个二十岁的年轻人，他有一头黑发，长得挺帅。」→ 纯静态描述，无情感、无对比、无旁观者视角

### T2-侧面描写入场
**结构**：借其他角色之耳目引入 + 被置入者样貌 + 他人之口叙述经历
**适用**：角色有 reputation，需要先建立神秘感或威望
**模板**：
```
[场景中]，[旁观角色]看见[被引入者]，喝采道："端的[评价]！"
[被引入者]听得，[反应]。[肖像描写]。
[旁观者]道："这位[身份]，[经历/名声]。"
```
**正向**：「智深正使得活泛，只见墙外一个官人看见，喝采道："端的使得好！"……众泼皮道："这位教师喝采，必然 是好。"智深问道："那军官是谁？"众人道："这官人是八十万禁军枪棒教头林武师，名唤林冲。"」
**违规**：「林冲是八十万禁军枪棒教头，武艺高强。」→ 直接告诉读者，没有通过他人之口建立神秘感

### T3-全面介绍入场
**结构**：时间 + 基本情况（称呼/年龄/身份）+ 关系 + 经历背景 + 突出特点 + 牵带故事
**适用**：主角或核心配角，需要完整背景铺垫
**模板**：
```
[时间]。[角色全名]，[表字/小名/外号]，[年龄]，[何许人]。
[家庭关系]。[职业/技艺]。[性格/爱好]。[旧事/名声]。
[牵出当前故事的引子]。
```
**正向**：「高中生侦探工藤新一，年仅十七岁，就读于帝丹高中，人称"平成年代的福尔摩斯"。在与青梅竹马的同学毛利兰一起到游乐圆游玩时，无意间却目击了黑衣男子的交易现场……」
**违规**：「林钰，男，二十岁，大学生，性格开朗。」→ 简历式罗列，无故事感、无引子

### T4-忽入法登场
**结构**：其他角色正在行动 → 特定角色突然入场（行为或对话先入）→ 注意力转移
**适用**：制造戏剧冲突、解围、转折
**模板**：
```
[危急场景，角色A正在被威胁/遇险]。
说时迟那时快。[突然声响/动作]，[新角色]跳出来，喝道："[台词]！"
[角色A]方才[反应]时，认得是[新角色]。
```
**正向**：「话说当时薛霸双手举起棍来，望林冲脑袋上便劈下来。说时迟，那时快，薛霸的棍恰举起来，只见松树背后雷鸣也似一声，那条铁禅杖飞将来……跳出一个胖大和尚来，喝道："洒家在林子里听你多时！"」
**违规**：「这时鲁智深出现了，他救了林冲。」→ 叙述代替场景，无突然感、无动作

## 场景切入方式（3种）

### S1-人物与场景结合
**结构**：人物状态（位置+行为）[全景] → 场景刻画（物景+环境）[烘托] → 人物特写（动作+肖像）[聚焦]
**适用**：场景转换、开篇建立氛围
**模板**：
```
[位置]，[角色][行为]。[环境物象1]，[环境物象2]，[环境物象3]。
[角色]屏着呼吸，[具体动作]，[身体细节]。
```
**正向**：「翻出窗外，千寻双手贴靠在墙缘上，手里沾满了锈粉。沿着墙边，一眼望去生锈斑斑的管道悬靠在外墙上，其中不少钉扣还缺了半块。海面上吹来的风带着湿气，鸥鸟贴在侧边盘旋……千寻屏着呼吸，用束带扎紧衣服和袖口，尽力平衡着颤巍巍的着身子往铁管上走去。」
**违规**：「千寻爬出了窗外，外面有很多管道，风很大。」→ 缺乏物象细节、缺乏身体感受、缺乏层次感

### S2-人物与环境交替
**结构**：
- 第1段：人物描写与环境描写交替出现
- 第2段：环境相关，稍拉远景，注意力在环境
- 第3段：人物相关，注意力在人物
**适用**：需要同时建立场景感和人物感
**模板**：
```
[环境细节]，[角色][行为]。[环境物象]，[角色反应]。
[远景环境]，[环境声音/光线]。[环境物象2]，[环境物象3]。
[角色特写]，[动作]，[身体细节]。
```
**正向**：「地基的柱座因雨天变得潮湿非常，阿莉埃蒂落寞地坐在残破的瓦片上……栅栏外的阴雨淅淅沥沥，爬墙草的藤蔓搭在边上，五角的叶子挂着雨珠。一块方糖被放在了窗台沿……阿莉埃蒂倚在墙边，看到翔逐渐远去。她连忙扯下一片叶子当做雨伞……」
**违规**：「下雨了，阿莉埃蒂坐在那里，很伤心。然后她看到了方糖。」→ 环境和人物混在一起，无层次切换

### S3-画面聚焦递进
**结构**：上段赋予情感色彩（抒发情绪）→ 下段紧接上段最后一句展开详写（情绪递进）
**适用**：情绪高光、转折点
**模板**：
```
[环境/事件]赋予[情感色彩]。[情绪判断句]。
[紧接上句展开]。[细节放大]。[身体/心理反应]。
```
**正向**：「青色火焰的心脏却感受不到灼热，生命如同黯淡的烟火消去。苏菲不安地握着的那团火焰心脏，抱到胸口前，可以想象得到失去心脏的霍尔的那般糟糕模样。霍尔既没刮胡子也没梳头发，眼睛的红眼圈仍在，黑袖子则破了好几处，看来跟稻草人一样糟。他会发疯的。」
**违规**：「苏菲拿着火焰心脏，很难过。霍尔的样子很糟糕。」→ 缺乏情感铺垫、缺乏细节放大、缺乏递进

## 章间衔接方式（5种）

### C1-引导型
**手法**：时间、地点、发问、设定等引导词
**正向**：「再说林冲踏着那瑞雪，迎着北风，飞也似奔到草料场门口。」「次日正是上元节候，天色晴明得好。」
**违规**：「接下来林冲去了草料场。」→ 过于平淡，缺乏画面感
**常用词**：「再说」「话说」「且说」「却说」「次日」「翌日」

### C2-暗示渲染型
**手法**：环境描写、赋比兴，暗示下文走向
**正向**：「……既见复关，载笑载言。尔卜尔筮，体无咎言。以尔车来，以我贿迁。于嗟女兮，无与士耽！」（先写甜蜜，再写警示，暗示悲剧）
**违规**：「他们结婚了。后来她后悔了。」→ 直接叙述，无暗示和铺垫

### C3-标识型
**手法**：切词、空段、分割线
**正向**：「——」「***」「第一章」「（一）」
**违规**：在同一段落内直接写"后来""然后"进行转换 → 缺乏明确的场景切割感

### C4-议论型
**手法**：带有价值判断的叙述，总结上文，为下段提供自由度
**正向**：「张小敬故意没有问任何问题，让囚犯在心理上产生错觉，以为拷问方无求于自己。这样才会让他愈加惶恐，愈加急切地想证明自己的价值。刑求这门艺术，和房事一样，精髓在于前戏。安排好之后，张小敬转身离开告解室……」
**违规**：「张小敬离开了房间，囚犯很害怕。」→ 缺乏价值判断和过渡感

### C5-蒙太奇
**手法**：以共有对象作为衔接点，切换镜头（时间/空间/视角）
**适用**：压缩时间跨度、对比场景、快速推进
**模板**：
```
[场景A，聚焦物象X]。
[场景B，物象X出现/呼应]。
```
**正向**：「林钰赶着下班到菜市场里买菜，照例在菜市口买了半斤猪肉，一朵菜花，一把韭菜和一托鸡蛋……酒足饭饱后，桌上只留下残羹剩菜。林钰到头在沙发上刷着手机。」（买菜→吃饭，跳过做饭过程）
**违规**：「林钰买了菜。然后他洗菜。然后切菜。然后炒菜。然后吃饭。」→ 流水账叙述，无跳跃感

## 开篇钩子技巧（第1章专用）

### K1-冲突前置法
**结构**：直接进入矛盾场景 → 用对话/动作引爆冲突 → 补充背景
**适用**：第1章开篇，快速抓住读者
**模板**：
```
[角色]正在[困境场景]。
[冲突方]出现/联系，抛出[矛盾]。
[角色反应]，情绪[压抑/愤怒/无奈]。
→ 补充背景（不超过3句）
```
**正向**：「林瑾蜷缩在急诊室的椅子上，左手手背扎着针头。手机亮了，赵秀芬的声音像刀片刮过铁皮：'今晚家宴你准备得怎么样了？'」
**违规**：「林瑾从小就喜欢文物修复，她在这个行业工作了八年……」→ 大段背景开场，无冲突

### K2-朋友圈/社交媒体引爆法
**结构**：女主发现关键信息（朋友圈/短信/消息）→ 情绪爆点 → 展开冲突
**适用**：现代都市文，快速揭露矛盾
**模板**：
```
[角色]刷到/看到[社交媒体内容]。
内容显示[矛盾信息]（背叛/羞辱/隐瞒）。
[角色情绪反应]，[身体反应]。
→ 冲突展开
```
**正向**：「盛惜看了眼手机屏幕，呼吸一滞。是林蓁的朋友圈，上面那条动态正是早上六点，姜霖发的动态。配文：下载进度停在了99%和0%有什么区别。」
**违规**：「盛惜发现姜霖最近总是很晚回家，她怀疑他有问题……」→ 慢慢铺垫，无爆点

### K3-悬念开场法
**结构**：抛出一个谜团/异常现象 → 引发好奇 → 逐步揭晓
**适用**：悬疑/逆袭类，制造好奇心
**模板**：
```
[场景中出现异常]。
[角色]注意到[细节]，但[原因不明]。
[悬念]：为什么/怎么回事/谁做的？
→ 后续揭晓
```
**正向**：「垃圾桶里塞满了东西。她走过去，低头一看——捡碎瓷片的刷子、那把自己用了十年的修复刀、还有半盒她收藏的矿物颜料，全被扔了。修复刀的刀刃有裂痕。」
**违规**：「林瑾回到家，发现婆婆把她的东西扔了。」→ 平铺直叙，无悬念

### K4-情绪共鸣法
**结构**：展现女主的委屈/困境 → 引发读者共情 → 给出希望/转机
**适用**：虐文/逆袭文，建立读者情感连接
**模板**：
```
[角色处于困境]，[具体委屈细节]。
[内心独白/身体反应]，展现[压抑/无奈]。
[转机出现]：电话/消息/事件 → 悬念
```
**正向**：「林瑾握着手机的手在发抖。不是因为生气，是因为冷。输液管里的液体冰凉，从手背进到血管，整个人都像被扔进了冰窖里。手机又震了。这次是个陌生号码。」
**违规**：「林瑾很难过，她觉得自己的人生很失败。然后她接到了一个电话。」→ 直接告诉读者情绪，无代入感

## 配角功能技巧（前3章必备）

### P1-闺蜜/助手"嘴替"功能
**结构**：女主受委屈 → 闺蜜/助手替读者骂人 → 调节情绪节奏
**适用**：压抑场景后的情绪释放
**模板**：
```
[女主遭遇不公]，[隐忍/沉默]。
[闺蜜/助手出现]，[愤怒反应]："他怎么敢/她凭什么/这也太过分了！"
[女主反应]：苦笑/摇头/沉默 → 展现人物性格
```
**正向**：「"这个老太婆把你当猴耍呢！"小余气不过，声音都高了，"什么叫'展示才艺'？什么叫'评判'？您可是……"她说到一半，林瑾按住了她的手腕，轻轻摇了摇头。」
**违规**：「林瑾很难过，但没人理解她。」→ 缺少情绪出口，读者憋屈

### P2-配角信息传递功能
**结构**：配角带来关键信息 → 推动情节 → 展现女主处境
**适用**：需要外部信息介入时
**模板**：
```
[配角出现]，带来[信息]。
[信息内容]：揭露真相/提供机会/制造冲突
[女主反应]：震惊/愤怒/期待
```
**正向**：「"姐，你看。"小余把手机递到她面前。屏幕上是朋友圈截图。赵秀芬发的："今晚家宴，准儿媳要来展示她的'修复才艺'。"」
**违规**：「林瑾发现婆婆在朋友圈发了不好的内容。」→ 直接叙述，无场景感

## 关系突破技巧（第3章必备）

### R1-合作契约法
**结构**：男主提出合作 → 条件交换 → 建立绑定关系
**适用**：职业线强的作品，用专业合作建立关系
**模板**：
```
[男主]提出[合作内容]。
条件：[女主帮他做什么] ↔ [他帮女主做什么]
[女主犹豫/震惊] → [男主给出理由]："因为你[具体原因]"
→ 达成协议
```
**正向**：「"你帮我修复三件东西，"顾西洲说，"北宋官窑碎片、我实验室里的一个仿品，还有一件私人物品。我帮你去把那件笔洗弄回来，保证不落在刘家手里。"」
**违规**：「顾西洲帮了林瑾，两人成了朋友。」→ 关系不明确，无张力

### R2-闪婚契约法
**结构**：男主提出闪婚 → 理由充分 → 快速建立关系
**适用**：先婚后爱题材，需要快速推进关系
**模板**：
```
[男主]提出[闪婚/假结婚]。
理由：[互惠互利/解决各自问题]
[女主震惊/犹豫] → [男主给出具体理由]："我看过你[具体信息]，知道你是[判断]"
→ 答应
```
**正向**：「"林小姐，"顾西洲忽然开口，"五分钟前，我刚收到实验室的'终身研究员'任命。条件是需要一个稳定的家庭证明。刚好，我的私生活有漏洞。你的婚姻也有漏洞。我们互为补丁。"」
**违规**：「顾西洲向林瑾求婚了。」→ 理由不充分，显得突兀

### R3-情感铺垫法
**结构**：男主透露长期关注 → 具体细节 → 建立信任基础
**适用**：需要解释男主为什么对女主特殊
**模板**：
```
[男主]透露[关注女主的时间/方式]。
具体细节：[看过她的作品/知道她的秘密/共同经历]
[女主震惊]："你怎么知道？"
[男主回答]："因为[具体原因]"
```
**正向**：「"我看过你的修复记录，"顾西洲说，"四年，九十七件，每一件我都看过。如果你不是想进朝露实验室，你不会在那个低温修复箱里待一年，就为了试出一种可逆粘合剂。"」
**违规**：「顾西洲一直关注林瑾。」→ 理由不具体，显得刻意

## 相遇方式技巧（避免撞梗）

### M1-专业场合相遇
**结构**：女主参加专业活动 → 男主也在场 → 因专业互动而相识
**适用**：职业线强的作品，建立专业认可
**模板**：
```
[女主]参加[专业活动：拍卖会/展览/学术会议]。
[男主]也在场，[身份/角色]。
[专业互动]：男主注意到女主的专业能力/观点
→ 建立第一印象
```
**正向**：「林瑾在拍卖预展上指出那件青花瓷是赝品，全场哗然。角落里一个穿白大褂的男人放下咖啡杯，目光落在她脸上。」
**违规**：「林瑾走在路上，差点被一辆车撞到。」→ 过于常见，缺乏专业性

### M2-目的性接近法
**结构**：男主主动来找女主 → 带着明确目的 → 建立合作关系
**适用**：需要快速建立男女主联系
**模板**：
```
[男主]出现在[女主的工作场所/住所]。
目的：[合作/求助/调查]
[对话]：男主说明来意，展示对女主的了解
→ 建立联系
```
**正向**：「顾西洲的宾利停在修复室门口。他推开车门，手里拿着一份修复方案：'林小姐，这个方案是你的？'」
**违规**：「顾西洲在路上遇到了林瑾。」→ 缺乏目的性，显得巧合

### M3-配角引荐法
**结构**：配角介绍男女主认识 → 背景交代 → 建立初步联系
**适用**：需要自然的相遇方式
**模板**：
```
[配角]安排[男女主]见面。
[配角介绍]：说明双方身份/背景
[男女主互动]：初次对话，建立印象
```
**正向**：「陈屿拉着林瑾走进拍卖行VIP室：'顾教授，这就是我跟您提过的林瑾，修远的金奖修复师。'」
**违规**：「林瑾和顾西洲在街上偶遇。」→ 缺乏自然性和目的性

### M4-工作场景重逢法
**结构**：男女主在工作场景重逢 → 回忆过去 → 建立新的联系
**适用**：有前史设定的作品
**模板**：
```
[女主]在[工作场景]工作。
[男主]出现，[身份/目的]。
[回忆]：两人曾经的交集
→ 建立新的关系
```
**正向**：「林瑾正在修复室整理碎片，门被推开。她抬头，看见三年前在修远协会见过的那个男人——顾西洲。他手里拿着一块永乐官窑碎片：'林小姐，这个你能修吗？'」
**违规**：「林瑾和顾西洲在咖啡馆偶遇。」→ 缺乏工作场景的专业性

## 原创情节框架技巧（避免换皮写作）

### 🎯 核心原则：改变"情节序列"，而非只换"表面细节"

**换皮写作**：只换了人名/职业/疾病，但情节序列与源文一致
**原创写作**：情节序列完全不同，只保留情绪曲线和结构功能

---

### F1-开篇方式原创

**源文常见开篇**：
- 医院输液+闺蜜告密+发现被屏蔽
- 家庭聚会+被羞辱+回忆委屈

**原创开篇方式**：

#### 方式A：工作场景开篇
```
[女主]在[工作场景]处理[工作事务]。
[突发事件]：收到坏消息/遭遇危机/发现真相
[情绪反应]：震惊/愤怒/心寒
→ 展开冲突
```
**示例**：「林瑾正在修复室加班，手机弹出一条拍卖通知——她修复了三个月的那件青花瓷，被委托人送拍了。委托人：刘铭远。」

#### 方式B：社交场合开篇
```
[女主]参加[社交场合：聚会/晚宴/同学会]。
[意外发现]：听到真相/看到不该看的/被当众羞辱
[情绪反应]：心寒/愤怒/觉醒
→ 展开冲突
```
**示例**：「沈若站在包厢门口，听见周景川对朋友说：'沈若？她就是个备胎，许诺不回来我才跟她结婚。'」

#### 方式C：日常场景开篇
```
[女主]在[日常场景：超市/健身房/咖啡馆]。
[偶遇/发现]：看到男主和别人在一起/发现男主的秘密
[情绪反应]：震惊/心碎/觉醒
→ 展开冲突
```
**示例**：「沈若在超市买菜，转角看见周景川推着购物车，车里坐着一个女人和孩子——许诺。」

### F2-分手方式原创

**源文常见分手**：
- 上门摊牌→发现关键物件→激烈争吵→留下信物离开
- 通过"测试某个物件"确认心意→引发清算式争吵

**原创分手方式**：

#### 方式A：冷静摊牌
```
[女主]找到[男主]，直接摊牌。
[证据]：拿出确凿证据（不是猜测）
[男主反应]：否认/狡辩/无所谓
[女主决定]：冷静离开，不做纠缠
```
**示例**：「沈若把体检报告放在桌上：'我怀孕了，孩子不是你的。'周景川愣住。沈若转身走了。」

#### 方式B：公开决裂
```
[社交场合]中，[女主]当众揭露[男主]的真面目。
[证据]：展示证据/说出真相
[男主反应]：尴尬/愤怒/狡辩
[女主决定]：当众宣布分手
```
**示例**：「婚礼现场，沈若抢过话筒：'我不会嫁给一个心里有别人的男人。'全场哗然。」

#### 方式C：自我觉醒
```
[女主]经历某件事后，突然觉醒。
[触发事件]：看到男主对白月光的态度/发现自己被利用
[内心独白]：回顾过去，看清真相
[行动]：主动离开，不做解释
```
**示例**：「沈若看着周景川为许诺忙前忙后，忽然笑了。她摘下戒指放在桌上，转身离开。周景川追出来：'你去哪？'沈若没回头：'去一个不用当替身的地方。'」

### F3-新男主出场原创

**源文常见出场**：
- 女主险被车撞→新男主出现
- 新男主是渣男的小舅/亲戚

**原创出场方式**：

#### 方式A：专业场合出场
```
[女主]参加[专业活动]。
[男主]也在场，[身份/角色]。
[互动]：男主注意到女主的专业能力/观点
→ 建立第一印象
```
**示例**：「林瑾在拍卖会上指出那件瓷器是赝品，全场哗然。角落里一个男人放下咖啡杯，目光落在她脸上。」

#### 方式B：工作合作出场
```
[男主]主动来找[女主]，带着[合作/求助]目的。
[背景]：男主知道女主的专业能力/作品
[互动]：提出合作，展示诚意
→ 建立联系
```
**示例**：「顾西洲的车停在修复室门口：'林小姐，我有个项目想请你帮忙。'」

#### 方式C：配角引荐出场
```
[配角]介绍[男女主]认识。
[背景]：双方的身份/专业
[互动]：初次对话，建立印象
→ 建立联系
```
**示例**：「陈屿拉着林瑾走进VIP室：'顾教授，这就是我跟您提过的林瑾。'」

### F4-闪婚动机原创

**源文常见闪婚**：
- 报复渣男+应付家里
- 互相利用的协议婚姻

**原创闪婚动机**：

#### 方式A：专业合作
```
[男主]需要[女主的专业能力]来解决[问题]。
[女主]需要[男主的资源/保护]来应对[困境]。
[协议]：以婚姻为条件，互相帮助
```
**示例**：「顾西洲需要一个修复师帮他修复国宝级文物，林瑾需要一个靠山来对抗刘家。两人各取所需。」

#### 方式B：家族压力
```
[男主]面临[家族压力：催婚/遗产/继承]。
[女主]面临[困境：被追债/被威胁/无家可归]。
[协议]：以婚姻为条件，互相解围
```
**示例**：「陈渡需要一个妻子来继承家族遗产，沈若需要一个庇护来躲避周家的报复。两人一拍即合。」

#### 方式C：意外事件
```
[意外事件]导致两人被迫结婚。
[原因]：醉酒/误会/被设计
[后果]：必须结婚来解决问题
```
**示例**：「沈若喝醉了，醒来发现手上多了一枚戒指。陈渡坐在旁边：'昨晚你说要嫁给我，我已经录音了。'」
