# 《辞凤台》章纲（第1-3章）

## 章纲格式说明
每章包含：章名、章首钩子类型、核心事件、情绪基调、章尾钩子类型

---

## 第一章

| 字段 | 内容 |
|------|------|
| **章名** | 凤台旧梦 |
| **章首钩子类型** | 场景描写开场（喜庆的及笄宴场景，与前世刑场形成极端对比） |
| **核心事件** | 1. 沈玉容在刑场被斩首的瞬间，意识回到三年前及笄宴当天清晨，发现自己重生<br>2. 她迅速回忆前世时间线：及笄宴→太子带兵包围→满门下狱→三个月后抄斩，决定必须立即行动<br>3. 沈玉容走出闺房，看到正在布置的及笄宴现场，母亲笑着为她整理发髻，她强忍泪水，暗下决心保护家族 |
| **情绪基调** | 喜庆→震惊→决断 |
| **章尾钩子类型** | 悬念断章（沈玉容透过窗棂，看到将军府门外出现太子府的马车，前世灾难的开端比她记忆中来得更早） |

---

## 第二章

| 字段 | 内容 |
|------|------|
| **章名** | 暗流涌动 |
| **章首钩子类型** | 对话开场（沈玉容试探性询问母亲今日宾客名单，母亲疑惑女儿为何突然关心这些） |
| **核心事件** | 1. 沈玉容借故离开宴席，在父亲书房暗中寻找沈家军虎符，发现虎符已被父亲提前藏入密室<br>2. 她偶遇前来赴宴的三皇子萧景珩，两人首次交锋，萧景珩察觉她与传闻中的温婉嫡女不同<br>3. 沈玉容从丫鬟口中得知，太子今日赴宴带了比平时多三倍的侍卫，印证前世记忆 |
| **情绪基调** | 冷静→试探→发现 |
| **章尾钩子类型** | 悬念断章（沈玉容在花园角落发现一个前世没有出现过的陌生侍卫，此人正悄悄向内院张望） |

---

## 第三章

| 字段 | 内容 |
|------|------|
| **章名** | 初试锋芒 |
| **章首钩子类型** | 动作/事件开场（及笄宴正席开始，太子萧景恒当众宣布要为沈将军"贺寿"，实则准备发难） |
| **核心事件** | 1. 太子当众指责沈将军拥兵自重，暗示其有谋反之心，沈将军正要辩解，沈玉容抢先一步跪地，以退为进主动请求交出虎符以示忠心<br>2. 太子措手不及，原本准备的"证据"失去作用，皇帝见沈家如此"忠顺"，反而训斥太子小题大做<br>3. 萧景珩在席间冷眼旁观，散席后单独拦住沈玉容，问她为何主动交出虎符，沈玉容只答"保命要紧" |
| **情绪基调** | 紧张→交锋→初胜 |
| **章尾钩子类型** | 悬念断章（沈玉容回到闺房，发现枕下多了一张纸条，上面只有四个字："虎符是假"） |

---

## 备注
- 每章2000-2500字
- 章名四字短语
- 开篇用"喜庆+灾难"对比（第一章及笄宴的喜庆与前世刑场的灾难形成心理对比）
- 结尾在悬念处切断
- 第三章为结局章，必须留悬念
