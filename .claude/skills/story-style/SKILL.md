# story-style：写作风格知识库

**职责：承载所有写作规则、风格定义、质量标准。story-rewrite 通过 --style 调用。**

---

## 风格发现

扫描两个位置，合并为统一注册表：

| 位置 | 来源 |
|------|------|
| `.claude/skills/story-style/*/SKILL.md` | 网文作者文风 |
| `.claude/skills/*-perspective/SKILL.md` | 通用人物 Skill |

**注册规则**：目录下存在 `SKILL.md` → 注册为可用风格。

**名称映射**：
- `.claude/skills/story-style/{name}/SKILL.md` → 风格名 = `{name}`
- `.claude/skills/{name}-perspective/SKILL.md` → 风格名 = `{name}`

## 通用写作知识（所有风格共享）

### 写作原则

#### 信任读者

读者是主动协作者。他们从行为重建情绪、从动作推断动机、跨场景保持张力、填补文本留白。

**LLM 的倾向**：过度解释、过度澄清、过度完成。
**正确做法**：留空间，让读者自己做 work。

#### 经济性

每个元素做多件事：
- 一行对话：推进剧情 + 揭示角色
- 一个感官细节：落地场景 + 展示 POV 角色
- 一个过渡：压缩时间 + 传递情绪

**单功能 prose 是小说变平的原因。**

#### 四通道读者心理

读者通过 4 个独立通道获得快感，任何一个受损体验就下降：

| 通道 | 含义 | 保护方式 |
|------|------|---------|
| **沉浸** | 进入故事世界 | 连贯叙事、一致POV、具体感官 |
| **审美** | 句子层面的愉悦 | 节奏变化、用词多样、句式丰富 |
| **社会模拟** | 把角色当真人 | 行为展示、内心独白、声音区分 |
| **心流** | 可读的挑战感 | 节奏匹配场景、支持理解的句子 |

**关键**：优化一个通道牺牲其他通道，会失败。

### 章首钩子类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **悬念型** | 抛出未解问题 | 「门外传来脚步声，越来越近。」 |
| **冲突型** | 直接进入矛盾 | 「离婚协议书就放在桌上。」 |
| **反转型** | 颠覆前章预期 | 「我以为他不爱我，直到看见那封信。」 |
| **对话型** | 用对话开场 | 「你是不是在外面有人了？」 |
| **动作型** | 直接进入动作 | 「我冲进火场的时候，什么都看不见。」 |
| **内心独白型** | 展示角色内心 | 「我不该来的。但我的脚不听使唤。」 |
| **感官型** | 用感官细节开场 | 「咖啡的苦味在舌尖化开。」 |
| **时间跳跃型** | 直接跳转时间 | 「三个月后，我再次站在这个门口。」 |

### 章尾钩子类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **悬念型** | 留下未解问题 | 「门开了，站在那里的人让我愣住了。」 |
| **反转钩** | 颠覆读者预期 | 「原来一切都是他安排的。」 |
| **情绪高点** | 在情绪最浓时切断 | 「他说完那句话，转身走进了雨里。」 |
| **选择困境** | 角色面临两难 | 「一边是救命的手术费，一边是妹妹的学费。」 |
| **信息炸弹** | 抛出关键信息 | 「DNA报告上写着：亲缘关系99.9%。」 |
| **动作中断** | 动作进行中切断 | 「我伸出手，却抓了个空。」 |
| **对话中断** | 对话进行中切断 | 「她张了张嘴，想说什么，最终什么都没说。」 |
| **时间锚点** | 指向未来/过去 | 「我不知道，三天后我会后悔这个决定。」 |

### 段落级钩子

| 类型 | 说明 |
|------|------|
| **短句起手** | 3-5字短句制造紧张感 |
| **问句引导** | 用问句引导读者思考 |
| **反差制造** | 前后句形成反差 |
| **细节放大** | 放大某个细节制造张力 |
| **节奏变化** | 长短句交替制造节奏 |

### 角色设计方法

#### 主角卡模板

| 维度 | 内容 |
|------|------|
| 姓名 | 中文名/昵称/外号 |
| 年龄 | 具体年龄或年龄段 |
| 身份 | 社会角色/职业 |
| 外貌 | 3个关键特征 |
| 性格 | 3个核心特质 + 1个反差 |
| 动机链 | 起因→意图→约束→风险 |
| 语言风格 | 口癖/用词习惯/禁忌词 |
| 人物弧线 | 起点状态→终点状态 |

#### 配角卡模板

| 维度 | 内容 |
|------|------|
| 姓名 | 中文名/昵称 |
| 与主角关系 | 亲/友/敌/暧昧 |
| 功能定位 | 帮手/对手/导师/催化剂 |
| 性格标签 | 2个核心特质 |
| 语言特征 | 与主角的区分点 |

#### 反派层级

| 层级 | 说明 | 示例 |
|------|------|------|
| 小反派 | 短期障碍，3-5章解决 | 恶毒同事、白莲花 |
| 中反派 | 中期威胁，10-20章解决 | 家族长辈、商业对手 |
| 大反派 | 全书威胁，最终解决 | 幕后黑手、宿敌 |

### 对话技法

#### 对话节奏

| 节奏 | 效果 | 使用场景 |
|------|------|---------|
| 快速交锋 | 紧张、冲突 | 吵架、谈判、试探 |
| 缓慢铺垫 | 暧昧、温情 | 告白、谈心、回忆 |
| 中断打断 | 悬念、转折 | 关键信息被打断 |
| 沉默留白 | 张力、压抑 | 无言以对、情感压抑 |

#### 潜台词

**核心原则**：角色不会直接说出真实想法。

| 类型 | 说明 | 示例 |
|------|------|------|
| 说反话 | 表面意思与真实相反 | 「我很好」= 我不好 |
| 说一半 | 只说部分真相 | 「我昨天去了商场」（没说遇见了谁） |
| 用问题回答 | 逃避直接回答 | 「你觉得呢？」 |
| 转移话题 | 回避敏感话题 | 「今天天气不错」 |
| 行动代替 | 用行动表达 | 沉默地递过纸巾 |

#### 角色语言差异化

**7维方法**：

| 维度 | 说明 |
|------|------|
| 句子长度 | 长句/短句/混合 |
| 用词正式度 | 口语/书面/混合 |
| 感叹词使用 | 多/少/无 |
| 比喻偏好 | 生活化/文艺化/专业术语 |
| 逻辑连接词 | 多用/少用/不用 |
| 问句频率 | 爱问/陈述为主 |
| 打断方式 | 插话/沉默/行动 |

### 去AI规则

#### AI写作典型特征

| 特征 | 表现 | 修复 |
|------|------|------|
| **过度解释** | 把情绪直接说出来 | 改为行为展示 |
| **句式模板化** | 「XX说道」「XX不禁XX」 | 强制换句式 |
| **段落太均匀** | 每段长度差不多 | 长短交替 |
| **情绪温吞** | 该激烈时不够激烈 | 注入强度指令 |
| **连接词过多** | 「然而」「但是」「因此」 | 减少连接词 |
| **比喻堆砌** | 连续用3个以上比喻 | 限制比喻密度 |
| **感叹号滥用** | 连续多个感叹号 | 控制标点 |
| **「不禁」泛滥** | 每章出现多次「不禁」 | 删除或替换 |

#### 去AI三遍法

**第一遍：结构去AI**
- 句长随机化（短句5-15字，长句20-40字）
- 段落打散（长短交替）
- 句首分散（不连续用同一词开头）

**第二遍：内容去AI**
- 动作-反应解耦（不是每个动作都有反应）
- 情绪留白（不是每个情绪都命名）
- 细节具体化（不是「花」而是「茉莉」）

**第三遍：语言去AI**
- 禁用词替换
- 句式多样化
- 对话标签变化

---

### 禁用词表

**一级禁用（必须删除）**：
- 不禁、忍不住、情不自禁
- 眼底、眸中、眸光
- 微微、轻轻、缓缓
- 嘴角上扬、嘴角微扬
- 心头一颤、心头一紧
- 脑海中浮现、脑海中闪过

**二级禁用（尽量避免）**：
- 说道、问道、叹道
- 然而、但是、不过
- 突然、忽然
- 终于、最终
- 不由得、不自觉

**三级禁用（限制使用）**：
- 仿佛、好像、似乎
- 一定、肯定、绝对
- 永远、一直、从来

### 质量检查清单

#### 黄金三章自查

| 检查项 | 标准 |
|--------|------|
| 第1章 | 建立代入感、展示核心冲突、留下悬念 |
| 第2章 | 深化冲突、展示角色魅力、制造期待 |
| 第3章 | 第一个小高潮、兑现第一个承诺、留下新钩子 |

#### 写作过程自查

| 检查项 | 标准 |
|--------|------|
| POV一致性 | 不超出当前角色视角 |
| 角色声音 | 遮住名字能分辨是谁在说话 |
| 情绪连贯 | 不跳跃、不矛盾 |
| 细节具体 | 不用模糊词，用具体细节 |
| 对话自然 | 有潜台词，不直白 |
| 节奏匹配 | 紧张时短句，舒缓时长句 |

#### 毒点排查

| 毒点 | 表现 |
|------|------|
| 圣母心 | 主角无原则原谅 |
| 降智 | 角色智商突然下降 |
| 注水 | 无意义的描写和对话 |
| 拖沓 | 3章内没有推进剧情 |
| 人设崩 | 角色行为与设定矛盾 |
| 三观歪 | 传递错误价值观 |

### 风格质量基线

| 维度 | 最低标准 |
|------|---------|
| 表达DNA | ≥5个量化特征 |

| 评级 | 标准 |
|------|------|
| ⭐⭐⭐ | 满足所有基线 + 有全量统计 + 有章纲模板 |
| ⭐⭐ | 满足基线 + 有章纲模板 |
| ⭐ | 满足基线 |
