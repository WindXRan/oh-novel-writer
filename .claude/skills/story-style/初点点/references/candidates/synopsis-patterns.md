# 初点点 · 书名与简介规则

> 基于7本已完结作品提取

---

## 一、书名模式

### 1.1 两字/三字/四字古典意象词（核心风格）

初点点最典型的命名方式，用凝练的古典词汇营造氛围。

| 书名 | 字数 | 类型 |
|------|------|------|
| 绮靡 | 2 | 形容词，华丽纤柔 |
| 惊华庭 | 3 | 名词性，华美庭院 |
| 折春漪 | 3 | 动宾，折断春水涟漪 |
| 烽火佳人 | 4 | 意象组合，战乱+美人 |
| 染指枭雄 | 4 | 动宾，染指+枭雄 |
| 王妃换嫁 | 4 | 主谓，王妃+换嫁 |

**规则**：择取具有画面感或情绪张力的2-4字组合，避免直白叙事，留白让读者自行想象。

---

### 1.2 长标题·逗号式（流量型）

用于主站/推荐场景，信息密度高，关键词前置。

> **闪婚，真千金嫁给了军阀大佬**

结构：`[热点词]，[身份]嫁给了[强势男主]`

- 热点词：闪婚
- 女主标签：真千金
- 男主标签：军阀大佬

> **侯府忘恩义？摄政王撑腰，不原谅**

结构：`[委屈设定]？[靠山]，[态度]`

- 委屈：侯府忘恩义（问号强化情绪）
- 靠山：摄政王撑腰
- 态度：不原谅

**规则**：长标题控制在11-15字，用逗号或问号分段。前半段堆委屈/冲突，后半段给解决方案/态度。

---

### 1.3 书名总结

- **短标题（2-4字）**：用于正文/书架，追求古典意象和辨识度
- **长标题（11-15字）**：用于首页展示，追求关键词密度和情绪冲击
- **核心意象词库**：绮、靡、惊、华、庭、折、春、漪、烽火、佳人、染指、枭雄

---

## 二、简介结构

### 2.1 标准结构：背景 → 冲突 → 反转/钩子

**示例①：《染指枭雄》**

```
背景：徐白家道中落。未婚夫与名门女暧昧，又不肯放开她。
冲突：徐白因这门婚约，受尽了羞辱与刁难。
解决方案：她搭上了未婚夫的小叔叔，南城有名的混蛋。
钩子（金句）：「你回头是岸。」未婚夫说。「他说他是岸。」男人附耳，「把这岸炸了吧。」
```

**规则**：3-5句话完成"处境→困境→破局→金句收尾"。

---

### 2.2 委屈→逆袭钩子（重生向）

**示例②：《烽火佳人》**

```
冲突铺排：少帅想将她养在私宅——她扇他耳光。
互动升级：他卑微求她——她又扇他耳光。
钩子：颜心重生后，虐渣男丈夫、虐恶毒表妹，又吊打夫家一群吸血鬼。她打人打脸特厉害，大概是在景元钊那里练的，熟能生巧。
```

**示例③：《折春漪》**

```
前世惨：表妹取代了她，家人联手害死了她。
钩子：做鬼十八年，看到了他们的下场，她重生了。这次，她想要活得痛快。
合作钩子：「郡主印换皇后金印，可愿意？」「不愿！」
```

**规则**：前世惨状浓缩1-2句，重生后用短句宣告态度（"她又活了""活得痛快"），结尾用对话金句收束。

---

### 2.3 先抑后扬·主母权谋（古言向）

**示例④：《惊华庭》**

```
背景：程昭被赐婚给最年轻的国公爷周元慎。
冲突：国公有强大娇妾、与寡嫂暧昧、憎恶高门贵女。
女主态度：程昭解决起来游刃有余。
陷阱钩子：「我嫁过来做主母，就是替国公爷收拾烂摊子的。」「待我地位稳固，有了子嗣，下一个被收拾的烂摊子，就是国公爷你。」
结尾钩子：他赞她完美，唯一不好的，是她心中另有所属，不爱他。
```

**规则**：先抛出男主的恶劣条件，再展示女主的从容能力，最后用"温柔一刀"（绵里藏针的内心独白）形成反差钩子。

---

### 2.4 各取所需→男主沦陷（民国向）

**示例⑤：《闪婚，真千金嫁给了军阀大佬》**

```
女主承诺：对他的事装聋作哑，不无理取闹、不索取感情。
婚后状态：两人各取所需、各自敷衍。
钩子（男主破防）：「你在我怀里肆无忌惮怀念你的旧情人，不适合吧？」「我说我心里有你，你耳朵装聋。提到你的旧情人，你不聋了？」
```

**规则**：建立"合作婚姻"的冷静状态，然后用男主的醋意/失控打破平衡，制造情感张力。

---

### 2.5 身份反转式（高概念）

**示例⑥：《绮靡》**

```
背景：宁祯被迫嫁给督军盛长裕。督军有白月光、两房妾室，内宅危机重重。
预期：不到三年，果然如众人预料，宁祯要离婚。
反转：不是她被扫地出门，而是不可一世的督军红眼哀求：「能不能别丢下我？」
```

**规则**：先铺垫"所有条件都指向女主会输"，然后用一句反转彻底颠覆预期。核心是"你以为她会被甩，实际是男主离不开她"。

---

### 2.6 简介金句公式

初点点简介中的金句通常出现在结尾，承担"钩子"功能：

| 类型 | 公式 | 示例 |
|------|------|------|
| 男主破防 | 你以为的强者，在女主面前卑微 | "能不能别丢下我？" |
| 温柔一刀 | 女主表面顺从，内心有算计 | "下一个被收拾的烂摊子，就是国公爷你。" |
| 霸气宣言 | 男主的侵略性台词 | "把这岸炸了吧。" |
| 女主态度 | 用行动定义人物 | "不愿！" |
| 幽默总结 | 用俏皮话收束 | "她打人打脸特厉害，大概是在景元钊那里练的，熟能生巧。" |

## 三、开篇第一段

### 3.1 从"反常时刻"切入

> **《闪婚，真千金嫁给了军阀大佬》**
> 秦言被折腾到后半夜，醒过来时腰酸腿软。床头柜有个公文袋。打开，映入眼帘的是"离婚书"……

**规则**：开局即反常——刚经历缠绵之夜，天亮却收到离婚书。悬念立刻建立。

---

### 3.2 从"冲突/对峙"切入

> **《绮靡》**
> 宁祯的汽车突然停下。一辆汽车横在面前。车门推开，下来一名女郎……繁繁从手袋里掏出一支枪，对准了宁祯。

**规则**：第一段就让反派登场，用"枪"制造紧张感。女主的镇定反应立刻塑造人设。

---

### 3.3 从"前世惨状"切入（重生文）

> **《烽火佳人》**
> 颜心从旖旎梦境中醒来。她上辈子很惨：丈夫偷了她的钱送表妹留学，那是她孩子治病的钱。丈夫、儿子和表妹一起逼死她。

**规则**：重生文开局直接抛出前世惨状，让读者迅速代入仇恨，期待逆袭。

> **《王妃换嫁》**
> 妹妹重生后，抢走了我未婚夫。她想要我的人生。"姐姐，我会做得比你好。"我与妹妹一向不睦，这次却难得宽容，直接让给了她。

**规则**：女主"心甘情愿让出"形成反常识——读者立刻好奇"为什么"。

---

### 3.4 从"设定告知"切入（重生古言）

> **《折春漪》**
> 骆宁为太后挡了一刀，重伤。全家因她富贵荣华。养病三年回来，表妹取代了她一切。家人联手害死了她。她做十八年鬼，看着侯府倒塌，重生了。她又活了。

**规则**：用5-6句完成"前世惨→死→做鬼→重生"的快速交代，然后立刻切入重生后的第一个行动（去找魏公公，不回家）。

---

### 3.5 从"新婚夜反常"切入（古言）

> **《惊华庭》**
> "我知你委屈。"这是程昭的新婚夫君周元慎跟她说的第一句话。"待将来明朗，拨乱反正，助你和离再嫁。"说完，他走了。

**规则**：新婚夜新郎说要和离——极致的反常设定，一句话抓住读者。

---

### 3.6 从"偶遇事件"切入（民国）

> **《染指枭雄》**
> 徐白为退婚烦恼。倏然楼下传来撕心裂肺的嘶喊声、打斗声、枪声。她瞧见了为首的男人——他身材高大，眼眸如点漆，璀璨明媚。

**规则**：女主先展示困境，然后通过突发事件引入男主，用场景化的暴力场面树立男主危险气质。

---

### 3.7 开篇原则总结

| 原则 | 说明 |
|------|------|
| 第一句制造疑问 | "离婚书""枪""助你和离"——第一句就让人想知道"为什么" |
| 冲突前置 | 不给背景铺垫，直接扔矛盾 |
| 女主立即行动 | 不沉浸于内心独白，立刻做出选择或反应 |
| 建立核心人设 | 秦言的冷静、宁祯的镇定、程昭的隐忍——第一段就让人看出女主性格 |

## 四、标签策略

### 4.1 标签分布

| 作品 | 标签 |
|------|------|
| 闪婚，真千金嫁给了军阀大佬 | 现代言情\|总裁\|先婚后爱\|民国 |
| 绮靡 | 现代言情\|民国\|大佬 |
| 王妃换嫁 | 重生\|打脸逆袭\|架空 |
| 烽火佳人 | 现代言情\|重生\|民国 |
| 染指枭雄 | 现代言情\|总裁\|民国\|豪门世家 |
| 惊华庭 | 古代言情\|古色古香 |
| 折春漪 | 古代言情\|重生\|王妃\|前世今生 |

### 4.2 标签规律

1. **民国文核心标签**：`现代言情|民国|大佬/总裁` —— 初点点民国文最多，男主身份集中在"少帅/督军/军阀"
2. **古言核心标签**：`古代言情|古色古香` —— 或 `重生|王妃|前世今生`
3. **重生标签**：重生文占多数，打脸逆袭是标配
4. **人设标签**：先婚后爱、大佬、总裁、豪门世家 —— 强调强男主+契约关系的设定
5. **不加"甜宠"标签**：初点点作品偏现实向拉扯，不标甜宠

### 4.3 标签组合建议

```
民国组：现代言情 + 民国 + 大佬/总裁 + 先婚后爱/豪门世家
古言组：古代言情 + 古色古香 + 王妃/重生/前世今生
重生组：(现代/古代)言情 + 重生 + 打脸逆袭
```

## 五、综合规则

| 维度 | 规则 |
|------|------|
| 书名·短 | 2-4字古典意象词，留白让读者想象 |
| 书名·长 | 11-15字逗号式，`[委屈]？[靠山]，[态度]` |
| 简介结构 | 背景→冲突→解决方案→金句钩子 |
| 简介金句 | 结尾用一句反常识对话/宣言收束 |
| 开篇 | 反常时刻/突发冲突，第一句制造悬念 |
| 女主 | 冷静/隐忍/有城府，不被情绪裹挟 |
| 男主 | 强势/有侵略性，但最终被女主驯服 |
| 标签组合 | 民国组/古言组/重生组，不标甜宠 |
