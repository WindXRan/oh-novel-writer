**典型高潮模式：**
```

```

### 2.4 断点策略

**分析样本：惜花芷第1-3章、第347-349章**

| 断章类型 | 出现频率 | 示例 |
|----------|----------|------|
| **悬念断章** | ~45% | "手都在微微打着颤"（惜花芷-1结尾） |
| **场景切换** | ~30% | 从花芷视角切换到顾晏惜（惜花芷-349） |

**断章特征：**
1. **不写完**：在事件进行中切断，而非完成后
2. **情绪钩子**：在情绪高点或转折处断开
3. **视角切换**：常在切换视角前断章
4. **对话中断**：对话进行中或刚结束时断开

**典型断章句式：**
- 动作未完成："手都在微微打着颤"
- 悬念抛出："心底有些不合时宜的庆幸"

## 四、章节模板总结

### 4.1 标准章节结构

```
[开篇：场景/对话/动作引入，3-5行内进入正题]
    ↓
[发展：事件推进，信息投放，对话驱动]
    ↓
[高潮：冲突/转折/情感高点，通常在章末70-90%处]
    ↓
[断章：在高潮点或悬念处切断，不写完]
```

### 4.2 字数控制

- **目标字数**：2000-2500字（98.5%的章节在此区间）
- **绝对下限**：1670字
- **绝对上限**：5543字（特殊情况）

### 4.3 命名规范

- **首选**：四字短语（58.4%）
- **次选**：2-3字或5字短语
- **点缀**：疑问句（4%）制造悬念
- **避免**：过长标题（>7字仅占6.3%）

### 4.4 断章技巧

1. **悬念断章**：在冲突最高点切断（45%）
2. **场景切换**：在视角转换前断开（30%）
3. **情绪留白**：在情感高点后断开（15%）
4. **事件未完成**：不写完，留到下一章（10%）

### 4.5 事件密度

- **每章核心事件**：2-3个
- **信息投放**：线性推进为主，穿插并行
- **节奏周期**：3-5章一个小循环，10-15章一个中循环
