# 统一评分标准

所有写作类 skill（story-long-write / story-short-write / story-rewrite / story-review / story-deslop）共用此评分体系。

---

## 评分维度

| 维度 | 权重 | 子项 | 来源 |
|------|------|------|------|
| 结构一致性 | 30% | 节奏对齐(12%) + 转折点对齐(10%) + 场景切换对齐(8%) | story-rewrite |
| 原创性 | 25% | 角色原创(10%) + 背景原创(8%) + 细节原创(7%) | story-rewrite |
| 去AI度 | 20% | 禁用词(5%) + 句式(4%) + 心理词(4%) + 对话标签(4%) + 段落密度(3%) | story-deslop |
| 情绪弧线 | 15% | 代入感(5%) + 节奏(5%) + 情绪起伏(5%) | story-long-write |
| 逻辑自洽 | 10% | 动机合理(5%) + 转折有铺垫(5%) | story-rewrite |

---

## 评分等级

| 等级 | 分数 | 含义 | 处理 |
|------|------|------|------|
| A | 90-100 | 优秀 | 可直接使用 |
| B | 80-89 | 良好 | 小改即可 |
| C | 70-79 | 及格 | 需较大修改 |
| D | <70 | 不及格 | 需要重写 |

---

## 去AI度客观指标

以下指标来自 `story-deslop`，用于量化评估 AI 痕迹：

| 指标 | 阈值 | 扣分规则 |
|------|------|----------|
| 一级禁用词 | 0 个 | 每个 -5 分 |
| 二级高频词 | ≤3 个/千字 | 超出每个 -2 分 |
| 连续排比 | ≤1 处 | 每处 -5 分 |
| 心理词占比 | ≤5% | 超出每 1% -3 分 |
| 对话标签密度 | ≤10% | 超出每 1% -2 分 |
| 平均段落句数 | 2-4 句 | 偏离每句 -1 分 |

### 一级禁用词（必须替换）

- 愣了一下
- 心跳漏了一拍
- 脸一下子红了
- 深吸一口气
- 嘴角微微弯了一下
- 命运齿轮
- 如潮水般
- 仿佛春风
- 心猛地一沉
- 眼眶泛红

### 二级高频词（控制密度）

- 好像、仿佛、突然、瞬间
- 不禁、微微、轻轻、淡淡
- 似乎、忽然、猛然

---

## 扣分规则

| 问题 | 扣分维度 | 扣分值 |
|------|----------|--------|
| 场景抄袭 | 原创性 | -20 |
| 金手指污染 | 原创性 | -30 |
| 逻辑硬伤 | 逻辑自洽 | -10 |
| 糖点稀疏 | 情绪弧线 | -10 |
| AI痕迹重度 | 去AI度 | 该维度≤60 |

---

## 使用方式

### 主线程评估

```
请根据 _shared/evaluation/rubric.md 对正文进行评分。
输出格式：
- 结构一致性：X/30
- 原创性：X/25
- 去AI度：X/20
- 情绪弧线：X/15
- 逻辑自洽：X/10
- 总分：X/100
- 等级：A/B/C/D
```

### narrative-writer 自检

```
写完后根据 _shared/evaluation/rubric.md 的去AI度指标自检：
1. 检查一级禁用词，有则替换
2. 统计二级高频词密度，超标则改写
3. 检查连续排比，有则打散
4. 统计心理词占比，超标则外化为身体状态
5. 统计对话标签密度，超标则删除或改为动作
6. 检查平均段落句数，偏离则调整
```
