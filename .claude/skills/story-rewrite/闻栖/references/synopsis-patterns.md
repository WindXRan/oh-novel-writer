# 闻栖 · 书名简介写作规则（RIA++ 结构化）

## 一、书名：两段式断句+身份反差

### R（原文引用）
> 「书名偏好12字，两段式断句，前半句≤6字设情境，后半句≤8字给冲突，中间用逗号/问号断开。人物身份词+核心设定+冲突/反差，三要素缺一不可。」
> ——闻栖·书名规则

### I（决策解读）
闻栖的书名不是文艺表达，而是**广告语**。她把书名当作"第一句简介"来设计：用身份词锁定目标读者，用反差动词制造点击欲望，用断句制造节奏感。问号断句是她的高频手法，直接在标题里植入疑问，逼读者点进去找答案。

### A1（书中案例）
- 《女配一睁眼，失忆男主冷脸洗床单》—「一睁眼」暗示穿越，「冷脸洗床单」是总裁做家务的身份×行为反差
- 《分手了？秦少火速领证上位》— 问号制造疑问，「火速领证」是爽感动词
- 《另嫁落魄权臣，他清冷人设崩了》—「另嫁」暗示前世/重生，「人设崩了」是反差钩子

### A2（触发场景）
- 如果写穿书/重生文 → 书名用「一睁眼+角色身份+反差行为」结构
- 如果写豪门总裁文 → 书名用「人物身份词+反常状态」结构
- 如果写古言文 → 书名用「权臣/相公+人设崩了」结构
- 如果想制造疑问感 → 书名中间用问号断句

### E（可执行步骤）
1. 确定身份词（女配/总裁/权臣/前夫）锁定题材信号
2. 找到反差行为（冷脸洗床单/人设崩了/火速领证）
3. 用逗号或问号断句，控制在10-15字

### B（边界与盲点）
- 书名避免纯文艺表达，必须有具象动词或画面感词汇
- 书名避免超过15字，手机端会被截断
- 身份词≠人名，不要用角色名当书名（读者不认识）

## 二、简介开头：标签墙+身份速递

### R（原文引用）
> 「简介第一行必须是标签墙，用【】包裹，3-6个标签，格式：时代+核心梗+情绪+安全牌。第一句用『前世/原书/一朝穿越』快速交代主角身份和来源，20字以内说完。」
> ——闻栖·简介开头规则

### I（决策解读）
闻栖把简介开头当作"筛选器"：标签墙过滤掉非目标读者（避免差评），身份速递用最少字数让读者进入故事。她不追求"优美开头"，追求的是"零门槛理解"——读者扫一眼就知道：什么时代、什么梗、什么情绪、主角是谁。

### A1（书中案例）
标签墙示例：
- 【双洁/同居日常/小甜文】
- 【男二上位+追妻火葬场+闪婚+暗恋+狗血】

身份速递示例：
- 「前世，沈禾因生的漂亮，被选为太子侍寝宫女」（另嫁落魄权臣）
- 「一朝穿越，宋云绯成了假冒失忆总裁女友的女配」（女配一睁眼）

### A2（触发场景）
- 如果写简介 → 第一行必须是【标签墙】，用3-6个标签
- 如果写穿越/重生文 → 第一句用「前世/原书/一朝穿越」开头，20字内交代来源
- 如果写甜宠文 → 标签必须包含「甜宠」或「双洁」

### E（可执行步骤）
1. 写标签墙：时代+核心梗+情绪+安全牌，4层结构
2. 写身份速递：用「前世/原书/一朝穿越」开头，一句话交代主角是谁、从哪来
3. 总字数控制在20字以内

### B（边界与盲点）
- 标签之间不要重复语义（如「甜宠」和「小甜文」只选一个）
- 标签墙不是越多越好，3-6个最佳，多了反而模糊定位
- 安全牌（双洁/HE）是给读者的"承诺"，写了就要兑现

## 三、简介主体：困境→反转→反差三段式

### R（原文引用）
> 「简介必须包含『困境→反转→反差』三段式：先写原书命运（惨），再写穿越/重生后的转折，最后写男主的反常行为。倒数第二段加入男主视角的金句或反差行为。」
> ——闻栖·简介三段式规则

### I（决策解读）
闻栖的简介不是"讲故事"，而是"制造期待"。三段式的核心逻辑是：先让读者心疼（困境），再给希望（反转），最后用男主的反常行为吊胃口（反差）。她把最有力的钩子放在倒数第二段——男主视角的金句，让读者好奇"这个男人到底怎么回事"。

### A1（书中案例）
《和落魄反派的同居日常》三段式：
1. 困境：「剧情告诉她，她还剩半年的时间，然后就会莫名其妙的死掉」
2. 转折：「没想到，最后却是她强迫来的落魄男友，将她收留」
3. 反差：「自己负债累累，穷的叮当响，还要赚钱养她」

男主视角钩子：
- 「恢复记忆后的楚靳寒，好友问他：她骗你吃了这么多苦，你打算怎么报复她？楚靳寒淡淡道：她爱我才会骗我」（女配一睁眼）

### A2（触发场景）
- 如果写简介主体 → 必须包含「困境→反转→反差」三段式
- 如果想增加男主魅力 → 倒数第二段插入男主视角的金句或反常行为
- 如果写穿书文 → 困境用"原书剧情"交代，反转用"这次不一样"推进

### E（可执行步骤）
1. 写困境：用一句话交代主角在原书/前世的悲惨命运
2. 写反转：用「没想到/这次/然而」引出转折
3. 写反差：聚焦男主的反常行为（为她破例/为她疯/为她改变）

### B（边界与盲点）
- 困境不要写太长，一两句话即可，留悬念给正文
- 反差行为要具体（冷脸洗床单），不要抽象（对她很好）
- 男主视角金句是"钩子中的钩子"，不能省略

## 四、简介结尾：问句/感叹句悬念收尾

### R（原文引用）
> 「简介末尾必须用问句或感叹句收尾，制造悬念。避免句号结尾，句号=结束感，读者会划走。」
> ——闻栖·简介结尾规则（频率：83%，5/6使用）

### I（决策解读）
闻栖把简介结尾当作"未完待续"的信号。句号意味着"故事说完了"，读者会毫不犹豫地划走；问号或感叹号意味着"故事还没完"，读者会产生"然后呢？"的好奇心，进而点开正文。这是最简单但最有效的"留钩子"手法。

### A1（书中案例）
- 「陈汐傻眼了，这离还是不离呢？」（一睁眼，前夫）
- 「骗完我就想跑？」（女配一睁眼）
- 「那天，姜霖疯了」（分手了？）

### A2（触发场景）
- 如果写简介结尾 → 必须用问句或感叹句，禁止句号
- 如果想制造选择悬念 → 用「A还是B」的问句（如"这离还是不离呢？"）
- 如果想制造情绪冲击 → 用短句感叹句（如"那天，姜霖疯了"）

### E（可执行步骤）
1. 写完简介主体后，删掉最后一个句号
2. 用问句重写（如"这X还是不X呢？""XX就完了？"）
3. 或用感叹句收尾（如"那天，XX疯了"）

### B（边界与盲点）
- 问句不要太开放（如"她会幸福吗？"太泛），要指向具体冲突
- 感叹句不要太长，短句冲击力最强
- 83%的使用率意味着有17%的情况可以不用，但至少要尝试问句结尾

## 五、标签组合：四层筛选公式

### R（原文引用）
> 「标签组合公式：时代+核心梗+情绪+安全牌（4层结构）。第1层时代，第2层核心金手指或设定，第3层情绪类型，第4层读者安全牌。」
> ——闻栖·标签选择策略

### I（决策解读）
闻栖的标签不是"关键词堆砌"，而是"读者筛选器"。四层结构的逻辑是：先用时代筛选类型偏好（现代/古代），再用核心梗筛选题材偏好（穿越/重生），再用情绪筛选阅读期待（甜宠/虐恋），最后用安全牌降低决策成本（双洁/HE）。每一层都在缩小目标读者范围。

### A1（书中案例）
| 书名 | 标签 |
|------|------|
| 女配一睁眼 | 现代言情\|穿越\|甜宠\|大佬 |
| 和落魄反派 | 现代言情\|甜宠\|反派\|双洁 |
| 另嫁落魄权臣 | 古代言情\|重生\|前世今生\|古色古香 |
| 分手了？ | 现代言情\|闪婚\|暗恋\|HE |

### A2（触发场景）
- 如果写甜宠文 → 标签必须包含「甜宠」或「双洁」
- 如果写穿越/重生文 → 标签必须包含「穿越」或「重生」
- 如果标签不够4个 → 优先补安全牌（双洁/HE/大佬）

### E（可执行步骤）
1. 第1层选时代：现代言情/古代言情
2. 第2层选核心梗：穿越/重生/系统/闪婚
3. 第3层选情绪：甜宠/暗恋/反派
4. 第4层选安全牌：双洁/HE/大佬

### B（边界与盲点）
- 标签之间不要重复语义（如「甜宠」和「小甜文」只选一个）
- 安全牌是"承诺"，写了「双洁」正文就必须双洁，否则会被骂
- 标签不是越多越好，4个最佳，超过6个反而模糊定位
