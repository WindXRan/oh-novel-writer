## 审稿流程

### 快速判定表

```
这个问题影响阅读体验吗？
├── 是 → 必须改
│   ├── 致命错误？→ 推翻重写
│   ├── 严重问题？→ 大改
│   └── 建议修改？→ 小改
└── 否 → 可以不改
    ├── 有风格特色？→ 保留
    └── 无伤大雅？→ 看心情
```

### 审稿步骤

**第一遍：章名→章尾→章首**
1. 先看章名——有没有悬念？有没有情绪词？是不是4-7字？
2. 再看章尾——有没有钩子？是悬念、甜蜜还是反转？
3. 最后看章首——是不是对话开场？有没有即时感？

**第二遍：通读，标记问题**
- [节] 节奏塌了（连续3个以上动作没有情绪推进）
- [人] 人设可疑（角色做出跟前面不符的行为）
- [对] 对话太长（关键对话超过12字/没有潜台词）
- [情] 情绪堆砌（连续2个以上情绪形容词）
- [钩] 钩子弱了（章尾没有悬念/甜蜜/反转）

### 重点看什么

1. **开篇3章**——必须完成"困境→相遇→绑定"，信息分层释放
2. **情感高潮章**——必须用延迟满足（3-5步递进），不能一步到位
3. **转折点章节**——必须有信息差或第三方介入，不能靠误会推动
4. **日常缓冲章**——必须承载人物关系进展，不能是流水账
5. **结尾章节**——必须留开放性尾巴，不能完美收束

## 去AI检查清单

写完正文后，按以下步骤去AI：

1. **删连接词**：扫描「然而」「此外」「因此」「综上所述」，删除或替换为「但是」「不过」「结果」「所以」
2. **拆长句**：扫描超过25字的句子，在逗号处拆成2-3个短句
3. **得地的**：将「得」「地」统一改为「的」
4. **加微重复**：在情绪激动处加入词语重复（"完了完了"）或字词重复（"穿、穿书了？"）
5. **加模糊词**：在心理活动前加「好像」「似乎」「大概」，删除「必然」「显然」「无疑」
6. **降级副词**：将「微微一笑」改为「一笑」，「缓缓开口」改为「开口」
7. **标点后处理**：情绪激动处用「？！」，犹豫处用省略号，反问处用问号

## 写作检查清单

### 写之前问自己
- 这章的核心事件是什么？（不能是「关系升温」这种空话，必须是具体事件）
- 开篇用什么钩子？（对话开场/感官开场/内心独白开场，三选一）
- 结尾断在哪里？（悬念断章/情感落点断章/动作中断断章，三选一）
- 信息差是什么？（谁知道了什么？谁不知道？）
- 这章属于什么类型？（日常章/冲突章/高潮章/过渡章）

### 写完后检查
- [ ] 章名在4-7字范围内，口语化，疑问句占比≥20%
- [ ] 开篇第一句是对话或声音或感官描写（不是叙述）
- [ ] 结尾有钩子（悬念/情感/动作，不能用句号收尾）
- [ ] 核心事件2个（双事件节拍），不是流水账
- [ ] 没有使用反模式词汇（此外/因而/综上所述/必然/显然/无疑/必定/俊美无俦/说时迟那时快）
- [ ] 没有超过25字的长句未拆分
- [ ] 心理描写不超过3句，用身体反应替代
- [ ] 对话有潜台词层，不是直白表达
- [ ] 没有角色直接说"我爱你"
- [ ] 情绪基调与章纲一致
- [ ] 动作描写具体到可拍摄程度
- [ ] 经济数字具体到个位（50块，不是"几十块"）
- [ ] 去AI检查：无正式连接词、长句已拆、有微重复、有模糊词、得地的已统一
