## AI痕迹检测（维度14扩展）

### 11条确定性规则

| 规则 | 检测内容 | 阈值 |
|------|----------|------|
| 1 | 连接词过频（然而、但是、因此） | 每1000字>5次 |
| 2 | 转折词过频（虽然...但是） | 每1000字>3次 |
| 3 | "了"字过频 | 每100字>8个 |
| 4 | 句式规律（连续3句以上结构相同） | 出现即扣分 |
| 5 | 段落长度单一（连续5段长度相近） | 出现即扣分 |
| 6 | 情绪标签化（感到、觉得） | 每1000字>3次 |
| 7 | 叙述者结论（显然、当然） | 出现即扣分 |
| 8 | 群像反应笼统（全场震惊） | 出现即扣分 |
| 9 | fatigue词 | 出现即扣分 |
| 10 | 抽象描述（强大的力量） | 出现即扣分 |
| 11 | 重复表达（同一段落相同词汇3次以上） | 出现即扣分 |

## 评分标准

### 单维度评分

- 9-10分：优秀
- 7-8分：良好
- 5-6分：及格
- 3-4分：较差
- 1-2分：不合格

### 总分计算

```
总分 = (维度1×权重 + 维度2×权重 + ... + 维度33×权重) / 总权重
```

### 通过标准

- **必须通过**：所有高权重维度≥7分
- **建议通过**：中权重维度≥5分
- **自动修复**：critical_issues中的问题必须修复
