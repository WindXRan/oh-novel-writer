## 一、对话指标

### 1. 对话占比

| 项目 | 内容 |
|------|------|
| **指标名称** | 对话字符占比 |
| **统计方法** | `对话字符数 / 总字符数 × 100%`（引号内所有字符，含引号本身） |
| **作者基线** | **30%**（绮靡 30.25%，染指枭雄 29.59%，SKILL.md 记载 25-30%） |
| **正常范围** | 25-35% |
| **🟡 建议修改** | <20% 或 >40% |
| **🔴 必须修改** | <15% 或 >50% |
| **关联处方** | <20% → 压缩旁白，增加对话驱动叙事（参考「对话即叙事」技法）；>40% → 增加行为描写和环境锚点，减少纯对话堆叠 |

### 2. 对话标签密度

| 项目 | 内容 |
|------|------|
| **指标名称** | 对话标签（说道/轻声道/低声道等）出现密度 |
| **统计方法** | `对话标签出现次数 / 总字数(百字)` |
| **作者基线** | **0.03 次/百字**（绮靡 0.035，染指枭雄 0.026） |
| **正常范围** | 0.01-0.05 次/百字 |
| **🟡 建议修改** | >0.08 次/百字（每章约 1.6 次） |
| **🔴 必须修改** | >0.12 次/百字（每章约 2.4 次） |
| **关联处方** | 去掉「说道/轻声道/低声道」标签，改用动作引出对话或直接写对话（参考反模式：句式反模式） |

## 二、节奏指标

### 3. 章节字数波动

| 项目 | 内容 |
|------|------|
| **指标名称** | 单章字数变异系数（CV = 标准差 / 均值） |
| **统计方法** | `CV = σ / μ × 100%`（按章分割后统计每章字符数） |
| **作者基线** | **CV ≈ 7.6%**（绮靡均值 2554 字/章、σ=194；染指枭雄均值 2530 字/章、σ=191） |
| **正常范围** | CV < 15%（章字数在 2100-3000 之间） |
| **🟡 建议修改** | CV > 20%（出现 <1800 或 >3500 的异常章节） |
| **🔴 必须修改** | CV > 30%（出现 <1500 或 >4500 的极端章节） |
| **关联处方** | 过短章 → 合并或补充核心事件至 2000+ 字；过长章 → 拆分为两章，每章保留独立钩子 |

### 4. 情绪周期长度

| 项目 | 内容 |
|------|------|
| **指标名称** | 情绪小高潮间隔章数 |
| **统计方法** | 两个情绪高点之间的章节数 |
| **作者基线** | **短周期 3-5 章，中周期 10-15 章，长周期 30-50 章**（SKILL.md 记载） |
| **正常范围** | 短周期 3-5 章，中周期 10-15 章 |
| **🟡 建议修改** | 短周期 >7 章（连续平淡无小高潮）或中周期 >20 章（情节线拖沓） |
| **🔴 必须修改** | 短周期 >10 章（读者可能弃书）或中周期 >30 章（情节线失控） |
| **关联处方** | 周期过长 → 在当前位置插入冲突事件或信息差揭示，制造小高潮（参考「信息差制造紧迫感」） |

## 三、表达指标

### 5. 副词密度

| 项目 | 内容 |
|------|------|
| **指标名称** | 高频副词（微微/轻轻/淡淡/缓缓/稍稍/略略）出现密度 |
| **统计方法** | `高频副词出现次数 / 总字数(百字)` |
| **作者基线** | **0.27 次/百字**（5本原著合计 2684 次 / 98 万字，去AI策略已验证；每章约 5.4 次） |
| **正常范围** | 0.10-0.30 次/百字 |
| **🟡 建议修改** | >0.35 次/百字（每章约 7 次） |
| **🔴 必须修改** | >0.50 次/百字（每章约 10 次） |
| **关联处方** | 触发降级副词模块：「微微蹙眉」→「蹙了蹙眉」，「轻轻放下」→「放下」，「淡淡道」→「冷冷道」或删除副词。对话中的副词可保留。 |

### 6. 连接词密度

| 项目 | 内容 |
|------|------|
| **指标名称** | 正式连接词（然而/但是/因此/所以/不过/而且/同时）出现密度 |
| **统计方法** | `连接词出现次数 / 总字数(百字)` |
| **作者基线** | **0.30 次/百字**（5本原著合计 2967 次 / 98 万字，去AI策略已验证） |
| **正常范围** | 0.10-0.40 次/百字 |
| **🟡 建议修改** | >0.60 次/百字 |
| **🔴 必须修改** | >1.00 次/百字 |
| **关联处方** | 删除或替换连接词，用语义自然衔接替代逻辑连接。注意：「不过」在初点点原文中多为口语用法（「不过是个……」），非书面连接词，可保留。 |

### 7. 句长分布

| 项目 | 内容 |
|------|------|
| **指标名称** | 平均句长 + 长句（>25字）占比 |
| **统计方法** | 平均句长 = `总字数 / 句子数`；长句占比 = `>25字的句子数 / 总句子数 × 100%` |
| **作者基线** | **平均句长 15-18 字，长句占比 10-15%**（去AI策略已验证） |
| **正常范围** | 平均句长 12-20 字，长句占比 <20% |
| **🟡 建议修改** | 平均句长 >22 字 或 长句占比 >25% |
| **🔴 必须修改** | 平均句长 >28 字 或 长句占比 >35% |
| **关联处方** | 拆分长句为短句，保持初点点的短句驱动节奏风格。>25字的句子拆为两个独立句或一个短句+一个半句话。 |

## 四、结构指标

### 8. 章首钩子覆盖率

| 项目 | 内容 |
|------|------|
| **指标名称** | 章节开篇使用钩子（对话/动作/悬念）的比例 |
| **统计方法** | `有钩子的章节数 / 总章节数 × 100%`（对话引号出现在章节前 80 字内视为对话开篇） |
| **作者基线** | **对话开篇 47%**（绮靡前 200 章统计：94/200），SKILL.md 记载目标 60% |
| **正常范围** | 对话开篇 40-65%，动作/描写开篇 20-40% |
| **🟡 建议修改** | 对话开篇 <30%（叙事节奏偏慢）或纯描写开篇 >40%（缺少紧迫感） |
| **🔴 必须修改** | 对话开篇 <20% 或无钩子开篇 >30%（读者流失风险） |
| **关联处方** | 参考章纲模板：对话开篇 60%、动作开篇 25%、描写开篇 15%。承接前章冲突时优先用对话开篇。 |

### 9. 感叹号密度

| 项目 | 内容 |
|------|------|
| **指标名称** | 感叹号（！）出现频率 |
| **统计方法** | `感叹号数 / 总字数(万字)` |
| **作者基线** | **5-10 次/万字**（绮靡 10.09，染指枭雄 5.44，去AI策略建议全文 ≤5 个感叹号/章） |
| **正常范围** | 3-12 次/万字 |
| **🟡 建议修改** | >15 次/万字（每章约 3 个感叹号） |
| **🔴 必须修改** | >25 次/万字（每章约 5 个感叹号） |
| **关联处方** | 标点后处理：感叹号替换为句号，保留关键情绪节点的感叹号。去AI策略建议全文感叹号 ≤5 个/章。 |

## 五、情绪指标

### 10. 情绪表达方式比例

| 项目 | 内容 |
|------|------|
| **指标名称** | 行为展示 vs 直接命名的情绪表达比例 |
| **统计方法** | `行为展示类情绪描写数 / (行为展示 + 直接命名) × 100%`<br>行为展示：动作、表情、生理反应（攥手指、咬唇、手抖）<br>直接命名：「她很愤怒」「她很痛苦」「她很开心」 |
| **作者基线** | **行为展示 80% : 直接命名 20%**（SKILL.md 记载） |
| **正常范围** | 行为展示 70-85% |
| **🟡 建议修改** | 直接命名 >35%（行为展示 <65%） |
| **🔴 必须修改** | 直接命名 >50%（行为展示 <50%） |
| **关联处方** | 将「她很愤怒」替换为具体动作（「她攥紧了拳头」「她咬着唇，指节发白」）。参考「克制的情感表达」技法：用行动代替言语、迂回比喻、沉默+动作。 |

## 六、一致性指标

### 11. 主谓宾结构占比

| 项目 | 内容 |
|------|------|
| **指标名称** | 标准主谓宾句式在所有句子中的占比 |
| **统计方法** | `主谓宾句式数 / 总句子数 × 100%`（需 NLP 句法分析，或抽样人工统计） |
| **作者基线** | **70-75%**（去AI策略已验证，低于 90% 阈值） |
| **正常范围** | 65-80% |
| **🟡 建议修改** | >85%（语序过于规律，缺乏变化） |
| **🔴 必须修改** | >92%（AI 特征明显，机械排列） |
| **关联处方** | 轻微倒装模块：在描写段落加入状语前置（「三月下旬的阳光明艳，官道两侧柳条款摆」），全文 3-5 句，仅限描写段落，对话不倒装。 |

### 12. 高潮段单句成段占比

| 项目 | 内容 |
|------|------|
| **指标名称** | 高潮场景中单句成段的比例 |
| **统计方法** | `高潮段单句成段数 / 高潮段总段落数 × 100%`（单句成段：独立一行、2-15字的短句） |
| **作者基线** | **40%+**（SKILL.md 记载） |
| **正常范围** | 35-50% |
| **🟡 建议修改** | <25%（高潮冲击力不足）或 >60%（滥用单句成段，冲击力递减） |
| **🔴 必须修改** | <15%（高潮场景缺乏视觉冲击）或 >70%（单句成段泛滥） |
| **关联处方** | <25% → 找到高潮核心信息，压缩为 2-15 字单独成段（「枪响。」「她没有笑。」）；>60% → 合并部分单句为复合句，保留最关键的 2-3 处。每章单句成段不超过 2-3 处。 |
