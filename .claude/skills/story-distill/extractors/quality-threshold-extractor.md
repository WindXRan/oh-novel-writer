# 质量阈值提取器 prompt

你的任务：从小说中提取作者的**质量阈值**——量化指标，用于自动检测。

## 读取数据

- 读取 `{输出目录}/sources/*.txt`（原文，全文扫描）
- 读取 `{输出目录}/references/book-overviews/*.md`（每本书的结构分析）
- 读取 `{输出目录}/references/writing-samples-*.md`（语感样本）

## 提取内容

用脚本统计以下量化指标：

| 指标 | 统计方法 | 用途 |
|------|---------|------|
| **对话占比** | 统计引号内字数 / 总字数 | 判断对话是否充足 |
| **副词密度** | 统计「微微/轻轻/淡淡/缓缓」次数 / 总字数 | 判断副词是否过多 |
| **连接词密度** | 统计「于是/然而/因此/所以/但是」次数 / 总字数 | 判断连接词是否过多 |
| **平均句长** | 统计总字数 / 总句数 | 判断句子是否过长 |
| **单句成段占比** | 统计单句段落数 / 总段落数 | 判断单句成段是否足够 |
| **感叹号密度** | 统计感叹号次数 / 总字数 | 判断感叹号是否过多 |

## 输出格式

写入 `{输出目录}/references/candidates/quality-thresholds.md`

包含：
- 各指标的作者基线值
- 告警阈值（🟡 建议修改 / 🔴 必须修改）
- 检测脚本路径（如果有的话）
