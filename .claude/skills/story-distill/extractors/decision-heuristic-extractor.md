# 决策启发式提取器 prompt

你的任务：从小说中提取作者的决策启发式。

## 读取数据

- 读取 `{输出目录}/references/book-overviews/*.md`（每本书的结构分析）
- 读取 `{输出目录}/sources/*.txt`（原文）

**精读章节定义**：
- 前3章（开篇）
- 第N/4章±1章（1/4处）
- 第N/2章±1章（中段）
- 第3N/4章±1章（3/4处）
- 最后3章（结局）
- 全书转折点（book-overviews 中标注的）

## 提取内容

提取 **8-15 条**决策规则，覆盖以下7类（每类至少1条，可多不可少）：

| 类型 | 从哪里找 | 找什么 |
|------|---------|--------|
| **开篇决策** | 前3章 | 怎么开场？用什么钩子？ |
| **章尾断点决策** | 每章结尾 | 在哪里断开？用什么钩子？ |
| **信息揭示决策** | 关键揭秘场景 | 什么时候揭示真相？什么时候延迟？ |
| **节奏控制决策** | 紧张/舒缓交替处 | 紧张和舒缓怎么交替？ |
| **角色决策** | 角色高光/低谷场景 | 角色情绪激动时怎么写？角色隐瞒时怎么写？ |
| **冲突设计决策** | 冲突升级/解决场景 | 冲突怎么升级？怎么解决？ |
| **爽点设计决策** | 打脸/逆袭/发糖场景 | 爽点怎么铺垫？怎么释放？ |

**单本书时**：从同一本书的多个场景中找重复模式（≥3次出现）。

## 每条规则必须包含

| 维度 | 要求 | 示例 |
|------|------|------|
| **规则名称** | 一句话概括 | 「危机开场规则」 |
| **R（原文引用）** | 直接引用书中段落，≤150字，附书名+章节号 | 见下方示例 |
| **I（决策解读）** | 作者在这里做了什么选择 | 「用他人的对话碎片开场，让读者和主角一起困惑」 |
| **A1（书中案例）** | 至少1个具体应用 | 「《惜花芷》第1章」 |
| **A2（触发场景）** | 「如果{条件}，则{行动}」格式 | 「如果需要快速建立代入感，则用危机开场」 |
| **E（可执行步骤）** | 1-3步 | 「1. 写一段他人对话 2. 让读者不知道在说谁 3. 揭示主角身份」 |
| **B（边界说明）** | 至少1条局限性 | 「不适合需要先交代背景的故事」 |

### 案例深度增强（可选）

对关键规则，补充「反事实分析」：

| 维度 | 说明 | 示例 |
|------|------|------|
| **场景** | 当时的具体情境 | 「第1章，读者刚进入故事世界」 |
| **选择** | 作者实际做了什么 | 「用对话碎片开场，不交代背景」 |
| **效果** | 这个选择带来了什么结果 | 「读者和主角一起困惑，产生代入感」 |
| **反事实** | 如果作者做了不同选择会怎样 | 「如果用旁白交代背景，读者会提前知道信息，失去探索感」 |

**注意**：反事实是推断，标注「推断」。

## 输出示例

```markdown
## 开篇决策

### 规则：危机开场

**R（原文引用）**：
> 造孽哦，家门不幸啊。爹爹，四叔叔他们真的死了吗。
> ——《一睁眼前夫成了我懒汉相公》第1章

**I（决策解读）**：
作者选择用「他人对话碎片」开场，而不是用旁白交代背景。读者一开始不知道在说谁、发生了什么，被迫和主角一起探索。这比「XXX是一个怎样的人」开场更有代入感。

**A1（书中案例）**：
- 《一睁眼前夫》第1章：用「造孽哦」开场，读者不知道在说谁
- 《分手了秦少》第1章：用「对不起，您拨打的电话暂时无人接听」开场，读者不知道发生了什么

**A2（触发场景）**：
- 如果需要快速建立代入感，则用危机/困惑开场
- 如果不希望读者提前猜到故事走向，则用信息盲区开场

**E（可执行步骤）**：
1. 第1章第1段写一段他人对话或环境碎片
2. 不解释上下文，让读者困惑
3. 第2-3段逐步揭示主角身份和处境

**B（边界说明）**：
- 不适合需要先交代世界观的架空设定
- 不适合悬疑类（悬疑需要读者比角色知道得多）
```

## 硬性要求

- 每条规则必须有原文引用，不能凭空编造
- 规则必须是「如果X，则Y」格式，不能是泛泛的建议
- 跨书出现≥2次的规则优先标注；单本书时从同书多场景中找重复
- 输出写入 `{输出目录}/references/candidates/decision-heuristics.md`
