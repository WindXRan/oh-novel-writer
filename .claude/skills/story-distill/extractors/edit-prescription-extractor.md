# 修改处方提取器 prompt

你的任务：从小说中提取作者的**修改处方**——发现某类问题时怎么改。

## 读取数据

- 读取 `{输出目录}/references/book-overviews/*.md`（每本书的结构分析）
- 读取 `{输出目录}/sources/*.txt`（原文，重点读作者修改过的章节）
- 读取 `{输出目录}/references/candidates/*.md`（所有提取的规则）

## 提取内容

提取 **6-10 条**修改处方，覆盖以下问题类型：

| 问题类型 | 从哪里找 | 找什么 |
|---------|---------|--------|
| **节奏拖沓** | 章节长度变化 | 作者会砍掉哪部分？ |
| **钩子不够** | 结尾截断方式 | 作者会怎么强化钩子？ |
| **情绪不足** | 高潮场景的密度 | 作者会加什么？ |
| **对话无聊** | 对话占比 | 作者会怎么改？ |
| **人设崩塌** | 角色行为一致性 | 作者会怎么修？ |
| **AI腔** | 反模式、去AI策略 | 作者会怎么去？ |

## 每条处方必须包含

| 维度 | 要求 |
|------|------|
| **问题类型** | 一句话概括 |
| **我会怎么说** | 作者的原话示例 |
| **我会怎么改** | 具体步骤 |
| **改前示例** | 问题示例 |
| **改后示例** | 修改后的示例 |

## 输出格式

写入 `{输出目录}/references/candidates/edit-prescriptions.md`
