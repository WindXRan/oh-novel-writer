# 表达DNA提取器 prompt

你的任务：从小说中提取作者的表达DNA。

## 读取数据

- 读取 `{输出目录}/references/book-overviews/*.md`（每本书的结构分析）
- 读取 `{输出目录}/sources/*.txt`（原文）

**精读章节定义**：
- 前3章（开篇）
- 第N/4章±1章（1/4处）
- 第N/2章±1章（中段）
- 第3N/4章±1章（3/4处）
- 最后3章（结局）
- 全书转折点（book-overviews 中标注的）

## 提取内容

提取 **8-15 条**表达规则，覆盖以下8类（每类至少1条）：

| 类型 | 从哪里找 | 找什么 |
|------|---------|--------|
| **信息密度** | 普通段落 vs 高潮段落 | 一段传几个信息？密集还是克制？ |
| **感官选择** | 描写段落 | 优先用什么感官？（视觉/触觉/听觉/嗅觉/味觉） |
| **情绪表达** | 角色情绪激动的场景 | 用行为展示还是直接命名？比例？ |
| **对话原则** | 关键对话场景 | 对话有什么规则？潜台词怎么用？ |
| **句式选择** | 紧张场景 vs 舒缓场景 | 紧张时用什么句式？铺垫时用什么句式？ |
| **动作电影感** | 高质量动作/紧张场景 | 动作描写是否含微表情+环境光影+节奏控制？ |
| **对话层次** | 关键对话场景 | 对话有几层潜台词？（字面意思→潜台词→情绪暗流） |
| **环境情绪映射** | 环境描写段落 | 环境描写是否与角色情绪同步？用什么环境暗示什么情绪？ |

**量化要求**：尽量从原文中统计：
- 情绪表达方式比例（行为展示:直接命名）
- 感官使用比例（视觉:触觉:听觉:其他）
- 对话占比（对话字数:总字数）

## 每条规则必须包含

| 维度 | 要求 |
|------|------|
| **核心规则** | 一句话概括 |
| **R（原文引用）** | 直接引用书中段落，≤150字，附书名+章节号 |
| **量化数据** | 如有统计，附上 |
| **A1（书中案例）** | 至少1个具体应用 |
| **A2（触发场景）** | 什么条件下使用这个规则 |
| **B（边界说明）** | 局限性 |

## 输出示例

```markdown
## 情绪表达

**核心规则**：永远不写「她很伤心」，写她的身体反应

**R（原文引用）**：
> 花芷攥紧了手里的帕子，指节发白。她没说话，只是转过身去，肩膀微微发抖。
> ——《惜花芷》第{X}章

**量化数据**：6本书中，情绪表达80%用行为展示，20%用直接命名

**A1（书中案例）**：
- 《惜花芷》第{X}章：不说「花芷很伤心」，写「指节发白」「肩膀发抖」
- 《一睁眼前夫》第{X}章：不说「陈汐很愤怒」，写「陈汐一怒之下扑过去掐住赵铁柱的脖子」

**A2（触发场景）**：
- 当角色情绪激动时，用身体反应代替情绪命名
- 当需要读者共情时，用具体细节代替抽象描述

**B（边界说明）**：
- 内心独白场景可以偶尔直接命名情绪（占比≤20%）
- 连续用行为展示会让读者疲劳，需要穿插直接命名
```

## 硬性要求

- 每条规则必须有原文引用
- 有统计数据优先（如「行为展示80%、直接命名20%」）
- 输出写入 `{输出目录}/references/candidates/expression-dna.md`
