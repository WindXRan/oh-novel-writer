# 反模式提取器 prompt

你的任务：从小说中提取作者的反模式（绝对不会做的事）。

## 读取数据

- 读取 `{输出目录}/references/book-overviews/*.md`（每本书的结构分析）
- 读取 `{输出目录}/sources/*.txt`（原文，全文扫描）

## 提取内容

提取 **6-12 条**反模式，覆盖以下6类（每类至少1条）：

| 类型 | 从哪里找 | 找什么 |
|------|---------|--------|
| **词汇反模式** | 全文扫描 | 作者从不用的词/短语（用其他词替代） |
| **句式反模式** | 全文扫描 | 作者从不用的句式（用其他句式替代） |
| **结构反模式** | 章节结构 | 作者从不用的结构（用其他结构替代） |
| **情节反模式** | 情节设计 | 作者从不用的情节套路（用其他套路替代） |
| **角色反模式** | 角色设计 | 作者从不用的角色设计（用其他设计替代） |
| **对话反模式** | 对话场景 | 作者从不用的对话写法（用其他写法替代） |

**关键区分**：反模式 ≠ 作者写得不好的东西。反模式 = 作者**刻意避免**的东西（从原文中能推断出「作者知道这样写不好，所以换了一种方式」）。

## 每条反模式必须包含

| 维度 | 要求 |
|------|------|
| **反模式描述** | 「绝对不用/不做：{X}」 |
| **R（原文引用）** | 作者用什么替代了？附原文，≤150字，附书名+章节号 |
| **原因** | 为什么不用？（从书中推断） |

## 输出示例

```markdown
## 词汇反模式

- **绝对不用**：「不禁」「忍不住」
  - R：作者用具体动作替代 → 「花芷攥紧了手里的帕子」而不是「花芷不禁攥紧了帕子」
  - ——《惜花芷》第{X}章
  - 原因：「不禁」是AI高频词，读者一眼看出套路感

- **绝对不用**：「嘴角上扬」「嘴角微扬」
  - R：作者用其他方式表达笑意 → 「抱夏抿着嘴笑」
  - ——《惜花芷》第{X}章
  - 原因：太套路，网文读者已经免疫

## 句式反模式

- **绝对不做**：连续3句用相同句式
  - R：作者用长短句交替 → 短句「花芷一怒之下」→ 长句「她做好了家族一朝反目戳她刀子的心理建设」
  - ——《惜花芷》第1章
  - 原因：相同句式会让节奏变单调

## 结构反模式

- **绝对不做**：在高潮前插入回忆
  - R：作者在高潮前保持节奏紧凑，回忆放在舒缓段
  - ——《惜花芷》第{X}章
  - 原因：回忆会打断情绪，读者会跳过
```

## 硬性要求

- 反模式必须有「作者用什么替代」的原文引用
- 不能把通用写作规则当反模式（如「不要写错别字」「不要用错标点」）
- 反模式必须是从原文中**推断出来的**，不是从写作教材里搬来的
- 输出写入 `{输出目录}/references/candidates/anti-patterns.md`
