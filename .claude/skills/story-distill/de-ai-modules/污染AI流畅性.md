# 污染AI流畅性

## 作用
AI写作追求完美流畅，人类写作有「卡顿感」。故意加入一些不完美的表达。

## 执行规则
- 在关键段落加入「卡顿」：短句断开、重复、省略
- 不是让文字变差，是让它像人写的

## 方式
| 类型 | 示例 |
|------|------|
| 短句断开 | 「她害怕。很害怕。」 |
| 省略主语 | 「站起来，走过去，推开门。」（省略「她」） |
| 半句话 | 「她想说什么——没说。」 |
| 插入打断 | 「她正要出门——不对，她不能出门。」 |

## 示例
- 原文：「沈玉容深吸一口气，稳定了一下情绪，然后推开门走了出去。」
- 处理：「沈玉容深吸一口气。稳了稳。推门出去。」

## 例外
- 每章最多3-5处
- 不能影响阅读理解
- 对话中不加（角色不会这样说话）
