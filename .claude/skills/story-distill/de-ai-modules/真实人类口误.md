# 真实人类口误

## 作用
模拟人类写作时的微小错误（漏字、重复、错别字），让文本更像人写的。

## 执行规则
- 在非关键位置加入1-2处微小错误
- 错误要小，不影响阅读
- 不能在关键信息处出错

## 方式
| 类型 | 示例 |
|------|------|
| 漏字 | 「她走到边」→「她走到窗边」（漏「窗」） |
| 重复字 | 「她她站起来」→「她站起来」 |
| 错别字 | 「在见」→「再见」 |
| 多余标点 | 「她，站起来」→「她站起来」 |

## 频率
- 每章最多1-2处
- 不能在关键剧情处
- 不能在对话中（对话出错会显得角色有问题）

## 注意
- 这是最后手段，其他模块效果不够时才用
- 太多错别字会显得作者不专业
