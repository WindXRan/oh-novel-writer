# 删连接词

## 作用
去掉AI常用的逻辑连接词，让句子之间靠语义衔接而非连接词。

## 触发条件
- 文中「然而」「但是」「因此」「所以」「不过」「而且」「同时」出现频率 > 每段1次
- 或单字连接词「但」「又」「却」「就」「也」「还」「而」总频率 ≥ 1次/百字

## 执行规则
- 删除可删的连接词（不影响理解的）
- 保留必要的转折（否则读者看不懂逻辑）
- 删除后如果句子不通，改用短句断开

## 替换方式
| 原词 | 处理 |
|------|------|
| 然而 | 删掉，用句号断开 |
| 但是 | 前面加句号，删「但是」 |
| 因此/所以 | 删掉，直接说结果 |
| 不过 | 删掉，用句号断开 |
| 而且 | 删掉，用逗号连接或断开 |
| 同时 | 删掉，直接并列 |
| 但 | 删掉或改句号断开（高频词，优先降频） |
| 又 | 删掉或用「再」替代 |
| 却 | 删掉，用句号断开 |
| 就 | 删掉或改短句 |
| 也 | 删掉或合并句子 |
| 还 | 删掉或用「再」替代 |
| 而 | 删掉，用逗号连接 |

## 示例
- 原文：「她很害怕，然而她没有退缩。」
- 处理：「她很害怕。她没有退缩。」

## 例外
- 对话中的连接词保留（角色说话自然会用）
- 否则读者看不懂逻辑关系的保留
