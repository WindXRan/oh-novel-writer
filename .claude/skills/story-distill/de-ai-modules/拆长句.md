# 拆长句

## 作用
把AI写的长句拆成短句，打破AI的「一句话说清楚」习惯。

## 触发条件
- 句子字数 > 30字

## 执行规则
- 扫描所有 > 30字的句子
- 在语义断点处拆成2-3个短句
- 优先用句号断开，其次用逗号

## 拆分方式
| 原句结构 | 拆法 |
|---------|------|
| A，B，C | A。B。C。 |
| 因为A，所以B | A。B。 |
| 虽然A，但是B | A。但B。 |
| A，然后B | A。B。 |
| A（定语很长）+B | 拆定语成独立句 |

## 示例
- 原文：「沈玉容看着铜镜里的自己，少女面容尚带稚气，眼神却已不同，镜中人目光沉静，没有十五岁该有的天真。」（42字）
- 处理：「沈玉容看着铜镜里的自己。少女面容尚带稚气，眼神却已不同。镜中人目光沉静，没有十五岁该有的天真。」

## 例外
- 对话中的长句保留（角色说话自然会说长句）
- 诗词/引用保留原样
