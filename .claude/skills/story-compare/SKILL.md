# story-compare：仿写对比文件生成

快速生成 `{书名}/对比/{区间}_对比.md`，可直接复制粘贴去问 AI。

## 用法

```
/story-compare {书名}
```

默认对比全书所有已写章节，每章和源文对应章节做对比。

也可指定区间：`/story-compare {书名} 1 10`

## 输出

生成 `{书名}/对比/对比_{start}-{end}.md`。

## 对比格式

对每章，从以下维度对比源文 vs 新书：

1. **开篇方式**（怎么开场/钩子设计）
2. **冲突类型**（利益冲突的本质）
3. **情绪弧线**（情绪流动）
4. **动作链**（推进方式）
5. **爽点位置**（爽感来源）
6. **章末钩子**（钩子类型）

### 对比原则

- 只对比，不评判
- 记录差异点即可
- 如果某一维度新书和源文完全照搬，标注 **⚠️ 疑似照搬**

## 注意事项

1. 源文在 `novel-download-authors/{作者名}/{书名}/` 缓冲区
2. 新书在 `{书名}/正文/`
3. 只对比有正文的章节
4. 如果源文缓存没有对应章节，跳过硬
