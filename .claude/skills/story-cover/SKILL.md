# story-cover · 小说封面生成

## 功能

根据书名、作者名、题材风格，自动生成专业级网文封面。

## 使用方式

```
/story-cover
```

用户提供：
- 书名
- 作者名（可选）
- 题材风格（可选，会自动分析）

## 流程

1. 分析书名，判断题材风格（古言/现言/玄幻/都市等）
2. 生成封面 prompt（含标题文字、署名、风格描述）
3. 调用 GPT-Image-2 生成封面
4. 输出 PNG 文件

## 输出

封面文件保存到项目根目录：`{书名}_封面.png`
