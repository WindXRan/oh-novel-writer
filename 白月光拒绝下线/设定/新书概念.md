# 新书概念：《白月光拒绝下线》

## 书名候选
1. **《白月光拒绝下线》** ← 确定使用
2. 《假失忆后霸总他慌了》
3. 《穿成前女友后我只想活命》

## 类型定位
- **题材**：现代言情/甜宠/穿书
- **风格**：轻喜剧日常 + 追妻火葬场
- **篇幅规划**：150-200章（约30-40万字）
- **受众**：番茄/七猫甜宠榜女性读者

## 核心卖点（保留的成功因子）
1. **身份伪装**（女主假装失忆隐瞒穿书身份）
2. **反差男主**（原黑暗霸主人设逐渐崩成忠犬）
3. **同居甜宠**（强制监护名义下的同居日常）
4. **女主觉醒成长**（从被动求生存到主动改写命运）
5. **旁观者视角世界观**（穿进自己审核过的短剧世界，知道剧情走向）

## 人物设定
**女主·许清欢（原名林晓）**
- 现实：28岁，短剧运营编辑，审过上千部狗血短剧
- 剧中：24岁，傅行之白月光前女友，原剧情中第3集车祸下线
- 性格：理性务实、吐槽役内心戏丰富、擅长随机应变

**男主·傅行之**
- 设定：30岁，傅氏集团总裁，原短剧中"白月光死后黑化"的黑暗霸总
- 后期揭示：实际上是原短剧作者穿越进了自己笔下角色，在女主穿越之前就已经到来
- 性格：原人设阴郁偏执 → 因为白月光没死，人设逐渐崩成傲娇忠犬

## 故事弧线（三幕结构）

**第一幕：我穿我自己（1-40章）**
- 女主穿成白月光，发现节点是车祸前夕
- 假装失忆逃过剧情杀，被男主以"监护权"名义强制同居
- 暗处使用"剧情引擎系统"监控偏离度，试图把主线推回正轨

**第二幕：剧情它崩了（41-120章）**
- 同居日常：女主拼命撇清关系 vs 男主越来越黏人
- 原女主本该在第40集出场，但因剧情偏离迟迟无法登场
- 女主发现男主行为与原人设严重不符，开始怀疑
- 两人在推拉中感情升温，女主从"想回家"变成"舍不得走"

**第三幕：两个穿书者的坦白（121-180章）**
- 第120章：男主发现女主失忆是假的，冲突爆发
- 两人互相试探后陆续坦白穿书真相
- 男主真实身份揭晓——他是原小说作者，因后悔写死白月光而穿书
- 双穿联手改写结局，打破"剧情引擎"桎梏
- HE：两人决定留在书中世界共创HE

## 关键转折点
| 章节 | 事件 |
|------|------|
| 第5章 | 女主车祸未死，获得"剧情引擎系统"提示存活时间 |
| 第38章 | 男主发现女主"失忆"痕迹有疑点 |
| 第75章 | 原女主迟迟无法登场，剧情偏离度过50% |
| 第100章 | 男主在白月光祭日表现出与剧本不符的克制 |
| 第120章 | 女主坦白失忆是假的，假"身份伪装"的第一次撕破 |
| 第145章 | 男主揭示自己也是穿越者（作者穿笔下的角色） |
| 第170章 | 两人合作对抗剧情引擎，改写原定结局 |
| 第180章 | 双穿HE，共同留在书中世界 |

## 与源文《女配一睁眼，失忆男主冷脸洗床单》的差异化

| 维度 | 源文 | 本作 |
|------|------|------|
| 女主身份 | 穿成恶毒女配 | 穿成炮灰白月光 |
| 穿越节点 | 剧情开头（初始设定） | 剧情中途（第3集前） |
| 男主设定 | 失忆时送外卖打工人→霸总 | 全程霸总，但人设逐渐ooc |
| 失忆方 | 男主真失忆 | 女主假失忆 |
| 冲突驱动 | 身份暴露+生存压力 | 剧情修正+死亡倒计时 |
| 世界观 | 穿书小说世界 | 穿短剧世界（带"剧情引擎"系统） |
| 同居动机 | 穷但温馨的被迫同居 | 霸总强制监护的同居 |
| 感情线节奏 | 细水长流日常为主 | 追妻火葬场+推拉为主 |
| 核心反转 | 男主恢复记忆 | 男主也是穿越者（作者本人） |
| 结局处理 | 单穿HE | 双穿HE（两人一起留在书中） |
