# 情绪弧线

## 全书情绪弧线

```
开篇（低开）→ 渐暖（上升）→ 甜蜜（高峰）→ 危机（下降）→ 高潮（谷底）→ 和解（上升）→ HE（高峰）
```

## 分阶段情绪弧线

### 第一阶段：生存（1-30章）
```
恐惧 → 迷茫 → 求生欲 → 试探 → 尴尬 → 微妙 → 好奇 → 依赖
```
- 第1章：穿越觉醒，恐惧
- 第2章：与顾衍舟对手戏，尴尬
- 第3章：发现原主痕迹，警觉
- 第4章：展现不同性格，好奇
- 第5章：肢体接触，暧昧
- 第6章：找工作，求生欲
- 第7章：职场新人，适应
- 第8章：经济困境，窘迫
- 第9章：生日/醉酒，温暖
- 第10章：误会/争吵，和好

### 第二阶段：动心（31-80章）
```
暧昧 → 心动 → 挣扎 → 甜蜜 → 危机 → 恐惧 → 甜蜜 → 循环
```
- 第31-40章：暧昧升温
- 第41-50章：关系确认
- 第51-60章：外部威胁
- 第61-70章：甜蜜日常
- 第71-80章：危机初现

### 第三阶段：危机（81-130章）
```
恐惧 → 挣扎 → 妥协 → 希望 → 绝望 → 反转 → 高潮
```
- 第81-90章：记忆碎片
- 第91-100章：身份暴露
- 第101-110章：报复开始
- 第111-120章：真相大白
- 第121-130章：情感高潮

### 第四阶段：收尾（131-188章）
```
冲突 → 僵持 → 妥协 → 和解 → 甜蜜 → HE
```
- 第131-150章：冲突升级
- 第151-170章：和解过程
- 第171-188章：甜蜜结局
