# 原书爽点公式提取

## 抽象公式（可借鉴）
1. **等待焦虑**：主角等待某件事很久，终于实现
2. **系统激活**：金手指系统触发，带来希望
3. **捡到宝**：获得徒弟/伙伴，开启新篇章
4. **扮猪吃虎**：主角看着废，实际深不可测
5. **师徒情深**：师徒之间的温情互动
6. **幽默吐槽**：轻松搞笑的日常
7. **反差萌**：看似不靠谱的师父，其实很靠谱

## 禁用词库（必须全部替换）

### 地名
- 桃花山 → 青云山/忘忧山/落霞山
- 桃花观 → 破庙/道观/草庐
- 桃花溪 → 清溪/小河/山涧

### 人名
- 陶眠 → 林九/张三/李四
- 一狗/顾园 → 小石头/阿旺/铁蛋
- 芦贵妃 → 大黄/黑子/花花

### 系统
- 留一手 → 逆天改命/否极泰来/绝处逢生
- 共享修炼 → 逆转修炼/反向增幅/以弱制强
- 云共享 → 同步削弱/逆向反馈

### 功法
- 劈柴剑法 → 挑水功/扫地掌/砍柴刀
- 切菜刀法 → 做饭功/洗衣掌/种地术
- 穿云剑法 → 落叶剑法/流水剑法
- 冥川刀法 → 清风刀法/明月刀法

### 动物
- 鸡 → 鸭/鹅/狗/猫
- 蟑螂 → 老鼠/蚂蚁/蚊子
- 芦贵妃（公芦花鸡） → 大黄（黄狗）/黑子（黑猫）

### 金句（必须原创）
- ❌ "鹅之大，一锅炖不下"
- ❌ "师徒一生一起走，师父有两口汤喝，肯定有你两个锅洗"
- ❌ "徒弟到底去哪里领？！抽奖送吗？！垃圾桶里捡吗？！充话费吗？！"

### 场景
- ❌ 溪边捡婴儿/澡盆飘来
- ❌ 做蛋糕插蜡烛许愿
- ❌ 师父懒洋洋躺着，徒弟干活

### 结构
- ❌ 穿越者+长生系统+等徒弟
- 必须「反向设计」：被封印魔头+逆徒系统+躲徒弟
