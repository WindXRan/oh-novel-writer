# 角色塑造清单

## 主角设计

### 核心性格（2-3个关键词）

**原则**：主角必须有2-3个鲜明的性格特点，通过场景/对话/行为展现

**性格类型库**：

| 类型 | 特点 | 展现方式 | 示例 |
|------|------|----------|------|
| 倔强型 | 不服输、有骨气、越挫越勇 | 被拒绝后更努力，被打击后更强硬 | 陈安（仿写版） |
| 腹黑型 | 表面老实、内心有算计、善于伪装 | 表面答应、背后反击，装傻充愣 | 陆明（原文） |
| 热血型 | 热情、冲动、重情义 | 冲动行事、为朋友两肋插刀 | 常见主角 |
| 冷静型 | 理性、克制、善于分析 | 遇事不慌、冷静分析、精准反击 | 常见主角 |
| 闷骚型 | 外表冷淡、内心狂热、不善表达 | 做出惊人举动、说出惊人台词 | 常见主角 |

### 性格展现方式

**原则**：性格通过场景/对话/行为展现，而非直接描述

**好的展现方式**：
- 通过场景展现：被裁员后不抱怨，直接回家
- 通过对话展现："别人越说他不行，他越要证明自己行"
- 通过行为展现：说走就走，绝不拖泥带水

**坏的展现方式**：
- 直接描述：他是一个倔强的人
- 旁白说明：他从小就倔
- 他人评价：他这个人就是倔

### 性格一致性

**原则**：主角性格必须前后一致，不能前后矛盾

**检查表**：
- [ ] 第1章的性格与第10章一致
- [ ] 面对不同情况时，性格反应一致
- [ ] 性格成长有铺垫，不是突然改变

## 配角设计

### 配角功能定位

| 功能 | 作用 | 示例 |
|------|------|------|
| 帮手 | 帮助主角解决问题 | 堂妹（财务）、同学（人事） |
| 对手 | 制造冲突和障碍 | 地头蛇、竞争对手 |
| 导师 | 指导主角成长 | 律师、行业前辈 |
| 情感线 | 增加情感戏份 | 女主角、暧昧对象 |
| 信息源 | 提供关键信息 | 线人、情报贩子 |

### 配角性格设计

**原则**：配角性格必须与主角形成对比或互补

**对比型配角**：
- 主角倔强，配角圆滑
- 主角冲动，配角冷静
- 主角外向，配角内向

**互补型配角**：
- 主角擅长决策，配角擅长执行
- 主角擅长谈判，配角擅长技术
- 主角擅长冒险，配角擅长风控

## 角色关系图谱

### 关系类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 盟友 | 帮助主角，共同目标 | 团队成员、合作伙伴 |
| 对手 | 阻碍主角，利益冲突 | 竞争对手、地头蛇 |
| 中立 | 不帮不阻，可争取 | 政府官员、普通群众 |
| 情感 | 增加情感戏份 | 女主角、家人 |

### 关系动态

**原则**：角色关系必须有变化，不能一成不变

**变化类型**：
- 陌生人 → 盟友（通过合作建立信任）
- 盟友 → 对手（因利益冲突反目）
- 对手 → 盟友（因共同敌人联手）
- 中立 → 盟友（被主角说服/打动）

## 角色塑造示例

### 示例1：陈安（仿写版）

**核心性格**：倔强、有野心、善于谈判

**展现方式**：
- 通过场景展现：被裁员后不抱怨，直接回家
- 通过对话展现："别人越说他不行，他越要证明自己行"
- 通过行为展现：说走就走，绝不拖泥带水

**配角关系**：
- 堂妹（帮手）：财务总监，负责管钱
- 同学（帮手）：人事经理，负责招人
- 律师（导师）：法律顾问，负责法务
- 王德发（对手→盟友）：原工厂老板，从对手变成盟友

### 示例2：陆明（原文）

**核心性格**：腹黑、有城府、善于伪装

**展现方式**：
- 通过场景展现：表面答应政府要求，背后硬刚
- 通过对话展现："英雄不问出身"
- 通过行为展现：砍价时说走就走，逼对方让步

**配角关系**：
- 陆鸢（帮手）：堂妹，财务总监
- 沈璃（帮手+情感线）：同学，人事经理
- 方瑜（导师）：律师，法律顾问
- 胡奎（对手）：地头蛇，主要对手
