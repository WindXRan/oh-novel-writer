# 情绪拉扯技法库

## 压抑技法

### 1. 身份贬低
**手法**：通过他人之口或行为贬低主角身份
**效果**：激发读者代入感，期待打脸
**示例结构**：
- 旁人嘲讽主角身份/能力/外表
- 主角选择暂时隐忍
- 读者积累不满情绪

### 2. 信息压制
**手法**：读者知道主角有底牌，但书中角色不知道
**效果**：制造期待感，等待底牌亮出
**示例结构**：
- 主角获得隐藏能力/身份
- 反派不知情，继续嚣张
- 读者等待主角亮牌

### 3. 情感勒索
**手法**：利用情感关系制造两难
**效果**：增加故事深度，引发共情
**示例结构**：
- 亲人/爱人被威胁
- 主角被迫妥协
- 读者心疼主角

### 4. 规则压制
**手法**：用规则/制度/等级压制主角
**效果**：为后续打破规则做铺垫
**示例结构**：
- 主角因等级/身份被限制
- 主角暂时遵守规则
- 后续打破规则时更爽

### 5. 谈判压抑
**手法**：谈判中不释放爽点，只做铺垫
**效果**：积累期待，为后续签约付款做铺垫
**示例结构**：
- 主角与对方谈判
- 对方开价过高，主角不急
- 主角给出两个选择，让对方思考
- 读者等待签约付款的爽点

### 6. 花不完的焦虑
**手法**：主角有钱但花不出去
**效果**：制造反差感，引发读者好奇
**示例结构**：
- 主角获得大额资金
- 测试发现限制条件
- 钱花不出去的焦虑
- 读者期待主角如何破局

## 爽点技法

### 1. 身份揭示
**手法**：亮出主角隐藏的真实身份
**时机**：压抑到极致时释放
**效果**：直接打脸，情绪释放

### 2. 实力碾压
**手法**：主角展示压倒性实力
**时机**：反派最嚣张时
**效果**：以力破巧，干脆利落

### 3. 智商碾压
**手法**：主角用计谋反杀
**时机**：反派以为胜券在握时
**效果**：智力优越感，反转快感

### 4. 意外援军
**手法**：强大盟友及时出现
**时机**：主角陷入绝境时
**效果**：柳暗花明，绝处逢生

### 5. 以彼之道还施彼身
**手法**：用反派的招数反击反派
**时机**：反派得意忘形时
**效果**：因果报应，大快人心

## 悬念技法

### 1. 信息断层
**手法**：关键信息只揭露一半
**效果**：读者主动猜测，继续阅读

### 2. 倒计时
**手法**：设定时间限制
**效果**：制造紧迫感

### 3. 身份谜团
**手法**：真实身份被隐藏
**效果**：读者想揭开谜底

### 4. 两难选择
**手法**：主角面临两难抉择
**效果**：读者想知道选择结果
