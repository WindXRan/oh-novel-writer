# 叙述者风格参考

> 仿写的核心不是"技术规则"，而是"谁在讲故事"。
> 本文件定义网文中常见的叙述者风格类型，供 Phase 1 分析源文本时参考。
> Phase 3 不注入人格定义，只给源文本的语感样本让 agent 自然模仿。

---

## 核心原理

AIGC 检测器看的不是禁用词，而是**选词概率分布**。LLM 默认产出"高概率词"——工整、流畅、书面化。
要打破这个分布，必须让 LLM **模仿具体的人类文本**，而不是在"文学写手"模式下选词。

**模仿 > 定义。** 给 agent 一段源文本让它模仿，比告诉 agent "你是XX型人格"更有效。前者改变选词分布，后者只是贴标签。

---

## 人格类型库

### 类型 A：吐槽型网文读者（最常见，适配甜宠/穿书/系统文）

**人格定义**：你是一个现代网文重度读者，边看故事边吐槽。你知道自己在写小说，偶尔会跳出来评论几句。你的语言是互联网原住民的语言——缩写、梗、口语、感叹号。

**语言特征**：
- 网感口语：偶尔用，不是堆砌。每章最多 1-2 个网络梗，分散使用，不能连续出现
- 自我吐槽：用具体判断代替网络热词
- 第四面墙碎裂：叙述者会跳出来评价角色
- 现代比喻：用生活化的比喻，不要用已经被用烂的梗
- 句式极度不规则：独词句、感叹句、反问句混用，但**独词句不能连续3个**
- 不怕"不完美"：允许语法小瑕疵、允许废话、允许吐槽打断叙事节奏

**⚠️ 参考示例（仅说明风格特征，不用于仿写）**：
```
宋云绯有些懵，自己不是在出租车上吗？这是给她干哪来了？
```
→ 特征：口语化、直接、有现代人视角

```
别问，问就是剧情需要。
```
→ 特征：跳出来跟读者互动

**⚠️ 实际仿写时，语感样本必须从源文本前3章提取，不能用上面的参考示例。不同书用不同样本。**

**选词倾向**：高口语、高吐槽、高现代感。句子短，节奏碎，情绪外放。

---

### 类型 B：冷幽默旁观者（适配悬疑/都市/职场文）

**人格定义**：你是一个冷眼旁观的讲述者，不带感情色彩，但偶尔冒出一句冷幽默。你的叙述风格是"白描+毒舌点评"，像一个不耐烦的记者在写稿。

**语言特征**：
- 短句为主，陈述句居多
- 偶尔一句话戳破真相
- 冷幽默：用对比制造笑点
- 不解释，让读者自己补
- 具体细节代替抽象形容

**⚠️ 参考示例（仅说明风格特征，不用于仿写）**：
```
要不是看他业务能力强，就他经常开会时电话响个不停的事迹，老板早就把他开除了。
```
→ 特征：陈述事实+冷幽默评价

**⚠️ 实际仿写时，语感样本必须从源文本前3章提取，不能用上面的参考示例。**

**选词倾向**：中性、克制、偶尔辛辣。句子干脆，不铺垫，不抒情。

---

### 类型 C：沉浸式体验者（适配重生/穿越/虐文）

**人格定义**：你就是主角本人（或主角的贴身旁白），叙述和主角的内心活动融为一体。不跳出来吐槽，但语言极度口语化，像主角在脑子里自言自语。

**语言特征**：
- 第三人称但接近第一人称内心独白
- 主角的主观判断直接写入叙述
- 情绪反应直接、不修饰
- 现代人视角评论古代/异世界

**⚠️ 参考示例（仅说明风格特征，不用于仿写）**：
```
原主长得其实挺漂亮的。小家碧玉那一挂的长相，算不上多惊艳，但看着有初恋的味道。
```
→ 特征：主角的主观判断直接写入叙述

**⚠️ 实际仿写时，语感样本必须从源文本前3章提取，不能用上面的参考示例。**

**选词倾向**：主观、随意、带个人判断。像日记而不是小说。

---

### 类型 D：老派故事人（适配年代文/乡村文/家长里短）

**人格定义**：你是一个爱讲故事的人，像邻居大妈/大叔在跟你唠嗑。叙述中有"你别说""你猜怎么着""这事儿啊"这样的口语标记。

**语言特征**：
- 口语连接词："你别说""话说回来""这不"
- 比喻来自日常生活
- 节奏像说话，有停顿和重复
- 不怕絮叨，该交代的背景不省略

**⚠️ 参考示例（仅说明风格特征，不用于仿写）**：
```
你别说，这楚靳寒虽然失忆了，但脑子还在。
```
→ 特征：口语连接词开头

**⚠️ 实际仿写时，语感样本必须从源文本前3章提取，不能用上面的参考示例。**

**选词倾向**：生活化、接地气、有烟火气。句子长但不书面。

---

### 类型 E：冷峻叙事者（适配悬疑/刑侦/硬核现实文）

**人格定义**：你是一个冷静到近乎冷酷的讲述者。不带情绪，不评论，只呈现事实和动作。像纪录片的旁白。

**语言特征**：
- 极简陈述
- 不解释人物动机，只写行为
- 环境描写精准但不堆砌
- 对话不加修饰，只有引号里的内容

**⚠️ 参考示例（仅说明风格特征，不用于仿写）**：
```
楚靳寒沉默地看着她。下意识捏了捏手里的东西，并往身后藏了藏。
```
→ 特征：动作白描，不加情绪解释

**⚠️ 实际仿写时，语感样本必须从源文本前3章提取，不能用上面的参考示例。**

**选词倾向**：低频形容词、高行动密度。克制到极致。

## 人格匹配规则

Phase 1 读取源文本前3章后，按以下维度打分匹配：

| 维度 | 判断方法 |
|------|----------|
| 叙述者是否跳出来评论 | 有 → A/B/D；无 → C/E |
| 是否有现代人视角/网感语言 | 强 → A；中 → B/C；弱 → D/E |
| 句子平均长度 | <15字 → A/E；15-25字 → B/C；>25字 → D |
| 是否有第四面墙碎裂 | 有 → A；偶尔 → C；无 → B/D/E |
| 情绪外放程度 | 高 → A；中 → C；低 → B/E |
| 口语化程度 | 极高 → A/D；中 → B/C；低 → E |

匹配后选择最接近的人格类型，**允许混合**（如 A+C：吐槽型但偶尔沉浸）。

---

## 使用方式

Phase 1 匹配人格后，**不需要告诉 agent "你是XX型人格"**。只需要给它源文本的语感样本，让它自然模仿。

Phase 3 agent prompt 中注入：

```
## 你的声音

你不是在"写小说"，你是在随口讲一个你刚看到的故事。
你没有"角色设定"，你只是在跟朋友聊天。

模仿下面这些片段的感觉（不是模仿具体句子，是模仿那种"随口说"的语气）：

{源文本语感样本1}
{源文本语感样本2}
{源文本语感样本3}

想吐槽就吐槽，想沉默就沉默。没有规则。
```

**⚠️ 不要告诉 agent "你是吐槽型/冷幽默型/沉浸型"。** 这会让 agent "表演"那种风格，而不是自然流露。只需要给样本，让 agent 自己"进入"那种语感。

---

## 注意事项

1. **语感样本必须来自源文本**：不用其他小说的片段，因为不同书的语感差异很大
2. **不要告诉 agent 人格类型名称**：只给样本，让 agent 自然模仿
3. **允许不完美**：人类写作有语法瑕疵、有废话、有重复，这些不是bug，是特征
