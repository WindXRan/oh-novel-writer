# 仿写成功模式

通过实际使用 story-rewrite 技能迭代，发现以下成功模式：

## 1. 开头从系统出场开始（置信度 95%）
- **问题**：从日常对话开始，缺乏冲击力
- **解决**：直接从系统出场开始，更有悬念
- **示例**："手机屏幕黑了。不是没电，是突然黑了。"

## 2. 系统出场仪式感（置信度 90%）
- **问题**：系统出场平淡，缺乏视觉冲击
- **解决**：增加金色字体、屏幕黑屏、情绪反应
- **示例**："屏幕上浮出一行字。金色的字。"

## 3. 章尾钩子明确（置信度 85%）
- **问题**：章尾缺乏悬念，读者没有继续阅读的动力
- **解决**：以具体计划或冲突点结尾
- **示例**："第一步：买下整条商业街。"

## 4. 主角性格通过行为展现（置信度 80%）
- **问题**：直接描述性格，缺乏说服力
- **解决**：通过具体行为、对话、决策展现性格
- **示例**："他掐了自己一把，疼。不是做梦。"
