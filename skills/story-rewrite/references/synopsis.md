# 简介生成指南

## 核心原则：黄金三秒吸引力

简介的目标是在3秒内抓住读者，制造"不点开会后悔"的冲动。

**⚠️ 禁止模仿源文本简介结构**：必须根据自己的故事特点重新设计，不能套用源文本的句式和节奏。

---

## 生成流程

### Step 1：要素提取（从框架中提取）

- 【核心金手指】：重生记忆/系统/特殊能力
- 【最大反差】：男女主的误解/偏见/身份差异
- 【装逼打脸瞬间】：男主/女主的高光时刻
- 【扎心情绪点】：前世的遗憾/牺牲/错过

### Step 2：选择简介风格

根据题材选择最适合的风格：

| 题材 | 推荐风格 | 特点 |
|------|----------|------|
| 甜宠/校园 | 反差流 | 认知偏差→真相揭示 |
| 虐文/救赎 | 情绪流 | 前世遗憾→今生弥补 |
| 悬疑/复仇 | 悬念流 | 身份揭秘→时间紧迫 |
| 系统/无敌 | 数据流 | 系统面板→数值展示 |
| 快穿/穿书 | 玩梗流 | 槽点吐槽→趣味互动 |

### Step 3：按风格生成

#### 风格A：反差流（适合甜宠）

```
{角色标签}

{女主对男主的误解}（2-3句具体表现）
{反转事件}（1句）
{真相揭示}（1-2句）

{重生/穿越契机}（1句）
{今生目标}（1句）
```

**示例**：
```
迷糊护士 x 冷面病人家属

林念念觉得612床的家属有病。
每天守在ICU门口，不吃不喝不睡觉。
她给他送巧克力，他看都不看。
她让他去休息，他说"不用"。
直到那天晚上，这个有病的人替她挡了一刀。

一睁眼，回到事发前两小时。
这一次，她不打算让他再死一次。
```

#### 风格B：情绪流（适合虐文/救赎）

```
{角色标签}

{前世悲剧场景}（1-2句，具体画面）
{女主的醒悟}（1句）

{重生契机}（1句）
{今生承诺}（1-2句，要有具体行动）
```

**示例**：
```
迷糊护士 x 冷面病人家属

刀刺进去的声音很闷。
林念念跪在地上，看着血从他指缝里往外涌。
他闭眼前说的最后一句话是"你叫什么来着"。

一睁眼，回到两小时前。
林念念站在走廊尽头，看着那个靠墙睡着的男人。
这次，她要记住他的名字。
```

#### 风格C：悬念流（适合悬疑/复仇）

```
{角色标签}

{表面事实}（1句）
{转折揭示}（1句）
{真相}（1-2句）

{时间限制}（1句）
{目标}（1句）
{后果}（1句）
```

**示例**：
```
迷糊护士 x 冷面病人家属

林念念以为顾言深只是个普通病人家属。
直到她看见他钱包里的照片。
那是她十年前的样子，可她完全不认识他。

距离医闹袭击还有两小时。
她必须弄清楚他到底是谁。
否则，历史会重演。
```

#### 风格D：数据流（适合系统/无敌）

```
【系统面板展示】
{能力/属性/词条}

{角色标签}

{金手指展示}（1-2句）
{反差/冲突}（1句）
```

**示例**：
```
【叮！检测到宿主重生，激活"时间回溯"能力】
【剩余回溯次数：0/1】
【警告：本次回溯为最终机会】

迷糊护士 x 冷面病人家属

林念念重生了，但没有任何金手指。
她只知道两小时后会有人持刀袭击急诊科。
以及，那个每天守在ICU门口的男人会替她死。
```

## 简介结构模板

```
【题材标签+卖点标签】（1行）
{角色标签}（1行）
*
{内容段1}（3-5行）
*
{内容段2}（2-3行）
*
TIPS:
1.{双洁/甜宠等读者关心的信息}
2.{男主隐藏属性/前世暗恋等爽点}
3.{无雌竞/女主成长等承诺}
4.{番外/隐藏剧情等彩蛋}
```

---

## ⚠️ 简介禁忌

1. **禁止模仿源文本结构**：不能套用"在XX眼里→直到→一睁眼"的模板
2. **禁止平铺直叙**：不要写"林念念是护士，顾言深是病人家属，他们相爱了"
3. **禁止剧透结局**：不要写"最后他们幸福地生活在一起"
4. **禁止空洞标签**：不要写"这是一个关于爱与救赎的故事"
5. **禁止过度中二**：飞卢风可以热血，但不要变成"龙傲天"
