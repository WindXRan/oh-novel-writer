# 结构提取方法论

## 核心原则：只仿骨相，不仿皮相

### 三层抽象模型

```
Layer 3（禁止）：具体情节
  "女主坐三趟班车到雨中小镇，敲错门撞见催债"
  → 这是皮相，不能提取

Layer 2（可选）：叙事模式
  "主角进入陌生环境，遭遇第三方冲突，意外与关键人物产生交集"
  → 这是骨肉，可以提取

Layer 1（必须）：叙事功能
  "铺垫 → 压抑 → 偶遇 → 钩子"
  → 这是骨架，必须提取
```

### 提取规则

| 层级 | 能否提取 | 示例 |
|------|---------|------|
| **叙事功能**（必须提取） | ✅ | 铺垫→压抑→转折→爆发→平缓 |
| **情绪弧线**（必须提取） | ✅ | 紧张→暧昧→危机→释放 |
| **人物关系模式**（必须提取） | ✅ | 保护/欺骗/试探/对立 |
| **信息差类型**（必须提取） | ✅ | 单向隐瞒/双向误解/读者先知 |
| **叙事技巧**（可选提取） | ⚠️ | 倒叙、多视角、内心独白交替 |
| **具体场景**（禁止提取） | ❌ | 撑伞站在巷口、在汽车站等人 |
| **具体道具**（禁止提取） | ❌ | 柿子树、母亲的信、清汤挂面 |
| **具体台词**（禁止提取） | ❌ | 任何可直接搬运的对白 |
| **具体事件**（禁止提取） | ❌ | 谎称警察解围、进错门 |
| **具体细节**（禁止提取） | ❌ | 座椅弹簧、三趟班车 |

### 抽象化测试

对每个要提取的要素，问自己：

> 「如果我把这个要素描述给一个完全没读过原文的人，他能据此写出一个完全不同但结构相似的故事吗？」

- 能 → 这是抽象结构，可以提取
- 不能 → 这是具体情节，禁止提取

## 骨架识别框架

### 1. 叙事单元划分

将源文本划分为最小叙事单元，每个单元包含：
- **触发事件**：推动情节发展的动作或信息
- **角色反应**：主角对事件的回应
- **结果/后果**：反应导致的状态变化

### 2. 功能标注

每个叙事单元标注其叙事功能：

| 功能类型 | 说明 | 示例 |
|----------|------|------|
| **铺垫** | 为后续事件埋设伏笔或建立背景 | 介绍世界观、展示角色日常 |
| **压抑** | 制造负面情绪，积累读者期待 | 主角被嘲讽、遭遇不公、陷入困境 |
| **转折** | 改变故事走向或信息状态 | 发现真相、获得能力、身份暴露 |
| **爆发** | 释放积累的情绪，满足读者期待 | 打脸反派、实力碾压、逆袭成功 |
| **平缓** | 情绪缓冲，为下一阶段做准备 | 日常互动、能力提升、关系发展 |
| **钩子** | 制造悬念，驱动读者继续阅读 | 留下疑问、暗示危机、引入新角色 |

### 3. 节奏曲线绘制

将章节序列转化为情绪曲线：

```
情绪强度
  ^
  |     *           *
  |    * *         * *
  |   *   *       *   *
  |  *     *     *     *
  | *       *   *       *
  |*         * *         *
  +-------------------------> 章节
    铺垫 压抑 爆发 平缓 压抑 爆发
```

### 4. 转折点识别

关键转折点的特征：
- 信息状态改变（不知道→知道）
- 力量对比改变（弱→强）
- 关系状态改变（敌对→同盟）
- 目标方向改变（A目标→B目标）

### 5. 金手指分析

金手指的核心要素：
- **机制**：每天X万/完成任务得奖励/投资返利等
- **限制条件**：地域限制/用途限制/时间限制等
- **升级路径**：如何让读者看到希望
- **爽点来源**：数字碾压/身份揭示/实力逆转等

**金手指设计公式**：
```
金手指 = 大额资金 + 简单限制 + 丰富场景
```

**检查表**：
- [ ] 数字足够大（每天至少百万级别）
- [ ] 限制条件一句话能说清
- [ ] 能衍生多种消费/投资场景
- [ ] 有清晰的升级路径

### 6. 角色塑造分析

角色塑造的核心要素：
- **主角性格**：2-3个关键词
- **性格展现方式**：通过什么场景/对话/行为展现
- **配角功能**：帮手/对手/导师/情感线
- **角色关系**：谁帮主角、谁阻碍主角、谁是情感线

**角色塑造检查表**：
- [ ] 主角有2-3个鲜明的性格特点
- [ ] 性格通过场景/对话/行为展现，而非直接描述
- [ ] 配角有明确的功能定位
- [ ] 角色关系图谱清晰

## 提取清单

### 节奏要素
- [ ] 每章的情绪基调（1-2词描述）
- [ ] 压抑段落的平均长度
- [ ] 爽点释放的频率（每X章一个）
- [ ] 高潮章节的位置

### 结构要素
- [ ] 场景切换的模式
- [ ] 视角切换的规则
- [ ] 时间跨度的处理
- [ ] 信息揭露的节奏

### 技法要素
- [ ] 钩子设置的手法
- [ ] 悬念维持的方法
- [ ] 反转铺垫的技巧
- [ ] 情绪拉扯的方式

### 金手指要素
- [ ] 金手指的机制
- [ ] 限制条件的设计
- [ ] 升级路径的清晰度
- [ ] 爽点来源的类型

### 角色要素
- [ ] 主角的核心性格
- [ ] 性格的展现方式
- [ ] 配角的功能定位
- [ ] 角色关系图谱
