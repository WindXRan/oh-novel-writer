# 快速仿写 Skill

## 核心理念
**借鉴「爽点公式」，创造「全新故事」**

洗稿 = 保留原书框架 + 换词 → 会被下架
仿写 = 借鉴爽点公式 + 创新故事 → 安全

---

## 什么是爽点公式？

爽点公式是**抽象的叙事技巧**，不涉及具体内容。

### 可借鉴的爽点公式
| 爽点公式 | 说明 | 示例 |
|----------|------|------|
| 低期待→高回报 | 主角看似废物，实际很强 | 废柴逆袭、扮猪吃虎 |
| 金手指 | 系统/传承/血脉带来优势 | 系统流、重生流 |
| 反差萌 | 角色表面和内在形成反差 | 懒癌师父、毒舌暖男 |
| 打脸 | 用实力证明自己 | 退婚流、装逼打脸 |
| 师徒情深 | 师徒之间的温情互动 | 收徒流、宗门流 |
| 轻松幽默 | 用幽默化解紧张 | 吐槽流、日常流 |
| 升级打怪 | 不断提升实力 | 修仙流、玄幻流 |
| 探索解谜 | 探索未知世界 | 无限流、诸天流 |

### 不可借鉴的具体内容
1. 原书的人名、地名、系统名、功法名
2. 原书的场景、情节、对话
3. 原书的句式、表达、短语
4. 原书的结构、框架、节奏

## 工作流程

### Phase 1：分析原书，提取爽点公式

```
输入：原书txt

输出：
1. 爽点公式列表（抽象层）
2. 原书风格特征（语调、节奏、人设类型）
3. 目标读者画像
```

#### 分析维度
| 维度 | 分析内容 |
|------|----------|
| 题材类型 | 玄幻/都市/仙侠/历史/科幻 |
| 爽点类型 | 扮猪吃虎/系统流/重生流/收徒流 |
| 人设类型 | 懒癌/话痨/毒舌/冷漠 |
| 节奏类型 | 快节奏/慢节奏/张弛有度 |
| 语调风格 | 轻松幽默/热血燃向/悬疑紧张 |

### Phase 2：创造全新设定

```
基于爽点公式，创造：
1. 全新的主角（名字、身份、性格、背景）
2. 全新的系统/金手指（名称、机制、触发条件）
3. 全新的世界观（地名、势力、力量体系）
4. 全新的徒弟/伙伴（名字、身世、性格）
5. 全新的核心冲突
```

#### 设定模板
```
## 核心设定表

### 基本信息
- 书名：{全新书名}
- 题材/类型：{主类型 + 副类型}
- 目标平台：{起点/番茄/晋江/其他}
- 预计字数：{X} 万字
- 目标读者：{画像}

### 一句话梗概
{主角 + 目标 + 阻碍 + 反转，一句话概括全书}

### 主角设定
- 姓名：{}
- 年龄：{}
- 核心特质：{2-3 个关键词}
- 金手指/核心能力：{}
- 弱点/缺陷：{让角色更立体的地方}
- 核心动机：{他为什么要做这件事}

### 世界观骨架
- 时代/背景：{}
- 核心设定：{区别于同类作品的独特设定}
- 力量体系：{如果有，简单概括}
- 社会结构：{影响故事的关键设定}

### 核心冲突
- 主线矛盾：{}
- 终极 Boss/终极阻碍：{}
```

### Phase 3：搭建全新大纲

```
基于全新设定，搭建：
1. 卷级大纲（全书结构）
2. 细纲（每章事件+钩子+爽点）
```

#### 卷级大纲模板
```
## 卷级大纲

### 第一卷：{卷名}（约 {X} 万字，{Y} 章）
- 功能：{铺垫/起步/第一个大爽点}
- 核心事件：{一句话}
- 起始状态 → 结束状态：{主角从 {A} 变成 {B}}

### 第二卷：{卷名}
...

### 最终卷：{卷名}
- 功能：{高潮 + 收尾}
- 核心事件：{一句话}
```

#### 细纲模板
```
## 细纲（第 N 章）

### 第 N 章：{章名}
- 核心事件：{一句话}
- 章首钩子：{具体内容}
- 爽点：{本章爽点}
- 章尾钩子：{具体内容，期待度：强/中/弱}
- 字数目标：{X} 字
```

### Phase 4：生成全新内容

```
基于全新大纲，生成：
1. 全新的正文内容
2. 所有对话必须原创
3. 所有描写必须原创
4. 所有系统提示必须原创
```

#### 写作规则
1. **所有内容必须原创**：不能借用原书任何具体内容
2. **所有角色必须原创**：不能借用原书任何角色
3. **所有情节必须原创**：不能借用原书任何情节
4. **所有对话必须原创**：不能借用原书任何对话
5. **所有句式必须原创**：不能借用原书任何句式
6. **所有表达必须原创**：不能借用原书任何表达

#### 字数与节奏参考
| 节奏 | 字数区间 | 内容密度 |
|------|----------|----------|
| 高速推进 | 2000-3000 字/章 | 每章一个明确事件 |
| 正常节奏 | 3000-4000 字/章 | 主线 + 少量副线 |
| 舒缓铺垫 | 3000-4000 字/章 | 人物互动 + 伏笔 |
| 高潮爆发 | 2000-3000 字/章 | 集中释放、不拖沓 |

### Phase 5：质量检查

```
检查内容：
1. 是否有原书标志性内容
2. 是否有原书标志性句式
3. 是否有原书标志性表达
4. 是否有原书标志性短语
5. 连续10字以上相同片段
```

#### 检查清单
| 检查项 | 标准 | 处理 |
|--------|------|------|
| 原书标志性内容 | 0处 | 命中即重写 |
| 原书标志性句式 | 0处 | 命中即重写 |
| 原书标志性表达 | 0处 | 命中即重写 |
| 原书标志性短语 | 0处 | 命中即重写 |
| 连续10字相同 | 0处 | 命中即重写 |

## 注意事项

1. **不是洗稿**：不能保留原书框架，只换词
2. **是创新**：借鉴爽点公式，创造全新故事
3. **所有内容必须原创**：不能借用原书任何具体内容
4. **所有角色必须原创**：不能借用原书任何角色
5. **所有情节必须原创**：不能借用原书任何情节
6. **所有对话必须原创**：不能借用原书任何对话
7. **通用性**：适用于任意原书，不针对特定案例
