# 金手指.md

## 基本信息
- **类型**：system（快穿系统）
- **名称**：快穿任务系统
- **系统名称**：笨笨

## 描述（约400字）

姜蜜的金手指是一个快穿任务系统，编号未知，系统名为"笨笨"。该系统允许宿主穿梭于三千世界执行任务，完成任务可获得积分奖励。姜蜜作为资深穿越者，已完成大量任务世界，积累了99万积分的巨额财富。

系统的核心功能包括：
1. **任务发布**：系统会根据当前世界的剧情线发布任务，如"欺负女主""纠缠男主""羞辱反派"等
2. **剧情查询**：宿主可查询世界剧情走向，了解原身背景和人物关系
3. **身体接管**：系统拥有最高权限，可在必要时暂时接管宿主身体，强制推进剧情
4. **积分兑换**：积分可用于兑换"时空票"，返回原世界报仇

系统的特殊之处在于：
- 穿越期间，原世界时间完全停滞
- 系统权限有限，无法查询顾川等高危存在的详细信息
- 系统性格呆萌话痨，经常被宿主忽悠
- 系统会根据剧情需要强制宿主维持人设，违反会受到电击警告

姜蜜的穿越者身份本身也是一种"金手指"——她在三千世界积累的丰富经验（特种兵、战神、风水师、驯兽师、影后等）使她拥有远超普通人的能力储备，能够在各种场景中游刃有余。

## 核心机制

### 积分系统
- **获取途径**：完成系统发布的任务
- **消耗途径**：兑换时空票返回原世界
- **惩罚机制**：任务失败扣除1万积分
- **姜蜜当前余额**：99万积分（足以支撑长期任务）

### 强制剧情推进
- 系统可在宿主违反人设时强制接管身体
- 接管期间宿主意识清醒但无法控制身体
- 接管结束后宿主会感到短暂麻痹
- 例：当姜蜜想揭穿沈铭轩时，系统强制她说"铭轩哥，你太帅了"

### 人设维持规则
- 必须维持恶毒女配人设
- 必须完成"欺负女主""纠缠男主""羞辱反派"的任务
- 违反人设会收到电击警告（不疼但有警示效果）

## 当前能力

### 已解锁能力
1. **穿越能力**：可穿梭不同世界执行任务
2. **剧情预知**：了解当前世界剧情走向
3. **身体强化**：穿越者体质，反应速度和体能优于常人
4. **技能库**：在其他世界习得的技能可保留（如空手道、骑马、驯兽等）
5. **系统商店**：用积分兑换各种道具（具体未详述）

### 能力限制
1. **人设束缚**：必须维持恶毒女配人设
2. **剧情惯性**：世界规则强制所有人喜欢女主
3. **系统权限不足**：无法查询顾川等高危存在的信息
4. **强制惩罚**：违反人设会遭受电击

### 潜在发展
1. **积分积累**：通过完成任务积累更多积分
2. **时空票兑换**：最终目标是返回原世界报仇
3. **世界规则突破**：可能找到绕过万人迷世界规则的方法
4. **反派攻略**：对顾川的"见色起意"可能发展为真正的感情线
