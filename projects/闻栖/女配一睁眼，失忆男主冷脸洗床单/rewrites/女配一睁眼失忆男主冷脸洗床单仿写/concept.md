# 新书设定

## 书名候选
1. 《当他失忆后，我成了白月光》（推荐）
2. 《失忆大佬的作精美娇娘》
3. 《穿成炮灰后，我捡了个失忆魔尊》

## 类型定位
- 古言 / 仙侠 / 穿书 / 女配觉醒 / 轻松 / 1v1 / HE
- 标签组合：穿书+女配+失忆大佬+江湖+萌宠+轻松向

## 核心卖点（保留的成功因子）
| 源文成功因子 | 新书保留方式 |
|---|---|
| 开篇即穿越，信息差带来的紧张感和喜剧感 | 保留穿书设定，女主开局就在“欺骗”高位者，时刻害怕掉马 |
| “失忆男主+欺骗型关系”的强设定 | 改为“失忆魔尊被我捡到，以为我是他道侣” |
| 男主表面贤惠/隐忍，实则是高位者 | 保留反差：堂堂魔尊，失忆后成了“贤惠夫君”，包揽家务、修炼资源全靠他 |
| 女主“作精”人设（前期惹人厌，后期觉醒反转） | 保留女主前期“作”，但改为“可爱作”+“被迫作”（系统任务），让读者提前共情 |
| 日常互动中的甜与虐的拉扯 | 保留大量日常：男主洗衣做饭、女主躺平，夹杂女主内心恐惧掉马的紧张 |
| 已知悲剧结局的时间压力 | 保留原书剧情线：三个月后魔尊恢复记忆，原主被挫骨扬灰 |

## 爽点公式
**虐 → 爽**，过渡方式是 **甜中带虐，最终虐渣翻身**。

- **虐**：女主穿越成必死炮灰，手持“欺骗大佬”的烂牌，时刻面临被恢复记忆的魔尊挫骨杨灰的结局。
- **甜**：失忆魔尊误以为女主是救命恩人+道侣，对她百依百顺、照顾有加，反向攻略女主。
- **爽**：女主凭现代思维+先知剧情，一边与魔尊“虚与委蛇”发展真感情，一边暗中积攒实力、埋下后手。最终魔尊恢复记忆时，女主已非吴下阿蒙，反而让魔尊陷入“追妻火葬场”。

## 人物设定（与源文差异化）
### 女主 · 沈鹿溪
- **身份**：21世纪社畜，穿进自己正在追的修仙虐文《九霄魔尊》中，成为同名炮灰女配。原主是青云门最低阶的外门弟子，靠招摇撞骗为生。
- **背景**：穿书后绑定“炮灰自救系统”，系统发布第一个任务：让失忆的魔尊李承夜相信她是他道侣，并存活三个月。否则抹杀。
- **性格**：本质咸鱼但惜命，小事上爱偷懒耍赖（被迫伪装“作精”），大事上心思缜密、能屈能伸。
- **目标**：第一阶段——活下去；第二阶段——攒够跑路资本，在魔尊恢复记忆前溜之大吉。
- **差异点**：女主不是纯恶毒炮灰，原主只是小骗子，女主的“作”有系统任务背锅，读者早知她有苦衷。比起源文的纯都市背景，改为仙侠世界，女主能力有成长空间。

### 男主 · 李承夜
- **身份**：三界闻之色变的魔尊，修为通玄，冷酷无情。实际年龄三百岁，因渡劫失败被天雷劈成重伤，流落青云门后山，被原主“捡尸”。
- **背景**：原书中，他会在三个月后被属下找到，恢复记忆，然后亲手将“欺骗侮辱”他的原主抽筋剥皮，神魂俱灭。之后重新一统魔界，成为女主（原书女主）复仇路上的最大阻碍。
- **性格**：（失忆后）沉默寡言但温柔体贴，因记忆空白极度缺乏安全感，将女主视为唯一依靠。（恢复记忆后）多疑、暴戾、掌控欲极强。
- **标签**：#失忆大佬 #贤惠忠犬 #自我攻略 #追妻火葬场预备役
- **差异点**：从现代霸总改为仙侠魔尊，武力值更高，恢复记忆后的压迫感更强。“贤惠”表现从洗衣做饭，变为为她炼制丹药、打造法器、清扫修炼障碍。

### 关键配角（每人一行）
- **系统（代号：柒柒）**：女主的辅助系统，话痨，爱吐槽，发布任务时总夹带私货，是女主的情绪垃圾桶和狗头军师。
- **顾听雪**：原书女主，重生者。前世被魔尊和渣男利用，今生只为复仇和变强，对男主毫无感情线，后期与女主成为亦敌亦友的联盟关系。
- **右护法·玄冥**：魔尊最忠心的属下，找遍三界寻主。外表冷酷，内心憨憨，坚定不移地认为女主是骗子，是后期男女主感情线的主要“绊脚石”。
- **小师妹·苏棠**：青云门天真可爱的小师妹，女主原身的死对头，处处看女主不顺眼，但本性不坏，日常制造无伤大雅的小麻烦。

## 世界观
- **时代背景**：架空修仙世界，人、魔、妖三族并立，维持表面和平已千年。
- **地理**：以人族“青云门”所在的苍梧山开局，后续扩展至魔界“九幽城”、妖族“万妖谷”等。
- **势力**：青云门（正道楷模）、九幽魔宫（魔界至尊）、天机阁（情报组织），以及暗中蠢蠢欲动的上古妖族。
- **核心设定**：女主穿越进一本“重生复仇流”仙侠虐文。原书女主重生归来，本应与魔尊势不两立，但因女主这个变数，三人命运轨迹全部偏离。女主手握剧本，却因自身行为不断“魔改”剧情，导致先知优势逐渐消失，必须依靠自己应对危机。
- **与源文差异化**：从现代都市平移至古代仙侠，世界观更宏大，女主成长体系更明确（修炼升级），冲突形式更多样（斗法、秘境、势力博弈）。

## 简介（三种风格各150-250字）
### 标准版
穿进修仙虐文，成了炮灰女配沈鹿溪。开局系统就发布死亡任务：欺骗失忆的魔尊李承夜，让他相信我是他的救命恩人兼道侣，活过三个月。
我看着眼前这个乖巧洗衣、温柔炼丹，口口声声唤我“夫人”的男人，只觉得头皮发麻。他现在有多体贴，恢复记忆后我的死法就会有多惨烈。
为了活命，我只能一边硬着头皮当他的“作精娇妻”，一边疯狂修炼、暗中谋划跑路大计。可谁知，当我终于攒够资本准备溜之大吉时，他却将我堵在墙角，眸光幽深似海：“夫人，你身上的秘密……比本尊想的更多。”
等等，他是不是……早就恢复记忆了？

### 悬念版
三界之中，无人敢直视魔尊李承夜。他冷酷、暴戾，弹指间可灭一城。
但没人知道，这位大魔头渡劫失败后失了忆，被我这个十八线小骗子捡回了家。
我骗他说，我是他青梅竹马的道侣，为他散尽修为才救回他的命。他信了，从此为我洗衣做饭，寻丹炼药，温柔得像只收起利爪的猛兽。
可我知道，这不过是场为期三个月的骗局。时间一到，利爪便会刺穿我的心脏。
我一边享受着他的无上宠爱，一边为自己的后事做准备。然而，魔尊的属下终于找上了门，那天夜里，他轻抚我的脸颊，笑着问：“夫人，你猜，我是什么时候想起来的？”

### 甜宠版
好消息：我穿书了，还捡了个失忆的帅神仙当夫君。
坏消息：他是未来会把我挫骨杨灰的魔尊，而我只是个骗他的小炮灰。
可这魔尊失忆后，怎么跟传闻中不一样？不杀人放火，反而天天钻研菜谱喂饱我；不修炼魔功，倒日夜替我疏通经脉、打造法器。他看我的眼神，温柔得能滴出水来。
“夫人，今日想吃什么？”
“夫人，天凉了，把这件法衣穿上。”
救命，他再这样宠下去，我怕不是真的要沦陷了？可那三个月后的大限，就像一把悬在我头顶的剑。既然如此——我能不能努努力，在他恢复记忆之前，先让他真正爱上我？
