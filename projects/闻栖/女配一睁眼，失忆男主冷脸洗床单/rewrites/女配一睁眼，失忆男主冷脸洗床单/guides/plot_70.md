## 换皮检验 ⚠️（必须逐项自检）

**第一步**：分别写源文和新书的一句话概括（剥掉人名地名）
```
源文概括：[女主]因为撞见[小学同学]意外与[男主]偶遇，[小学同学]挑衅[男主]，[女主]尴尬逃离后又分别打电话给两人试探态度，最后找[小学同学]借钱。
新书概括：[女主]因为在[男主]公司楼下被[前女友]堵住打探关系，[男主]突然出现解围后，[前女友]以替嫁秘密威胁[女主]必须谈话，留下钩子。
```

**第二步**：逐项对比骨架
| 对比项 | 源文 | 新书 | 是否同构？ |
|--------|------|------|-----------|
| 冲突本质 | 身份冲突（穿书女配身份） | 利益冲突（替嫁秘密+前女友争夺） | 必须不同 |
| 事件结构 | 偶遇→挑衅→尴尬→逃离→电话试探→借钱 | 遭遇→打探→解围→逃离→威胁→留钩子 | 不同 |
| 人物关系模式 | 女主+小学同学（合作+依赖）vs男主 | 女主+前女友（对抗+秘密）vs男主 | 不同 |

**第三步**：台词比对
- 逐句比对源文和新书的台词
- **6字以上连续匹配 = 违规**，必须重写该台词
- 即使换了人名，句式结构相同也算违规

**第四步**：判断
- 3项骨架全部不同 + 台词无雷同→合格
- 任意1项同构或台词违规→换皮，重做该节拍

**违规示例**：
- 源文"母亲担心早恋→发现打架→介入→女孩装可怜→母亲反思"
- 新书"母亲担心成绩→发现争吵→介入→女孩装无辜→母亲反思"
- 冲突本质同构（都是身份冲突），事件结构同构（担心→发现→介入→装→反思）→换皮，重做

**台词违规示例**：
- 源文：「"你怎么能这样对妈妈说话？"」
- 新书：「"你怎么能这样跟妈妈讲话？"」
- 6字连续匹配（"你怎么能这样"）→违规，重写

## 角色高光时刻 ⭐（每章至少一个）

每章必须设计至少一个让人记住的场景。从以下类型中选 ≥1 个，标注发生在哪个节拍：

| 类型 | 说明 | 示例 |
|------|------|------|
| **甜** | 两人之间不经意的心动瞬间 | 他把她做的糊鸡蛋全吃了；她睡着了他给她盖被子 |
| **虐** | 不得不隐瞒/伤害对方的时刻 | 嘴上说分手，手在发抖；骗他出门，自己在屋里哭 |
| **笑** | 轻松幽默的桥段 | 女主逞强做饭差点烧厨房；男主面无表情说土味情话 |
| **反差** | 角色做出人意料的事 | 一直冷漠的人突然护短；一直怂的人突然硬刚 |
| **细节** | 一个小动作/物件承载情感 | 他把唯一的热牛奶推给她；她偷偷存着他写的小票 |
| **打脸** | 被嘲笑后实力碾压 | 众人嘲笑废物，主角一招震全场 |
| **装逼** | 低调展示实力 | 淡淡一句"还行"，实际碾压所有人 |
| **绝境翻盘** | 濒临失败时逆转 | 快死时觉醒新能力/援兵赶到 |
| **推理突破** | 关键线索串联 | 众人迷惑时主角看穿真相 |
| **热血** | 激发斗志的宣言/行动 | "这一战，我接了！" |

**格式**：类型 + 发生在第几个节拍 + 一句话描述 + 为什么读者会记住

```
## 本章高光
- 类型：爽点/打脸 | 节拍#5 | 顾衍突然出现喊陈茉找合同，当着林瑶的面只跟她说话，无视前女友 | 读者记住：失忆男主对“保姆”的关注远超白月光，暗示身体记忆比情感记忆更诚实
- 类型：细节/甜 | 节拍#6 | 顾衍发消息“别理她”，三个字让陈茉心跳加速 | 读者记住：冷漠男主的保护欲藏在细节里
```
