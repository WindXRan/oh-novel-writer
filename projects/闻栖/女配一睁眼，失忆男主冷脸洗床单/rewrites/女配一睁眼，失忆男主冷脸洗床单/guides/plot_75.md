**第一步**：分别写源文和新书的一句话概括（剥掉人名地名）
```
源文概括：[失忆男主]因为[车祸触发记忆碎片但又失败]，[情绪波动]，[女主用拥抱打断追问转移话题]；[回家后给狗洗澡]，[日常互动中暧昧升温]
新书概括：[失忆男主]因为[身体记忆对女主关联的势力（顾氏）产生本能的排斥和恨意]，[陷入困惑和不舒服]，[女主用伪装日常和做饭转移话题，内心不安]；[菜市场选中展现他另类记忆（对女主的体贴）]，[发现林瑶回国短信，冲突将至]

```

## 本章高光
- **类型**：甜 + 细节 | **节拍#8** | 顾衍注意到陈茉想吃肋排但忍着没买，说你想要什么就买。陈茉边切洋葱边哭（被辣哭+被触动）| 读者会记住：他失忆了忘了她是谁，但记得对她好；暗藏的点是——他只是在身体记忆里对她下意识好，不是真的忘了恨，这里藏了“甜中带虐”的感觉
