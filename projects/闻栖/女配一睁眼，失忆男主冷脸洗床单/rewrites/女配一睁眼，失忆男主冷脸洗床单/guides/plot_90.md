## 结构技巧运用 ⭐

### 人物入场方式：T2-侧面描写入场（节拍#3）
**位置**：林瑶首次正式登场
**结构**：管家通报+陈茉视角+林瑶主动对话
```
管家在门口通报时，陈茉正端着茶杯的手顿了顿——'林瑶小姐来了，说有急事要见顾总。'
陈茉放下杯子，『她』怎么来了？三年无声无息，现在挑这个时间点……
顾衍皱眉道：'不见。'
陈茉却站起来：'我去见她。'
穿堂过厅，在花园石桌旁，她看见一个女人侧身站着——
长发，高跟鞋，手腕上的翡翠镯子随动作晃出一抹绿光。
『这就是他的白月光。』
林瑶转身，微微一笑：'陈茉？久仰了。'
```
**技巧说明**：先通过管家之口引入“林瑶”之名，再通过陈茉的走动缓缓展开人物形象，最后以林瑶的笑和台词完成正式亮相。建立“白月光”的神秘感后，再释放人物全貌。

### 场景切入方式：S1-人物与场景结合（节拍#1）
**结构**：人物状态（陈茉位置+行为）→场景刻画（书房物景）→人物特写（顾衍推门）
```
陈茉在书房里整理文件，手指划过一排文件夹标签，停下了。
办公桌上，一封未拆的信封静静地躺在台灯底座旁——
收件人：顾衍，寄件人落款：林瑶。
窗外午后的阳光斜照进来，在信封上拉出一道细长的影子。
陈茉屏着呼吸，盯着那个名字，指尖微凉。
门被推开时，顾衍站在门口，逆着光，声音冷淡：'你在做什么？'
```
**技巧说明**：全景（书房）→聚焦（信封）→细节（阳光斜照）→身体反应（屏着呼吸、指尖微凉）→人物入场（顾衍推门+台词）

### 章间衔接方式：C5-蒙太奇（节拍#8→#9）
**手法**：以“手机/水声”作为共有对象切换时间
```
陈茉翻过手机，屏幕暗下去的瞬间，映出她自己的脸。
楼上水声停了。
时间跳转——
顾衍出差前夜，行李箱摊在房间地板上。
她靠在衣柜边，看他清点完衣服，沉默片刻，走向卫生间——
顺手拿起床上的床单，扔进洗衣机。
```
**技巧说明**：同以“水声”作为衔接点，压缩中间两天的日常，直接推进到出差前夜。没有流水账，保持节奏感。

### 钩子类型：行为悬疑（节拍#10）
**结构**：陈茉被半夜动静惊醒 → 发现顾衍坐在床边 → 他看着手上伤口说“不记得怎么弄的” → 留白
```
半夜，陈茉被隔壁房间传来的动静惊醒——'砰'的一声闷响。
她条件反射跳下床，光脚跑出走廊，推开顾衍房间的门。
他坐在床沿，台灯亮着，脸色苍白地盯着自己手上的那道伤口——
白天给她包扎时划破的。
'你怎么了？'
顾衍抬头，神情有些恍惚：'我……不记得这个是怎么弄的了。'
陈茉愣在原地。
```
**技巧说明**：不是对话式悬念（“我要离开”），而是行为式悬疑——伤口本身其实已经不再重要，重要的是顾衍开始困惑于自身的记忆断层。这种“遗忘者对遗忘本身的觉醒”比直接揭露真相更能制造持续期待。
