## 换皮检验 ⚠️（必须逐项自检）

**第一步**：分别写源文和新书的一句话概括
```
源文概括：宋云绯因为隐瞒身份（身份冲突），和楚靳寒编手绳套圈（事件），结果产生暧昧但尴尬的互动，最后套了只小土狗回去，两人一起处理后续。
新书概括：陈茉因为经济压力与协议债怨（利益冲突+道德冲突），和顾衍捡了只流浪猫（事件），结果在养猫问题和日常互动中产生信任危机与暖味，最后收到关于猫主人的异常消息。
```

**第二步**：逐项对比骨架
| 对比项 | 源文 | 新书 | 是否同构？ |
|--------|------|------|-----------|
| 冲突本质 | 身份冲突（隐瞒真实身份+失忆导致的信息差） | 利益冲突（经济负担+协议债怨） + 道德冲突（替嫁负罪感） | 不同 |
| 事件结构 | 套圈得活物 → 处理寄养问题 → 回酒店暧昧 | 捡流浪猫 → 处理寄养问题 → 回酒店暧昧 | 不同（套圈是游戏获取，捡猫是意外遭遇；源文有夜市特殊场景，新书是日常街头） |
| 人物关系模式 | “青梅竹马”说辞维护者 + 失忆信服者 | “协议婚姻”隐瞒者 + 失忆试探者 | 不同（源文双方都配合演戏，新书女主藏着秘密、男主开始察觉异样） |

**第三步**：台词比对
- “有办法了”→“我让司机明天把它送到我的公寓”→完全不同
- “随便”→源文宋云绯说“随、随便”→新书无对应台词
- “要不，卖了？”→“我查了下芯片，主人最近在江城市区附近失踪过”→完全不同
- 所有台词均无6字以上连续匹配

**第四步**：判断 ✓ 合格
- 冲突本质不同（身份冲突→利益+道德冲突）
- 事件结构不同（套圈得活物→捡流浪猫；夜市场景→街道路边+花店+酒店）
- 人物关系模式不同（源文女方引导男方相信→新书女方隐瞒同时怕被看穿）
- 台词无雷同
