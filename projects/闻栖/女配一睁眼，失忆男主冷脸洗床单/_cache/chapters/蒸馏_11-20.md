# 文风蒸馏 第11-20章

## 统计指纹（Layer A）
- **总字数**：约18500字（10章）
- **均章字数**：约1850字
- **对话占比**：约55%（高对话密度）
- **段落长度**：以1-3句短段为主
- **对话与描写的比例**：对话55% / 动作描写25% / 心理描写15% / 环境5%
- **人称**：第三人称有限视角（跟随宋云绯）

## 8维度分析（Layer B）

### 1. 叙事声音与语气
- **声音定位**：紧贴女主视角，第三人称有限视角，偶尔穿插内心OS
- **语气特征**：轻松幽默为主→紧张焦虑→温情→暧昧暗涌
- **典型句式**：
  - "宋云绯……"（主语开头，动作/心理描写）
  - 内心OS带调侃语气："果然，风水轮转啊。""诡计多端的男人！"
- **节奏控制**：日常对话节奏快→情感场景节奏慢（如18章生病照顾）

### 2. 对话风格
- **高密度对话**：每章50%+是对话
- **对话特征**：
  - 短句对白为主（一句一换行）
  - 标签多用"XX道""XX说"（偶尔省略标签靠上下文）
  - 女主对话活泼（内心戏多），男主冷酷简短（2-5字）
- **典型**：
  - 宋云绯：一大堆解释/吐槽/OS
  - 楚靳寒："嗯""没有""说啊""是吗"（惜字如金）
  - 互动模式：宋云绯喋喋不休→楚靳寒简短回应→宋云绯内心消化

### 3. 场景描写特征
- **极简环境描写**：几乎不描写环境，一笔带过
  - "太阳越来越大""风扇还在转着""商场里冷气真足"
- **聚焦人物动作**：少量但精准的动作描写
  - "楚靳寒把小电驴推了出来""他伸出一根手指，指了指自己的鼻尖"
- **特色**：用细节展现生活质感（破电动车/五十块奶茶/打折洗衣粉）

### 4. 转折与衔接手法
- **章末钩子**：每章结尾留悬念或情感钩子
  - 11章："不要骗我"
  - 14章：楚靳寒追上来给奶茶
  - 20章：楚靳寒看余额后黯然
- **章节衔接**：时间推移+日常场景自然过渡
- **冲突推进**：一个冲突解决→引入新冲突

### 5. 节奏特征
- **总体**：慢→快→慢的波浪节奏
- **10章整体**：慢（日常）→快（冲突）→慢（温情）
- **每章内**：开头日常→中间冲突→结尾钩子
- **关键手法**：用"老鼠"式（第20章）的幽默化解尴尬

### 6. 词汇偏好
- **高频词**：宋云绯、楚靳寒、没有、知道、觉得、忽然
- **口头禅/特色词**：
  - "不由"（不由开始闪躲）
  - "愣了下/怔了怔"（频繁使用）
  - "咯噔一下"（心里咯噔一下）
  - "心头一颤/心跳漏了一拍"
- **避免词**：无明显华丽辞藻，极少成语

### 7. 情绪表达方式
- **方式**：外化在动作+内心OS，不直接描写情绪词汇
  - 紧张："头皮发麻""眼神闪躲"
  - 心动："心跳漏了一拍""指尖传来的触感"
  - 感动："鼻尖竟有些发酸"
- **特色**：情绪通过对话反衬（男主说"怕你饿死"实则关心）

### 8. 独特习惯
- **用"吐槽"推进性格**：女主内心疯狂吐槽，制造笑点
- **"大喘气"对话**：角色说话半句停顿，制造张力
- **内心独白信息量**：女主心理活动解释剧情（帮助读者理解）
- **情景喜剧式尴尬**：隔壁声音/偷看被发现/跨坐等尴尬场景
- **物质细节象征**：破电动车/五十块/两百块的余额→经济困境+男主自尊心
