## 换皮检验 ⚠️（必须逐项自检）

### 第一步：一句话概括
```
源文概括：林望夕因为被蚊子咬，周今远帮她涂药时她偷亲失败，后来发现他穿破内裤，想帮他买新被拒绝。
新书概括：宋知渺因为裴行舟用错胶水修复古籍，陷入说与不说的两难，递给他半颗卤蛋被拒，后来发现他用旧矿泉水瓶装绿豆汤，想帮他但不知道怎么开口。
```

### 第二步：逐项对比骨架

| 对比项 | 源文 | 新书 | 是否同构？ |

| 冲突本质 | 身份冲突 | 利益冲突（修复方法分歧） | 不同 ✓ |
| 事件结构 | 发现蚊子包→涂药→偷亲→被推→发现破内裤→对话 | 发现修书缺陷→关心→尴尬→被拒绝→发现旧瓶子→内心挣扎 | 不同 ✓ |

| 经典桥段 | 破内裤（幽默反转） | 破绿豆汤瓶（心酸反转） | 不同 ✓ |
