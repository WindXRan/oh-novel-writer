## 换皮检验 ⚠️（必须逐项自检）

**第一步**：分别写源文和新书的一句话概括（剥掉人名地名）
```
源文概括：女主腿伤被男主照顾，男主朋友来找他接一个两万块的小项目，男主因缺钱接下，并借钱给女主买衣服/药。
新书概括：女主发现男主有亲人患癌，主动提出参与男主的廉价项目来帮忙还债，男主因自尊和生存压力挣扎后接受，两人建立微妙的合作关系。
```

**第二步**：逐项对比骨架
| 对比项 | 源文 | 新书 | 是否同构？ |
|--------|------|------|-----------|
| 冲突本质 | 生存冲突+人情债（男主欠债→接廉价项目） | 生存冲突+道德债（女主肇事逃逸→想帮忙还债） | 不同 ✓ |
| 事件结构 | 照顾→朋友来→接项目→借钱→买衣服 | 发现秘密→困境→朋友来→接项目→贡献技能→建立合作 | 不同 ✓ |
| 人物关系模式 | 男主单方面照顾/亏欠 | 双向亏欠→开始合作 | 不同 ✓ |
| 经典桥段 | 朋友来访+廉价项目+借钱 | 发现秘密+贡献技能+建立合作 | 不同 ✓ |

**桥段检查**：源文用了“借钱买衣服”→新书用“贡献技能+药品账单”，完全不同。

**场景设计检查**：源文是做饭→朋友来→借钱→买衣服；新书是快递柜→对话→朋友来→贡献技能→合作。

**第三步**：台词比对
- 源文：「“你的腿不能沾水。”」「“医生说的。”」「“借我两万。”」「“你到底有没有？”」

- 3项骨架全部不同 ✓
