## 换皮检验 ⚠️（必须逐项自检）

**第一步**：分别写源文和新书的一句话概括
```
源文概括：林望夕因为失恋和无处可去，在工厂打螺丝，忍受身体的痛苦和同事的嫉妒，
被暗中观察的周今远和沈澈看到，因某种原因不能联系。

新书概括：宋知渺因为躲避某种危险和无处可去，在小餐馆洗菜，
忍受身体的痛苦和老板娘的压榨，被暗中观察的裴行舟和沈澈看到，因某种危险不能联系。
```

**桥段检查**：源文使用了"工厂打工"经典桥段，新书使用"小餐馆洗碗"替代，两者虽同属底层劳动场景，但具体环境、人际关系、冲突方式完全不同。

**场景设计检查**：
- 源文：工厂车间、宿舍（有虫子）、食堂门口、车里
- 新书：城中村小餐馆后门、后厨、廉价旅馆、便利店门口
- 场景类型全部更换

**第三步**：台词比对
- 源文台词：「我不想打螺丝」「我不喝奶茶」「你咋了，不就是被咬了一口，你哭啥？」
- 新书台词：「老板娘说工资日结」「早知道现实这么难」「她就这么能扛？」
- 无6字以上连续匹配

**第四步**：判断
- 冲突本质不同（身份冲突→意志冲突）
- 事件结构不同（工厂流水线→小餐馆后厨）
- 人物关系模式不同（因禁忌→因危险）
- 台词无雷同
→ 合格
