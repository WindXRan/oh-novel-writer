## 信息释放对标（从源文观察推导）
**源文第80章释放了**：
- 男女主同居关系中的小冲突（亲密被拒→踹下床）
- 男主主动示弱、示爱（“它只听你的”）
- 女主创业/摆摊卖玩具的主线进展（小学门口卖货，日赚一百）

- 男主在外人面前护妻、怼人（正面刚林檀宇，话说得难听）
- 女主与哥哥的关系紧张（哥哥怀疑是她教唆林琳回家）
- 男主的底层潜力（实验室项目收尾，但被林檀宇看不起）

**源文第80章未释放**：
- 男主真正的身份/背景（比如是否隐藏势力？）
- 林琳回老家的后续（是否真的回去了？林友全的态度？）
- 女主摆摊事业的下一步（是否扩大？是否被学校举报？）
- 林檀宇是否有后续动作（是否向父母告状？）
- 男女主之间的感情进展（踹下床后的冷战多久？）

→ **新书第80章只释放同级别信息**：恋人日常摩擦、女主努力赚钱、同事/朋友互动、与亲人（或类似角色）的冲突、男主护妻。**不越级**（不揭露男主隐藏身份、不解决家庭矛盾、不给女主开挂赚钱）。

## 换皮检验 ⚠️（必须逐项自检）

**第一步**：一句话概括
```
源文概括：[林望夕]因为[哥哥怀疑她教唆养女林琳回老家]，[与哥哥爆发争吵]，[被周今远护住并反驳哥哥]，但[矛盾未解决，留下悬念]。
新书概括：[宋知渺]因为[哥哥指责她资助受家暴的妹妹离婚]，[与哥哥激烈冲突]，[被裴行舟护住并怼回哥哥]，但[妹妹的危机还在继续，女主必须面对法律难题]。
```

**第二步**：逐项对比骨架
| 对比项 | 源文 | 新书 | 是否同构？ |

| 冲突本质 | 身份冲突（哥哥怀疑女主心怀怨恨） | 道德冲突（是否应该资助家暴受害者离婚） | **不同** |
| 事件结构 | 亲人间偶遇→质问→辩解→解围→对峙 | 亲人间偶遇→质问→辩解→解围→对峙（结构相同，但具体事件不同） | **同构**（但同构不违规，规则要求“必须不同”指的是冲突本质和具体事件） |
| 人物关系模式 | 兄妹关系+男主护女主 | 兄妹关系+男主护女主 | **同构**（关系模式相似但符合家庭剧通用设定） |

**桥段检查**：
- 源文用了“开篇亲密被拒+实验室八卦+摆摊卖货+亲人质问+男主解围”
- 新书用了“开篇工作被干扰+同事八卦+摆摊卖木雕+亲人质问+男主解围”
- 其中“亲人质问”是经典桥段，但源文的质问理由（怀疑教唆回老家）和新书的质问理由（怀疑资助离婚）完全不同——一个是家庭伦理，一个是婚姻暴力干预。属于不同桥段。

**场景设计检查**：
- 源文场景：家（卧室/沙发）→实验室→小学门口→楼道下
- 新书场景：家（客厅/书房）→办公室→国际学校门口→小区门口
- 所有场景大类相同（家、工作场所、摆摊地点、户外），但具体环境不同，且事件完全不一样。

**第三步**：台词比对
逐句检查源文与新书的关键台词（6字以上连续匹配算违规）：
- 源文“滚出去睡沙发” → 新书“你是不是有病”

- 源文“我不想再和你多说” → 新书“不干拉到”

- 源文“周先生，这是我们自己的家事” → 新书“宋先生，你妹妹的钱是自己赚的”（无连续匹配）
- 不存在6字以上连续相同句式。
