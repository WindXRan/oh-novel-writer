# 第28章 章纲

## 全局节奏位置（从 arc.md 读取）
- 当前章节在第28章 / 全书共101章
- 所处阶段：过渡期 → 蓄力期（为后续大高潮铺垫）
- 本章应交付的读者情绪：压抑→小爽→悬念（章末埋钩子）
- 前章结尾梗概：林瑾在修复大赛初赛上技惊四座，但被前婆婆赵秀芬当众羞辱"离了婚的女人还抛头露面"，林瑾沉默离场，回到公寓楼下时，恰遇顾西洲开车来接她。
- 本章在本卷中的功能：建立林瑾与前夫家族的第二个回合的矛盾冲突（专业尊严打压），展现林瑾在哥哥层面的关系困境（而非直接与前夫冲突），为后续让顾西洲介入"打脸"做铺垫。同时也是林瑾与顾西洲感情线推进的过渡章。

## 源文字数
2090字 → 新书目标 2090字（1:1对标）

## 角色行为模式（从concept.md取，write-chapter 据此保持一致）
- **女主**：林瑾 → 应激：遇到冲突第一反应是冷静观察，情绪内敛，但会通过手上的小动作（如摩挲工具）暴露内心；决策：感性直觉与理性分析并存，对文物修复直觉极准，对人则理智分析利弊；情感：极度间接和压抑，表达愤怒的方式是沉默和专注工作；弱点：害怕被否定"价值"，尤其是自己钟爱的事业被贬低。
- **男主**：顾西洲 → 应激：遇到对林瑾不敬的人，第一反应不是发怒，而是用数据、事实或碾压级的技术优势让对方哑口无言；决策：极端理性优先，将一切视为"项目"或"课题"；情感：直接、笨拙、学术化，行动远大于语言；弱点：害怕林瑾认为他也是"不懂她"的俗人。
- **场景A（哥哥质问）需展现**：林瑾的"遇到冲突冷静观察、情绪内敛"应激模式 + "害怕被否定专业价值"的弱点
- **场景B（顾西洲接人）需展现**：顾西洲的"行动大于语言"情感表达模式 + "用技术优势护短"的应激反应雏形（只是暗示，本章不做实）

## 信息释放对标（从源文观察推导）
- **源文第28章释放了**：
  - 女主的哥哥（盛霄）对女主的严厉态度——核心矛盾（家族声誉 vs 个人选择）
  - 哥哥用"父亲之死"作为道德武器攻击女主
  - 哥哥用"盛家"作为绑架工具，逼迫女主要么离婚要么自己处理
  - 男主（秦钰）与女主的最好朋友（林蓁）建立了联系
  - 男主主动来接女主回家，暗示"有阴谋"——埋钩子
- **源文第28章未释放**：
  - 女主内心是否真的认同哥哥的指责
  - 男主的"阴谋"具体是什么
  - 哥哥与男主之间的关系真相（两人是朋友）
  - 女主的真正能力/秘密身份
- **新书第28章只释放同级别信息**：
  - 释放：林瑾的哥哥（设定为"晏铭洲"，国家级修复协会理事）对她"不务正业"的指责；前夫家族利用资源施压；顾西洲主动来接、暗示"已经在查"/"有计划"

## 换皮检验 ⚠️（必须逐项自检）

**第一步：分别写源文和新书的一句话概括**
```
源文概括：盛霄因为盛家声誉受损，指责盛惜自私、不懂得为家族考虑，逼迫她要么离婚要么自己解决问题，盛惜沉默答应，后被秦钰接走，秦钰暗示"有阴谋"。
新书概括：晏铭洲因为怕协会对林瑾的专业身份有非议，斥责她修复前夫家的传家宝是"打了刘家的脸"，逼迫她退出修复大赛或跟刘家道歉，林瑾痛苦挣扎，后被顾西洲以"编造的合作项目"解围，顾西洲暗示"有计划"。
```
