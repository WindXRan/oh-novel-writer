# 第7章 章纲

## 全局节奏位置
- 当前章节在第7章 / 全书共101章
- 所处阶段：过渡期→蓄力期（离婚后适应期，新关系确立中）
- 本章应交付的读者情绪：压抑后的轻松感 → 轻微的甜与好奇 → 结尾小悬念
- 前章结尾梗概：林瑾在医院醒了，顾西洲去办了出院手续，两人初步接触，顾西洲提出去他那儿。
- 本章在本卷中的功能：展现离婚后女主的心理释放过程；深化男女主关系的日常互动；铺垫新书核心冲突（价值认知冲突）的雏形。

## 角色行为模式
- 女主：林瑾 → 应激(遇到冲突第一反应是冷静观察，情绪内敛，但会通过手上的小动作暴露内心)，决策(感性直觉与理性分析并存，对人会理智分析利弊)，情感(极度间接和压抑，表达愤怒的方式是沉默和专注工作)，弱点(害怕被否定价值，害怕被抛弃)
- 男主：顾西洲 → 应激(遇到对林瑾不敬的人，用数据、事实或技术优势让对方哑口无言，行动力极强)，决策(极端理性优先，将一切视为"项目"或"课题")，情感(直接、笨拙、学术化，行动远大于语言)，弱点(害怕林瑾认为他也是"不懂她"的俗人，害怕林瑾的手受伤)
- 本章需体现：场景A(车内反思)→展现女主决策模式与情感表达；场景B(买早餐)→展现男主行动力与情感表达模式；场景C(讨论工作)→展现女主情感弱点

## 信息释放对标
- 源文第7章释放了：离婚后的心理状态(疲惫、释然)；与前夫的彻底切割(拉黑删除)；新关系的初步定义(契约婚姻)；男主的日常面(暴躁又细心)；女主的工作背景(基金经理，在姜氏旗下)；双方对婚姻期限的模糊态度。
- 源文第7章未释放：男主的核心能力/实力；女主的隐藏身份/技能；男主的家庭背景深层细节；契约婚姻的具体条款。
- 新书第7章只释放同级别信息：离婚后的心理状态；与修复工作的关系；新关系的定义；男主的日常面；女主的职业背景对下一步的考虑；双方对未来的模糊态度。
