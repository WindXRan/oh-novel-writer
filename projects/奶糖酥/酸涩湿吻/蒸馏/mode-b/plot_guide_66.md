# 写章指南：第66章

## 写章目标
全书最昂扬的高光时刻。百日誓师大会上，林溪在台上发光，她在气球上写下"溯光"和"晴雨"——把两个人的名字和命运紧紧绑在一起，放飞到蓝天。

## 鼓点节奏
- **章首0-500字**：周五，天气晴朗。操场上彩旗飘扬，学生们按班级列队。林溪穿着校服站在主席台侧面的候场区，手里攥着演讲稿，手心出汗。她扫视台下密密麻麻的人群，紧张得想逃。然后她看见了陆深——他本来没打算来，但听说她发言，还是赶来了。他站在队伍最后排，在人群中一眼锁定了她的位置。隔着操场两人目光相撞。林溪的心忽然定下来了。
- **发展500-1500字**：轮到她走上前台，声音清脆有力。演讲稿没有空洞口号，而是写满了高三学子的迷茫与坚持。她说："我们都在等一场日出。或许现在还身处黑夜，但只要还在奔跑，那轮红日终将升起。" 台下掌声雷动。外婆坐在家长席用力鼓掌，看到林溪看过来又立刻假装在看横幅。发言结束，最后的环节是放飞梦想气球。
- **高潮1500-2000字**：林溪拿到一个粉色气球。她穿过人群跑向操场边缘的陆深，手里多拿了一个深蓝色的。"大家都写梦想，你也写一个。" 陆深不写。她说那我帮你写。她在他那个蓝色气球上写了两个字——【溯光】。又在自己粉色的气球上写了【晴雨】。"溯光就是逆流而上追寻光亮。晴雨就是雨过天晴。" 那是藏在他名字里的方向和藏在她名字里的天气。她把两个气球的线系在一起，打了个死结。她说："系在一起，就不会走散了。"
- **钩子最后200字**：全校开始倒计时——三、二、一！上千个彩色气球同时升空。她拉过他的手，把那两根系在一起的线塞进他手里。两人掌心交叠。"一起。"她说。松手的瞬间，那一蓝一粉两个绑在一起的气球在漫天气球中格外独特——它们相互依偎着，坚定地朝最高的云层飞去。陆深仰头看着，阳光刺得眼眶发热。他心底默默许愿：如果真的有人能听见——请保佑这场雨，永远天晴。
