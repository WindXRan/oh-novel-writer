# 第139章 风格速查

## 定量锚点（写章后自动校验用，必须填准确数字）

- 总字数：1381字 → 仿写目标1381字（**必须在1243~1519字范围内，否则判定为不合格**）
- 比喻句数量：2句（含“像/仿佛/犹如/恍如”的句子）
- AI路标词出现次数：0次（首先/其次/然后/最后/与此同时/值得注意的是/此外）→ **仿写必须为0**
- 直抒情出现次数：3次（“充满了”“感到无比”“心中涌起”“不由得”“不禁”等）
- 省略号数量：1个（**仿写不得超过2个**）

## 本章执行规则（write-chapter 必须逐条遵守）

### 句式规则（4条）

**规则1：短句占比≥45%，锚定47.8%**
- ✅正向：用15字以下短句切割情绪，尤其情感爆发处用5-8字短句。“胸口一窒。眼睫微颤。指节发白。”
- ❌反向：连续3句超过20字 → 判定为节奏拖沓。违例：“她望着他离开的背影，在昏黄的路灯下渐渐模糊，最后消失在了街道尽头。”

**规则2：对话段落用一句话段落，叙事段不超过3句**
- ✅正向：对话段独立成句，叙事与对话交替，每2-3个叙事段必有1个对话段打断。
- ❌反向：连续4段纯叙事无对话 → 违规。违例：段落间形成“叙述→叙述→叙述→叙述”无对话打断。

**规则3：段落句尾多用“。”结尾，段落尾句制造悬停感**
- ✅正向：章末段留白悬停，不解释情绪。“她没回头。他也没追。”
- ❌反向：章末用圆润总结句 → 违规。违例：“最终，一切都回到了它该有的样子。”

**规则4：长句（≥30字）控制在26.1%以内，最长句不超过67字**
- ✅正向：重要描写或心理活动用长句堆叠细节，但不超过文内最长句长度。
- ❌反向：出现超过75字的长句 → 违规。违例：一个句子包含5个以上逗号分句。

### 对话规则（3条）

**规则5：对话占比≈18%，不用“他说/她道”标签**
- ✅正向：用动作标签或无标签直接引语。“凑近半步压低声音：‘疼不疼？’”
- ❌反向：出现“她说”“他开口说”“XX道” → 违规。违例：“她小声说道：‘你先走吧。’”

**规则6：对话口语化，有打断、抢话**
- ✅正向：短句对轰，台词不过8字。“你疯了吗？”“你管我。”
- ❌反向：对话超过15字带解释性台词 → 违规。违例：“我觉得你不应该在这个时候做这个决定，因为风险太大了。”

**规则7：内心独白不加引号，直接用意识流**
- ✅正向：“怎么是他。怎么偏偏是他。腕上的温热漫到指尖。”
- ❌反向：内心活动加引号变成台词 → 违规。违例：“他心想：‘她怎么会在这里。’”

### 描写规则（4条）

**规则8：情绪外化，用身体反应替代直抒**
- ✅正向：“呼吸滞了一下。喉间生涩，像咽了把碎玻璃。”“眼眶一酸。”“指腹狠狠抵进掌心。”
- ❌反向：出现“心中涌起”“充满”“感到无比” → 违规，除非源文出现。违例：“她心中涌起一阵刺痛。”

**规则9：环境描写控制在2处以内，每处不超过1句**
- ✅正向：只在场景转换处用1句环境，嵌在动作中。“巷口灯影斜斜落下来，在两人之间隔了一道明暗界线。”
- ❌反向：连续3段以上写环境 → 违规。违例：“楼道里很黑。窗开着。风灌进来。雨打在玻璃上。”

**规则10：比喻用日常物象，嵌入叙事**
- ✅正向：比喻不超过1句，物象简单可感。“声音像从水底捞起来的。”“锁骨凹下去一块，像盛了半盏光。”
- ❌反向：比喻抽象或陈词滥调 → “他的心像天气一样阴沉”“她美得像一朵花”。

**规则11：触觉和嗅觉偏好**
- ✅正向：多用触觉（温度、质感、痛感）和嗅觉（气味）造沉浸感。“指尖碰到的皮肤凉薄的厉害。”“空气里有潮湿的烟草味和隔夜的雨。”
- ❌反向：只用视觉描写，缺乏触嗅觉感官 → 违规。违例：“他站在门口，穿着白衬衫，手里拿着伞。”

### 防AI检测（以源文为准，严格执行）

- **✅正向**：源文用「然而」「但是」→ 仿写也可以用；源文不用 → 仿写也不用
- **❌禁止**：直抒情词「感到无比」「心中涌起」「不由得」「不禁」「充满」→除非源文有，否则禁止
- **❌禁止**：AI路标词「首先/其次/然后/最后/与此同时/此外/值得注意的是」→ **绝对禁止，发现即删除**
- **❌禁止**：省略号超过源文数量 → 源文1个→仿写最多2个
- **写完后必须逐项检查，违规必须修改后才能保存**

### 字数控制（必须达标，严格执行）

- 目标字数：1381字
- 允许范围：1243~1519字（±10%）
- **写完后必须数字数，不在范围内必须修改**
- **字数不足**：加动作细节（手指、呼吸、眼神）、触觉/嗅觉描写
- **字数超标**：删冗余副词和修饰语，合并短段

### 反抄袭规则（必须遵守）

- **台词原创**：所有对话必须完全重写，禁止复制源文台词
- **连续字数限制**：任何连续8字以上不得与源文匹配（换了人名也算违规）
- **句式变换**：源文用长句→仿写用短句；源文用对话→仿写用心理描写+动作

### 节奏要求（写章后自检）

- **内心戏占比**：≤30%（超过=节奏慢）
- **女主/男主必须有主动性**：做决定、采取行动、逼近/推开对方
- **每章至少一个“小胜”或“钩子”**：一句台词、一个眼神、一个触感（参考plot_guide的高光时刻）
