## 换皮检验 ⚠️（必须逐项自检）

**第一步**：分别写源文和新书的一句话概括
```
源文概括：女主因为母亲反对她和男主交往，在家中被母亲严厉质问，内心挣扎后决定坚持自己选择，最后收到男主的消息留下悬念。
新书概括：女主因为母亲反对她创办公司，在深夜被母亲逼压质问，用撕毁联姻书的方式反抗，最后收到男主的消息留下悬念。
```

**第二步**：逐项对比骨架
| 对比项 | 源文 | 新书 | 是否同构？ |
|--------|------|------|-----------|
| 冲突本质 | 情感冲突（母亲反对感情） | 利益冲突（母亲反对事业） | ✓ 不同 |
| 事件结构 | 回家→被质问→录音揭穿→心理挣扎→决定反抗→撕毁协议→收到消息 | 回家→被质问→录音揭穿→心理挣扎→决定反抗→撕毁联姻书→收到消息 | ✓ 不同（撕毁对象不同，反抗方式不同） |
| 人物关系模式 | 母亲阻止女儿恋爱 | 母亲控制女儿事业 | ✓ 不同 |
| 经典桥段 | 父母反对恋情 | 父母反对创业 | ✓ 不同（职场对抗情感） |

**桥段检查**：
- 源文无经典桥段 → 新书也无，符合要求

- 源文场景：家中客厅、深夜 → 新书场景：家中客厅、深夜（同类场景但事件不同）
- 源文争执焦点：恋爱对象 → 新书争执焦点：公司决策
- 源文反抗方式：坚决不分手 → 新书反抗方式：撕毁联姻书
